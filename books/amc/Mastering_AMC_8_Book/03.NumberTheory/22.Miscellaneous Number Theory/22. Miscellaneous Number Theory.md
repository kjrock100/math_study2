# 22. 기타 정수론 (Miscellaneous Number Theory)

## 1. 개요 (Overview)

이 장에서는 소수, 약수, 합동식 등 정수론의 핵심 주제 외에 AMC 8과 같은 경시대회에서 자주 등장하는 흥미로운 주제들을 다룹니다. **진법(Base Systems)**, **회문수(Palindromes)**, **순환소수(Repeating Decimals)** 등이 포함됩니다.

## 2. 주요 개념 (Key Concepts)

### 2.1 진법 (Base Systems)

우리가 사용하는 10진법(Decimal System)은 각 자릿수가 $10$의 거듭제곱($1, 10, 100 \dots$)을 나타냅니다. $b$진법(Base $b$)은 각 자릿수가 $b$의 거듭제곱을 나타냅니다.

- **표기:** $(d_n d_{n-1} \dots d_0)_b$
- **값:** $d_n \cdot b^n + d_{n-1} \cdot b^{n-1} + \dots + d_1 \cdot b^1 + d_0 \cdot b^0$
- **변환:**
  - $b$진법 $\rightarrow$ 10진법: 위 다항식 계산.
  - 10진법 $\rightarrow$ $b$진법: 수를 $b$로 계속 나누어 나머지를 역순으로 나열.

### 2.2 회문수 (Palindromes)

앞에서 읽으나 뒤에서 읽으나 똑같은 수를 말합니다.

- 예: $121, 1331, 5, 777$.
- 특징: 대칭성을 가지며, 자릿수 배치 문제로 자주 출제됩니다.

### 2.3 순환소수 (Repeating Decimals)

소수점 아래 특정 구간이 무한히 반복되는 소수입니다. 유리수(분수)와 동치입니다.

- **표기:** $0.121212\dots = 0.\overline{12}$
- **분수 변환:**
  - $x = 0.\overline{abc} \implies 1000x = abc.\overline{abc}$
  - $1000x - x = abc \implies 999x = abc \implies x = \frac{abc}{999}$.

## 3. 예제 (Examples)

### 예제 1: 진법 변환

$123_5$ (5진법)를 10진법으로 나타내시오.

- $1 \times 5^2 + 2 \times 5^1 + 3 \times 5^0$
- $= 25 + 10 + 3 = 38$.
- **정답:** 38.

### 예제 2: 회문수 만들기

두 자리 자연수 중 회문수는 몇 개인가?

- 형태: $AA$ (11, 22, ..., 99).
- $A$는 1부터 9까지 가능.
- **정답:** 9개.

### 예제 3: 순환소수 계산

$0.\overline{3} + 0.\overline{6}$의 값은?

- $0.\overline{3} = \frac{3}{9} = \frac{1}{3}$.
- $0.\overline{6} = \frac{6}{9} = \frac{2}{3}$.
- 합: $\frac{1}{3} + \frac{2}{3} = 1$.
- **정답:** 1.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 진법 변환과 회문수 판별을 수행하는 함수입니다.

```python
def base_to_decimal(number_str, base):
    """
    b진법 문자열을 10진법 정수로 변환
    """
    return int(number_str, base)

def decimal_to_base(n, base):
    """
    10진법 정수를 b진법 문자열로 변환 (base <= 10)
    """
    if n == 0:
        return "0"
    digits = []
    while n:
        digits.append(str(n % base))
        n //= base
    return "".join(digits[::-1])

def is_palindrome(s):
    """
    문자열이 회문인지 판별
    """
    return s == s[::-1]

def fraction_from_repeating_decimal(repeating_part):
    """
    0.aaaa... 형태의 순환소수를 분수(문자열)로 반환
    예: '3' -> 3/9 -> 1/3
    """
    numerator = int(repeating_part)
    denominator = int('9' * len(repeating_part))
    import math
    gcd_val = math.gcd(numerator, denominator)
    return f"{numerator // gcd_val}/{denominator // gcd_val}"

# 테스트
print(f"123_5 to decimal: {base_to_decimal('123', 5)}") # Expected: 38
print(f"38 to base 5: {decimal_to_base(38, 5)}")       # Expected: 123
print(f"Is 121 palindrome? {is_palindrome('121')}")     # Expected: True
print(f"0.1212... as fraction: {fraction_from_repeating_decimal('12')}") # Expected: 4/33
```

## 5. 요약 (Summary)

- **진법:** 자릿수의 위치값($b^n$)을 이해하고 변환할 수 있어야 합니다.
- **회문수:** 대칭성을 이용하며, 진법과 결합되어 출제되기도 합니다.
- **순환소수:** 분모가 9, 99, 999... 꼴인 분수로 바꿀 수 있습니다.

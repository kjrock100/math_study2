# 22.5 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 22.1장부터 22.4장까지 학습한 기타 정수론 주제들을 종합하여 실전 문제를 해결합니다. **회문수(Palindromes)**, **화폐 문제(Money Problems)**, **정의된 연산(Defined Operations)**, **치킨 맥너겟 정리(Chicken McNugget Theorem)**, 그리고 **진법(Base Numbers)** 등 다양한 유형의 문제가 포함됩니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **회문수 (Palindromes):** 특정 범위 내의 회문수 개수 세기, 주어진 수보다 큰 가장 작은 회문수 찾기.
2.  **화폐 문제 (Money Problems):** 동전으로 특정 금액을 만드는 방법의 수, 만들 수 없는 금액 찾기.
3.  **정의된 연산 (Defined Operations):** $a \star b$와 같이 새롭게 정의된 기호에 맞춰 계산하기.
4.  **치킨 맥너겟 정리:** 서로소인 두 수로 만들 수 없는 가장 큰 수 구하기 ($ab - a - b$).
5.  **진법 (Base Numbers):** $b$진법을 10진법으로 변환하거나 그 반대 연산.

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: 회문수 찾기

2024보다 큰 가장 작은 회문수는?

- **전략:** 2024는 4자리 수이므로, 다음 회문수도 4자리일 가능성이 높습니다. 형태는 $ABBA$.
- 천의 자리가 2이므로 $2BB2$ 형태를 찾습니다.
- $2002$는 2024보다 작습니다.
- 가운데 숫자 $B$를 1 증가시킵니다: $2112$.
- $2112 > 2024$.
- **정답:** 2112.

### 문제 2: 치킨 맥너겟 정리

5센트와 8센트 우표만 사용하여 만들 수 **없는** 가장 큰 우표 금액은?

- **조건:** $GCD(5, 8) = 1$ (서로소).
- **공식:** $mn - m - n$.
- **계산:** $5 \times 8 - 5 - 8 = 40 - 13 = 27$.
- **정답:** 27센트.

### 문제 3: 정의된 연산

연산 $\diamond$를 $a \diamond b = a^2 - 2b$라고 정의할 때, $(4 \diamond 3) \diamond 5$의 값은?

1.  괄호 안 계산: $4 \diamond 3 = 4^2 - 2(3) = 16 - 6 = 10$.
2.  전체 계산: $10 \diamond 5 = 10^2 - 2(5) = 100 - 10 = 90$.

- **정답:** 90.

### 문제 4: 진법 변환

$213_4$를 10진법으로 나타내면?

- 자릿수 값: $4^2, 4^1, 4^0$.
- 계산: $2 \times 4^2 + 1 \times 4^1 + 3 \times 4^0 = 2(16) + 4 + 3 = 32 + 4 + 3 = 39$.
- **정답:** 39.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 위 문제들을 해결하는 함수입니다.

```python
import math

def find_next_palindrome(n):
    """n보다 큰 가장 작은 회문수 찾기"""
    curr = n + 1
    while True:
        s = str(curr)
        if s == s[::-1]:
            return curr
        curr += 1

def chicken_mcnugget_limit(a, b):
    """ax + by로 만들 수 없는 가장 큰 수 (a, b 서로소)"""
    if math.gcd(a, b) != 1:
        return None
    return a * b - a - b

def defined_operation(a, b):
    """a <> b = a^2 - 2b"""
    return a**2 - 2*b

def base_to_decimal(num_str, base):
    """base진법 문자열을 10진법 정수로 변환"""
    return int(num_str, base)

# 테스트
print(f"Next palindrome after 2024: {find_next_palindrome(2024)}") # Expected: 2112
print(f"Largest impossible sum (5, 8): {chicken_mcnugget_limit(5, 8)}") # Expected: 27
print(f"(4 <> 3) <> 5: {defined_operation(defined_operation(4, 3), 5)}") # Expected: 90
print(f"213_4 to decimal: {base_to_decimal('213', 4)}") # Expected: 39
```

## 5. 요약 (Summary)

- **회문수:** 대칭성을 이용하여 숫자를 구성합니다.
- **맥너겟 정리:** $ab - a - b$ 공식을 기억합니다.
- **정의된 연산:** 기호의 정의에 따라 순서대로 대입합니다.
- **진법:** 위치값(Place Value)을 이용하여 변환합니다.

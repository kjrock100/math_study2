# 22.3 정수 연산 (Integer Operations)

## 1. 개요 (Overview)

이 장에서는 정수의 사칙연산(덧셈, 뺄셈, 곱셈, 나눗셈)과 관련된 기본 규칙인 **연산 순서(Order of Operations)**, **연산 법칙(Properties)**, 그리고 경시대회에서 자주 등장하는 **새롭게 정의된 연산(Defined Operations)**을 다룹니다.

## 2. 주요 개념 (Key Concepts)

### 2.1 연산 순서 (PEMDAS)

복잡한 식을 계산할 때는 약속된 순서를 따라야 합니다.

1.  **P**arentheses (괄호): 안쪽부터 계산.
2.  **E**xponents (지수): 거듭제곱 계산.
3.  **M**ultiplication / **D**ivision (곱셈/나눗셈): 왼쪽에서 오른쪽으로.
4.  **A**ddition / **S**ubtraction (덧셈/뺄셈): 왼쪽에서 오른쪽으로.

### 2.2 연산 법칙 (Properties)

- **교환법칙 (Commutative):** $a + b = b + a$, $a \times b = b \times a$.
- **결합법칙 (Associative):** $(a + b) + c = a + (b + c)$.
- **분배법칙 (Distributive):** $a(b + c) = ab + ac$. (암산에 매우 유용)

### 2.3 정의된 연산 (Defined Operations)

문제에서 특수 기호($\star, \#, \diamond$ 등)를 사용하여 새로운 연산을 정의하는 경우입니다.

- 예: $a \star b = 2a - b$.
- 전략: 정의된 식에 숫자를 **그대로 대입**하여 계산합니다.

## 3. 예제 (Examples)

### 예제 1: 연산 순서

$10 - 2 \times (3 + 1)^2 \div 4$ 의 값은?

1.  괄호: $3+1=4$. 식: $10 - 2 \times 4^2 \div 4$.
2.  지수: $4^2=16$. 식: $10 - 2 \times 16 \div 4$.
3.  곱셈/나눗셈 (왼쪽부터): $2 \times 16 = 32$, $32 \div 4 = 8$. 식: $10 - 8$.
4.  뺄셈: $2$.

- **정답:** 2.

### 예제 2: 정의된 연산

연산 $\diamond$를 $a \diamond b = a^2 - b$로 정의할 때, $(3 \diamond 5) \diamond 2$의 값은?

1.  괄호 안 먼저: $3 \diamond 5 = 3^2 - 5 = 9 - 5 = 4$.
2.  다음 계산: $4 \diamond 2 = 4^2 - 2 = 16 - 2 = 14$.

- **정답:** 14.

### 예제 3: 분배법칙 활용

$47 \times 99$를 쉽게 계산하시오.

- $47 \times (100 - 1) = 4700 - 47 = 4653$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 정의된 연산을 함수로 구현하고, 연산 순서를 확인하는 코드입니다.

```python
def custom_operation(a, b):
    """
    정의된 연산: a <> b = a^2 - b
    """
    return a**2 - b

def solve_expression():
    # 식: 10 - 2 * (3 + 1)^2 / 4
    # Python의 연산자 우선순위는 수학적 PEMDAS와 동일함
    result = 10 - 2 * (3 + 1)**2 / 4
    return result

# 테스트
# (3 <> 5) <> 2
step1 = custom_operation(3, 5) # 3^2 - 5 = 4
step2 = custom_operation(step1, 2) # 4^2 - 2 = 14

print(f"(3 <> 5) <> 2 = {step2}") # Expected: 14
print(f"10 - 2 * (3 + 1)^2 / 4 = {solve_expression()}") # Expected: 2.0
```

## 5. 요약 (Summary)

- **순서:** 괄호 $\rightarrow$ 지수 $\rightarrow$ 곱/나 $\rightarrow$ 덧/뺄.
- **정의된 연산:** 기호에 현혹되지 말고 주어진 식에 대입한다.
- **분배법칙:** 복잡한 곱셈을 덧셈/뺄셈으로 분해하여 계산한다.

# 17.2 배수 판정법 (Divisibility Rules)

## 1. 개요 (Overview)

배수 판정법(Divisibility Rules)은 어떤 정수가 특정 수로 나누어 떨어지는지(나머지가 0인지)를 나눗셈을 직접 하지 않고 빠르게 확인하는 방법입니다. 소인수분해나 약분, 최대공약수/최소공배수 계산 시 매우 유용하게 사용됩니다.

## 2. 주요 배수 판정법 (Key Divisibility Rules)

### 2.1 마지막 자리를 보는 경우

- **2의 배수:** 일의 자리 숫자가 짝수 ($0, 2, 4, 6, 8$)인 수.
- **5의 배수:** 일의 자리 숫자가 $0$ 또는 $5$인 수.
- **10의 배수:** 일의 자리 숫자가 $0$인 수.

### 2.2 자릿수의 합을 보는 경우

- **3의 배수:** 각 자리 숫자의 합이 3의 배수인 수.
  - 예: $123 \rightarrow 1+2+3=6$ (3의 배수 O).
- **9의 배수:** 각 자리 숫자의 합이 9의 배수인 수.
  - 예: $729 \rightarrow 7+2+9=18$ (9의 배수 O).

### 2.3 마지막 여러 자리를 보는 경우

- **4의 배수:** 끝 두 자리 수가 $00$이거나 4의 배수인 수.
  - 예: $1024 \rightarrow 24$는 4의 배수.
- **8의 배수:** 끝 세 자리 수가 $000$이거나 8의 배수인 수.
  - 예: $1008 \rightarrow 008$은 8의 배수.

### 2.4 합성수의 판정법

- **6의 배수:** 2의 배수이면서 동시에 3의 배수인 수. (짝수이고 자릿수 합이 3의 배수)
- **12의 배수:** 3의 배수이면서 동시에 4의 배수인 수.

## 3. 특수한 판정법 (Special Rules)

### 3.1 11의 배수

홀수 번째 자릿수의 합과 짝수 번째 자릿수의 합의 차이가 0이거나 11의 배수인 수.

- 예: $1331 \rightarrow (1+3) - (3+1) = 0$ (11의 배수 O).
- 예: $91828 \rightarrow (9+8+8) - (1+2) = 25 - 3 = 22$ (11의 배수 O).

### 3.2 7의 배수

일의 자리를 두 배 하여 나머지 수에서 뺀 결과가 7의 배수인 수. (수가 작아질 때까지 반복)

- 예: $343 \rightarrow 34 - (3 \times 2) = 28$ (7의 배수 O).

## 4. 예제 (Examples)

### 예제 1: 3과 9의 배수 판정

$456$은 3의 배수인가? 9의 배수인가?

- 자릿수 합: $4+5+6 = 15$.
- 15는 3의 배수이므로 $456$은 **3의 배수**.
- 15는 9의 배수가 아니므로 $456$은 **9의 배수가 아님**.

### 예제 2: 4의 배수 판정

$2022$년은 윤년(4의 배수)인가?

- 끝 두 자리: $22$.
- $22$는 4로 나누어 떨어지지 않으므로 **아님**. ($2024$년은 $24$가 4의 배수이므로 맞음)

### 예제 3: 11의 배수 판정

$2728$은 11의 배수인가?

- 홀수 자리 합: $8 + 7 = 15$.
- 짝수 자리 합: $2 + 2 = 4$.
- 차이: $15 - 4 = 11$.
- **정답:** 11의 배수 맞음.

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 주요 배수 판정법을 구현한 함수입니다.

```python
def check_divisibility(n):
    """
    정수 n에 대해 2~11까지의 배수 여부를 딕셔너리로 반환
    """
    results = {}
    s_n = str(n)
    digits = [int(d) for d in s_n]

    # 2: 짝수
    results[2] = (n % 2 == 0)

    # 3: 자릿수 합
    results[3] = (sum(digits) % 3 == 0)

    # 4: 끝 두 자리
    if len(s_n) >= 2:
        last_two = int(s_n[-2:])
    else:
        last_two = n
    results[4] = (last_two % 4 == 0)

    # 5: 끝자리 0 or 5
    results[5] = (s_n[-1] in '05')

    # 6: 2와 3의 배수
    results[6] = (results[2] and results[3])

    # 8: 끝 세 자리
    if len(s_n) >= 3:
        last_three = int(s_n[-3:])
    else:
        last_three = n
    results[8] = (last_three % 8 == 0)

    # 9: 자릿수 합
    results[9] = (sum(digits) % 9 == 0)

    # 10: 끝자리 0
    results[10] = (s_n[-1] == '0')

    # 11: 교대 합
    odd_sum = sum(digits[-1::-2])  # 일의 자리부터 홀수 번째 (인덱스 역순)
    even_sum = sum(digits[-2::-2]) # 십의 자리부터 짝수 번째
    results[11] = ((odd_sum - even_sum) % 11 == 0)

    return results

# 테스트
num = 2728
div_checks = check_divisibility(num)
print(f"Divisibility for {num}:")
for k, v in div_checks.items():
    if v:
        print(f"- Divisible by {k}")
# Expected for 2728: 2, 4, 8, 11 (and others if applicable)
# 2728 / 8 = 341. 2728 / 11 = 248.
```

## 6. 요약 (Summary)

- **자릿수 합:** 3, 9.
- **끝자리:** 2, 4, 5, 8, 10.
- **교대 합:** 11.
- **조합:** 6 ($2 \times 3$), 12 ($3 \times 4$).

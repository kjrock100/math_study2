# 17.1 소수 (Primes)

## 1. 개요 (Overview)

소수(Prime Number)는 정수론(Number Theory)의 가장 기초가 되는 구성 요소(Building Blocks)입니다. AMC 8에서는 소수의 정의, 소인수분해(Prime Factorization), 그리고 약수의 개수와 관련된 문제가 매우 빈번하게 출제됩니다.

## 2. 핵심 개념 (Core Concepts)

### 2.1 정의 (Definition)

- **소수 (Prime Number):** 1보다 큰 자연수 중, 1과 자기 자신만을 약수로 가지는 수. (약수가 정확히 2개)
  - 예: $2, 3, 5, 7, 11, 13, 17, 19, \dots$
- **합성수 (Composite Number):** 1보다 큰 자연수 중 소수가 아닌 수. (약수가 3개 이상)
  - 예: $4, 6, 8, 9, 10, \dots$
- **1의 성질:** 1은 소수도 아니고 합성수도 아닙니다. (Unit)

### 2.2 소수의 성질

- **2:** 유일한 **짝수** 소수입니다. 나머지 모든 소수는 홀수입니다.
- **에라토스테네스의 체 (Sieve of Eratosthenes):** 특정 범위 내의 소수를 찾는 효율적인 방법입니다. 2의 배수 지우고, 3의 배수 지우고... 를 반복합니다.

## 3. 소인수분해 (Prime Factorization)

### 3.1 산술의 기본 정리 (Fundamental Theorem of Arithmetic)

1보다 큰 모든 자연수는 소수들의 곱으로 유일하게 표현될 수 있습니다. (곱하는 순서는 무시)
$$ N = p_1^{e_1} \times p_2^{e_2} \times \dots \times p_k^{e_k} $$

### 3.2 활용

- **약수의 개수:** $(e_1 + 1)(e_2 + 1)\dots(e_k + 1)$
- **약수의 합:** $\frac{p_1^{e_1+1}-1}{p_1-1} \times \dots$

## 4. 예제 (Examples)

### 예제 1: 소수 판별

다음 중 소수인 것은? $1, 2, 9, 15, 27, 31$

- 1: 아님.
- 2: 소수 (약수 1, 2).
- 9: $3 \times 3$ (합성수).
- 15: $3 \times 5$ (합성수).
- 27: $3^3$ (합성수).
- 31: 소수.
- **정답:** 2, 31.

### 예제 2: 소인수분해

$72$를 소인수분해하시오.

- $72 = 8 \times 9 = 2^3 \times 3^2$.

### 예제 3: 약수의 개수

$72$의 약수의 개수는?

- $72 = 2^3 \times 3^2$.
- 지수에 1씩 더해서 곱함: $(3+1) \times (2+1) = 4 \times 3 = 12$개.

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 소수 판별과 소인수분해를 수행하는 함수입니다.

```python
def is_prime(n):
    """소수 판별 함수"""
    if n <= 1:
        return False
    if n == 2:
        return True
    if n % 2 == 0:
        return False

    # 제곱근까지만 확인하면 됨
    for i in range(3, int(n**0.5) + 1, 2):
        if n % i == 0:
            return False
    return True

def get_prime_factors(n):
    """소인수분해 함수: {소인수: 지수} 형태 반환"""
    factors = {}
    d = 2
    temp = n
    while d * d <= temp:
        while temp % d == 0:
            factors[d] = factors.get(d, 0) + 1
            temp //= d
        d += 1
    if temp > 1:
        factors[temp] = factors.get(temp, 0) + 1
    return factors

# 테스트
print(f"Is 31 prime? {is_prime(31)}") # Expected: True
print(f"Is 91 prime? {is_prime(91)}") # Expected: False (7 * 13)

num = 72
factors = get_prime_factors(num)
print(f"Prime factors of {num}: {factors}") # Expected: {2: 3, 3: 2}
```

## 6. 요약 (Summary)

- **소수:** 약수가 1과 자기 자신뿐인 수. (2는 유일한 짝수 소수)
- **소인수분해:** 수를 소수들의 곱으로 분해하는 것. 약수 문제의 핵심.
- **판별법:** $\sqrt{n}$까지만 나누어 떨어지는지 확인하면 된다.

# 17.5 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 17.1장부터 17.4장까지 학습한 정수론(Number Theory)의 핵심 개념들을 종합하여 실전 문제를 해결합니다. 소수 판별, 배수 판정법, 소인수분해, 약수의 개수, 그리고 팩토리얼의 끝자리 0의 개수(Trailing Zeros) 등 다양한 유형의 문제가 출제됩니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **약수 응용 (Divisor Applications):** 약수의 개수가 주어졌을 때 가장 작은 수 찾기, 약수의 합 구하기.
2.  **배수 판정법 활용 (Divisibility Rules):** 미지수가 포함된 수에서 특정 수의 배수가 되도록 하는 숫자 찾기.
3.  **팩토리얼과 0의 개수 (Factorials & Trailing Zeros):** 르장드르 공식을 이용한 $n!$의 소인수 지수 계산.
4.  **최대공약수와 최소공배수 (GCD & LCM):** 문장제 문제에서 주기성이나 공통된 나눔수 찾기.

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: 약수의 개수 역추적

약수의 개수가 정확히 6개인 가장 작은 자연수는?

- **전략:** 약수의 개수 공식 $d(n) = (e_1+1)(e_2+1)\dots$을 이용합니다.
- 6이 되는 경우:
  1.  $6$ (소인수 1개): $p^5$ 형태. 가장 작은 수는 $2^5 = 32$.
  2.  $3 \times 2$ (소인수 2개): $p_1^2 \times p_2^1$ 형태. 가장 작은 소수부터 배치하면 $2^2 \times 3^1 = 4 \times 3 = 12$.
- **비교:** $12 < 32$.
- **정답:** 12.

### 문제 2: 배수 판정법 (8의 배수)

5자리 수 $12,34d$가 8의 배수일 때, 한 자리 정수 $d$의 값은?

- **조건:** 8의 배수는 끝 세 자리($34d$)가 8로 나누어 떨어져야 합니다.
- **확인:**
  - $340 \div 8 = 42.5$ (X)
  - $344 \div 8 = 43$ (O)
  - $348 \div 8 = 43.5$ (X)
  - $340 + d$ 형태이므로 $d$는 0~9 사이.
- **정답:** $d = 4$.

### 문제 3: 팩토리얼의 0의 개수

$50!$의 값은 끝에 0이 몇 개 있는가?

- **전략:** 소인수 5의 개수를 셉니다 (르장드르 공식).
- **계산:**
  $$ \left\lfloor \frac{50}{5} \right\rfloor + \left\lfloor \frac{50}{25} \right\rfloor = 10 + 2 = 12 $$
- **정답:** 12개.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 약수의 개수 역추적, 배수 판정, 팩토리얼 0의 개수 문제를 검증하는 함수입니다.

```python
def find_smallest_number_with_n_divisors(target_divisors):
    """
    약수의 개수가 target_divisors인 가장 작은 수 찾기 (Brute-force for small range)
    """
    n = 1
    while True:
        # 약수 개수 계산
        count = 0
        for i in range(1, int(n**0.5) + 1):
            if n % i == 0:
                count += 1
                if i*i != n:
                    count += 1

        if count == target_divisors:
            return n
        n += 1
        if n > 10000: # 안전장치
            return None

def solve_missing_digit_divisibility(prefix, divisor):
    """
    prefix + d 형태의 수가 divisor로 나누어 떨어지는 d 찾기
    prefix: 예) 1234 (문자열 또는 정수)
    """
    solutions = []
    base = int(str(prefix) + '0')
    for d in range(10):
        num = base + d
        if num % divisor == 0:
            solutions.append(d)
    return solutions

def count_trailing_zeros_factorial(n):
    count = 0
    i = 5
    while n // i > 0:
        count += n // i
        i *= 5
    return count

# 테스트
print(f"Smallest number with 6 divisors: {find_smallest_number_with_n_divisors(6)}") # Expected: 12
print(f"Digit d for 1234d divisible by 8: {solve_missing_digit_divisibility(1234, 8)}") # Expected: [4] (12344)
print(f"Trailing zeros in 50!: {count_trailing_zeros_factorial(50)}") # Expected: 12
```

## 5. 요약 (Summary)

- **약수 문제:** 소인수분해 형태($p^e$)로 변환하여 지수 조합을 생각합니다.
- **배수 판정:** 끝자리(2,4,5,8)나 자릿수 합(3,9) 규칙을 적절히 사용합니다.
- **Trailing Zeros:** 5의 거듭제곱으로 계속 나누어 몫을 더합니다.

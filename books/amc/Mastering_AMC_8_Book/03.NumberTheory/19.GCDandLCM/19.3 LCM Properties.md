# 19.3 GCD와 LCM의 심화 성질 (More GCD/LCM Properties)

## 1. 개요 (Overview)

이 장에서는 최대공약수(GCD)와 최소공배수(LCM)의 정의를 넘어, 복잡한 수나 분수 형태를 다룰 때 유용한 심화 성질들을 학습합니다. 특히 **유클리드 호제법의 뺄셈 원리**와 **분수의 LCM** 개념은 주기성 문제나 대수적 약수 문제에서 매우 강력한 도구가 됩니다.

## 2. 주요 성질 (Key Properties)

### 2.1 뺄셈 성질 (Euclidean Property)

두 수의 최대공약수는 큰 수에서 작은 수를 뺀 값과 작은 수의 최대공약수와 같습니다.
$$ GCD(a, b) = GCD(a - b, b) \quad (\text{단, } a > b) $$

- **응용:** $GCD(n, n+1) = GCD(1, n) = 1$. (연속하는 두 자연수는 항상 서로소)
- **심화:** $GCD(a, b) = GCD(a, b \pmod a)$. (나머지 연산)

### 2.2 배수 성질 (Scaling Property)

두 수에 같은 수 $k$를 곱하면, GCD와 LCM도 $k$배가 됩니다.
$$ GCD(ka, kb) = k \times GCD(a, b) $$
$$ LCM(ka, kb) = k \times LCM(a, b) $$

### 2.3 분수의 GCD와 LCM (Fractions)

분수 $\frac{a}{b}$와 $\frac{c}{d}$가 기약분수일 때:

- **분수의 LCM (최소공배수):** 두 분수의 배수 중 가장 작은 정수(또는 분수). "분자들의 LCM / 분모들의 GCD".
  $$ LCM\left(\frac{a}{b}, \frac{c}{d}\right) = \frac{LCM(a, c)}{GCD(b, d)} $$

  - 예: 1.5초($\frac{3}{2}$)와 2초($\frac{2}{1}$)마다 깜빡이는 전구의 동시 점등 주기.

- **분수의 GCD (최대공약수):** 두 분수를 나누어 떨어지게 하는 가장 큰 분수. "분자들의 GCD / 분모들의 LCM".
  $$ GCD\left(\frac{a}{b}, \frac{c}{d}\right) = \frac{GCD(a, c)}{LCM(b, d)} $$

## 3. 예제 (Examples)

### 예제 1: 큰 수의 GCD

$GCD(1002, 1000)$을 구하시오.

- 성질 이용: $GCD(1002, 1000) = GCD(1002 - 1000, 1000) = GCD(2, 1000)$.
- $1000$은 2의 배수이므로, 정답은 **2**.

### 예제 2: 대수식의 GCD

$GCD(2n + 3, n + 1)$은?

- $GCD(2n + 3, n + 1) = GCD((2n + 3) - 2(n + 1), n + 1)$
- $= GCD(2n + 3 - 2n - 2, n + 1) = GCD(1, n + 1) = 1$.
- 따라서 $n$에 상관없이 항상 서로소입니다.

### 예제 3: 분수의 LCM

$\frac{2}{3}$분과 $\frac{4}{5}$분마다 울리는 알람이 있다. 동시에 울리는 간격은?

- $LCM(\frac{2}{3}, \frac{4}{5}) = \frac{LCM(2, 4)}{GCD(3, 5)}$.
- 분자 LCM: $LCM(2, 4) = 4$.
- 분모 GCD: $GCD(3, 5) = 1$.
- 결과: $\frac{4}{1} = 4$분.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 분수의 GCD와 LCM을 계산하는 함수입니다.

```python
import math
from fractions import Fraction

def lcm(a, b):
    """정수 LCM 계산"""
    if a == 0 or b == 0: return 0
    return abs(a * b) // math.gcd(a, b)

def fraction_lcm(f1, f2):
    """
    분수의 LCM = LCM(분자) / GCD(분모)
    입력: fractions.Fraction 객체
    """
    num = lcm(f1.numerator, f2.numerator)
    den = math.gcd(f1.denominator, f2.denominator)
    return Fraction(num, den)

def fraction_gcd(f1, f2):
    """
    분수의 GCD = GCD(분자) / LCM(분모)
    입력: fractions.Fraction 객체
    """
    num = math.gcd(f1.numerator, f2.numerator)
    den = lcm(f1.denominator, f2.denominator)
    return Fraction(num, den)

# 테스트
frac1 = Fraction(2, 3)
frac2 = Fraction(4, 5)

print(f"Numbers: {frac1}, {frac2}")
print(f"Fraction LCM: {fraction_lcm(frac1, frac2)}") # Expected: 4/1 = 4
print(f"Fraction GCD: {fraction_gcd(frac1, frac2)}") # Expected: 2/15
```

## 5. 요약 (Summary)

- **뺄셈 성질:** $GCD(a, b) = GCD(a-b, b)$. 큰 수를 줄일 때 유용함.
- **분수 LCM:** $\frac{LCM(\text{Top})}{GCD(\text{Bottom})}$. 주기성 문제의 핵심.
- **분수 GCD:** $\frac{GCD(\text{Top})}{LCM(\text{Bottom})}$. 공통 단위(Unit)를 찾을 때 사용.

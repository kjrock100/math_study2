# 19.2 최대공약수와 최소공배수의 곱 (GCD and LCM Product)

## 1. 개요 (Overview)

두 자연수 $A, B$와 그들의 최대공약수 $G$, 최소공배수 $L$ 사이에는 $A \times B = G \times L$이라는 매우 중요한 관계가 성립합니다. 이 성질은 단순히 $L$을 구하는 것을 넘어, $G$와 $L$이 주어졌을 때 원래의 두 수 $A, B$를 추론하는 문제(역추적)를 해결하는 핵심 열쇠가 됩니다.

## 2. 핵심 공식 및 원리 (Formula & Principle)

### 2.1 곱의 관계식

$$ A \times B = GCD(A, B) \times LCM(A, B) $$

### 2.2 구조적 해석

$A = G \times x$, $B = G \times y$ 라고 할 때 (단, $x, y$는 **서로소**),

- **최대공약수 (GCD):** $G$
- **최소공배수 (LCM):** $L = G \times x \times y$
- **두 수의 곱:** $A \times B = (G \times x) \times (G \times y) = G \times (G \times x \times y) = G \times L$

이 구조를 이해하면 $L$과 $G$가 주어졌을 때 $x, y$를 구할 수 있습니다.
$$ x \times y = \frac{L}{G} $$

## 3. 두 수 찾기 전략 (Finding the Numbers)

$G$와 $L$이 주어졌을 때 가능한 $(A, B)$ 쌍을 찾는 방법입니다.

1.  **곱 구하기:** $k = x \times y = L / G$를 계산합니다.
2.  **인수분해:** $k$를 두 수의 곱으로 나타냅니다.
3.  **서로소 조건 확인:** 두 수가 서로소(Coprime)인 경우만 선택합니다. ($GCD(x, y) = 1$)
4.  **복원:** 구한 $x, y$에 $G$를 곱하여 $A, B$를 구합니다.

## 4. 예제 (Examples)

### 예제 1: 기본 확인

두 수의 곱이 432이고 최대공약수가 6일 때, 최소공배수는?

- $LCM = \frac{\text{Product}}{GCD} = \frac{432}{6} = 72$.

### 예제 2: 두 수 찾기 (심화)

최대공약수가 6이고 최소공배수가 72인 두 자연수 $A, B$ ($A < B$)를 모두 구하시오.

1.  $G=6, L=72$.
2.  $x \times y = L / G = 72 / 6 = 12$.
3.  곱해서 12가 되는 $(x, y)$ 쌍 ($x < y$):
    - $(1, 12)$: 서로소 O. $\rightarrow A=6 \times 1 = 6, B=6 \times 12 = 72$.
    - $(2, 6)$: 서로소 X ($GCD(2,6)=2$). 제외.
    - $(3, 4)$: 서로소 O. $\rightarrow A=6 \times 3 = 18, B=6 \times 4 = 24$.
4.  **정답:** $(6, 72)$ 또는 $(18, 24)$.

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 GCD와 LCM이 주어졌을 때 가능한 두 수의 쌍을 모두 찾는 함수입니다.

```python
import math

def find_pairs_from_gcd_lcm(gcd_val, lcm_val):
    """
    주어진 GCD와 LCM을 만족하는 모든 (A, B) 쌍을 찾음 (A <= B)
    """
    if lcm_val % gcd_val != 0:
        return [] # LCM은 GCD의 배수여야 함

    product_xy = lcm_val // gcd_val
    pairs = []

    # product_xy의 약수 쌍 (x, y) 찾기
    for x in range(1, int(product_xy**0.5) + 1):
        if product_xy % x == 0:
            y = product_xy // x

            # x와 y가 서로소인지 확인
            if math.gcd(x, y) == 1:
                a = gcd_val * x
                b = gcd_val * y
                pairs.append((a, b))

    return pairs

# 테스트
g, l = 6, 72
result_pairs = find_pairs_from_gcd_lcm(g, l)
print(f"Pairs with GCD={g}, LCM={l}: {result_pairs}")
# Expected: [(6, 72), (18, 24)]
```

## 6. 요약 (Summary)

- **공식:** $A \times B = G \times L$.
- **핵심:** $L = G \times x \times y$ (단, $x, y$는 서로소).
- **전략:** $L/G$를 서로소인 두 수의 곱으로 분해하여 가능한 숫자 쌍을 찾는다.

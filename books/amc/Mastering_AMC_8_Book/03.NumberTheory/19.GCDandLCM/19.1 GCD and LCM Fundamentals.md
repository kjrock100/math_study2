# 19.1 최대공약수와 최소공배수의 기초 (GCD and LCM Fundamentals)

## 1. 개요 (Overview)

정수론에서 가장 기본적이면서도 중요한 두 개념인 **최대공약수(GCD, Greatest Common Divisor)**와 **최소공배수(LCM, Least Common Multiple)**를 다룹니다. 이들은 분수의 통분, 약분, 그리고 주기적인 현상을 다루는 문제에서 필수적인 도구입니다.

## 2. 정의 및 계산 방법 (Definitions & Methods)

### 2.1 최대공약수 (GCD)

두 개 이상의 자연수의 공통된 약수 중 가장 큰 수입니다.

- **소인수분해 이용:** 공통인 소인수를 찾고, 그 중 **지수가 작은 것**을 선택하여 곱합니다.
- **유클리드 호제법 (Euclidean Algorithm):** 큰 수에서 작은 수를 나눈 나머지를 이용하여 반복적으로 계산합니다.
  $$ GCD(a, b) = GCD(b, a \pmod b) $$

### 2.2 최소공배수 (LCM)

두 개 이상의 자연수의 공통된 배수 중 가장 작은 수입니다.

- **소인수분해 이용:** 모든 소인수를 찾고, 그 중 **지수가 큰 것**을 선택하여 곱합니다.

## 3. 관계식 (Relationship)

두 자연수 $A, B$에 대하여 다음 관계가 성립합니다.
$$ A \times B = GCD(A, B) \times LCM(A, B) $$
즉, 두 수의 곱은 최대공약수와 최소공배수의 곱과 같습니다. 이를 이용하면 $LCM(A, B) = \frac{A \times B}{GCD(A, B)}$로 쉽게 구할 수 있습니다.

## 4. 예제 (Examples)

### 예제 1: 소인수분해 이용

$12$와 $18$의 GCD와 LCM을 구하시오.

- $12 = 2^2 \times 3^1$
- $18 = 2^1 \times 3^2$
- **GCD:** $2^1 \times 3^1 = 6$ (지수가 작은 쪽 선택)
- **LCM:** $2^2 \times 3^2 = 4 \times 9 = 36$ (지수가 큰 쪽 선택)

### 예제 2: 관계식 이용

두 수의 곱이 $240$이고 최대공약수가 $4$일 때, 최소공배수는?

- $LCM = \frac{\text{Product}}{GCD} = \frac{240}{4} = 60$.

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 GCD와 LCM을 계산하는 함수입니다. 유클리드 호제법을 직접 구현해 봅니다.

```python
import math

def euclidean_gcd(a, b):
    """
    유클리드 호제법을 이용한 최대공약수 계산
    GCD(a, b) = GCD(b, a % b)
    """
    while b != 0:
        a, b = b, a % b
    return a

def calculate_lcm(a, b):
    """
    GCD * LCM = a * b 관계를 이용한 최소공배수 계산
    """
    if a == 0 or b == 0:
        return 0
    return abs(a * b) // euclidean_gcd(a, b)

# 테스트
num1, num2 = 12, 18
gcd_val = euclidean_gcd(num1, num2)
lcm_val = calculate_lcm(num1, num2)

print(f"Numbers: {num1}, {num2}")
print(f"GCD: {gcd_val}") # Expected: 6
print(f"LCM: {lcm_val}") # Expected: 36

# Python 내장 함수와 비교
print(f"Built-in GCD: {math.gcd(num1, num2)}")
# Python 3.9+ 부터 math.lcm 사용 가능
if hasattr(math, 'lcm'):
    print(f"Built-in LCM: {math.lcm(num1, num2)}")
```

## 6. 요약 (Summary)

- **GCD:** 공통 소인수의 최소 지수 곱. 유클리드 호제법이 효율적.
- **LCM:** 모든 소인수의 최대 지수 곱.
- **공식:** $GCD \times LCM = A \times B$.

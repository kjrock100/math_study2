# 19.5 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 19.1장부터 19.4장까지 학습한 최대공약수(GCD)와 최소공배수(LCM)의 개념을 종합하여 실전 문장제 문제를 해결합니다. 단순히 GCD/LCM을 계산하는 것을 넘어, 문제 상황을 분석하여 **언제 GCD를 사용하고 언제 LCM을 사용해야 하는지** 판단하는 능력을 기릅니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **주기성 문제 (Periodicity Problems):** 버스가 동시에 출발하거나, 알람이 동시에 울리는 등, "다음에 다시 만나는" 시점을 묻는 문제는 **LCM**을 사용합니다.
2.  **분배/타일링 문제 (Distribution/Tiling Problems):** 사탕을 똑같이 나누어 주거나, 정사각형 타일로 직사각형 바닥을 채우는 등, "가장 큰 공통 단위"를 찾는 문제는 **GCD**를 사용합니다.
3.  **GCD/LCM 역추적 문제 (Reverse Problems):** GCD와 LCM이 주어졌을 때, 원래의 두 수를 찾는 문제입니다. ($A \times B = G \times L$ 활용)

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: 버스 문제 (LCM)

A 버스는 12분마다, B 버스는 15분마다 정류장을 출발한다. 오전 9시에 두 버스가 동시에 출발했다면, 다음에 처음으로 동시에 출발하는 시각은?

- **전략:** 12와 15의 최소공배수를 구합니다.
- $12 = 2^2 \times 3$
- $15 = 3 \times 5$
- $LCM(12, 15) = 2^2 \times 3 \times 5 = 60$.
- **정답:** 60분 후인 오전 10시.

### 문제 2: 타일링 문제 (GCD)

가로 48cm, 세로 60cm인 직사각형 모양의 벽을 남는 부분 없이 가장 큰 정사각형 타일로 덮으려고 한다. 필요한 타일의 개수는?

- **전략:** 타일의 한 변의 길이는 48과 60의 최대공약수입니다.
- $GCD(48, 60)$:
  - $48 = 12 \times 4$
  - $60 = 12 \times 5$
  - $GCD(48, 60) = 12$.
- 타일 한 변의 길이: 12cm.
- **필요한 타일 개수:**
  - 가로: $48 \div 12 = 4$개.
  - 세로: $60 \div 12 = 5$개.
  - 총 개수: $4 \times 5 = 20$개.

### 문제 3: 두 수 찾기 (GCD/LCM Product)

두 자연수의 최대공약수는 4이고 최소공배수는 48이다. 가능한 두 자연수의 쌍을 모두 구하시오.

- **전략:** $A=Gx, B=Gy$ (단, $x, y$는 서로소)이고 $L=Gxy$를 이용합니다.
- $G=4, L=48$.
- $xy = L/G = 48/4 = 12$.
- 곱해서 12가 되는 서로소 쌍 $(x, y)$ 찾기:
  - $(1, 12)$: 서로소 O.
  - $(2, 6)$: 서로소 X.
  - $(3, 4)$: 서로소 O.
- **복원:**
  - $(1, 12) \rightarrow A=4 \times 1=4, B=4 \times 12=48$.
  - $(3, 4) \rightarrow A=4 \times 3=12, B=4 \times 4=16$.
- **정답:** $(4, 48)$과 $(12, 16)$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 위 문제들을 해결하는 함수입니다.

```python
import math

def lcm(a, b):
    return abs(a * b) // math.gcd(a, b) if a != 0 and b != 0 else 0

def solve_tiling_problem(width, height):
    """
    타일링 문제 해결: 타일 한 변의 길이와 총 개수 반환
    """
    tile_side = math.gcd(width, height)
    num_tiles_width = width // tile_side
    num_tiles_height = height // tile_side
    total_tiles = num_tiles_width * num_tiles_height
    return tile_side, total_tiles

def find_pairs_from_gcd_lcm(g, l):
    """
    GCD와 LCM으로부터 가능한 (A, B) 쌍 찾기
    """
    if l % g != 0:
        return []

    xy = l // g
    pairs = []
    for x in range(1, int(xy**0.5) + 1):
        if xy % x == 0:
            y = xy // x
            if math.gcd(x, y) == 1:
                pairs.append((g * x, g * y))
    return pairs

# 테스트
print(f"문제 1 (Bus): Next meeting in {lcm(12, 15)} minutes.") # Expected: 60

tile_side, total_tiles = solve_tiling_problem(48, 60)
print(f"문제 2 (Tiling): Tile side = {tile_side}cm, Total tiles = {total_tiles}") # Expected: 12, 20

print(f"문제 3 (Find Pairs): {find_pairs_from_gcd_lcm(4, 48)}") # Expected: [(4, 48), (12, 16)]
```

## 5. 요약 (Summary)

- **"동시에", "다시 만나는"** $\rightarrow$ **LCM** (배수).
- **"똑같이 나누는", "가장 큰 정사각형"** $\rightarrow$ **GCD** (약수).
- **두 수 찾기:** $xy = L/G$를 풀고, $x, y$가 서로소인지 반드시 확인합니다.

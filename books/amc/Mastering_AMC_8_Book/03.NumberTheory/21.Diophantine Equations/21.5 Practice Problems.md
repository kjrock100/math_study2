# 21.5 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 21.1장부터 21.4장까지 학습한 다양한 인수분해 기법(이차식, SFFT, 세제곱 공식 등)을 종합하여 실전 문제를 해결합니다. 식의 값을 간단히 계산하거나, 복잡한 방정식을 정수 조건 하에서 푸는 능력을 기릅니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **식의 값 계산 (Evaluating Expressions):** 숫자를 직접 계산하는 대신 인수분해 공식을 사용하여 약분하거나 간단히 만듭니다.
2.  **부정방정식 (Diophantine Equations):** SFFT($xy+ax+by$)나 합차 공식($x^2-y^2$)을 이용하여 정수 해를 구합니다.
3.  **방정식의 해 (Solving Equations):** 고차 방정식을 인수분해하여 근을 찾습니다.

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: SFFT 활용

양의 정수 $m, n$에 대하여 $mn + 5m + 3n = 30$일 때, $m+n$의 값은?

- **전략:** $mn + 5m + 3n$ 꼴이므로 SFFT를 적용합니다.
- 양변에 $5 \times 3 = 15$를 더합니다.
  $$ mn + 5m + 3n + 15 = 30 + 15 $$
    $$ m(n+5) + 3(n+5) = 45 $$
    $$ (m+3)(n+5) = 45 $$
- $m, n$이 양의 정수이므로 $m+3 \ge 4$, $n+5 \ge 6$.
- 45의 약수 중 조건을 만족하는 쌍 $(A, B)$ 찾기 ($A \times B = 45$):
  - $(5, 9) \rightarrow m+3=5 \implies m=2, n+5=9 \implies n=4$.
  - $(9, 5) \rightarrow m+3=9 \implies m=6, n+5=5 \implies n=0$ (양의 정수 아님, 제외).
  - $(15, 3)$ 등은 $B \ge 6$ 조건 불만족.
- 유일한 해: $(m, n) = (2, 4)$.
- **정답:** $m+n = 6$.

### 문제 2: 세제곱 공식 활용

$\frac{20^3 + 30^3}{20^2 - 600 + 30^2}$ 의 값을 구하시오.

- **전략:** $a^3 + b^3 = (a+b)(a^2 - ab + b^2)$ 공식을 사용합니다.
- $a=20, b=30$이라 하면, 분모는 $a^2 - ab + b^2$ ($20 \times 30 = 600$).
- 식: $\frac{(a+b)(a^2 - ab + b^2)}{a^2 - ab + b^2} = a + b$.
- **정답:** $20 + 30 = 50$.

### 문제 3: 이차식의 활용

$x^2 - y^2 = 45$를 만족하는 자연수 $x, y$의 순서쌍 개수는?

- **전략:** $(x-y)(x+y) = 45$.
- $x, y$가 자연수이므로 $x+y > x-y$ 이고, 둘의 대소 관계와 홀짝성을 고려합니다.
- 곱해서 45가 되는 쌍 $(A, B)$ ($A < B$, $A=x-y, B=x+y$):
  1.  $1 \times 45$: $2x = 46 \implies x=23, y=22$. (가능)
  2.  $3 \times 15$: $2x = 18 \implies x=9, y=6$. (가능)
  3.  $5 \times 9$: $2x = 14 \implies x=7, y=2$. (가능)
- **정답:** 3개.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 SFFT 문제와 합차 공식 문제를 해결하는 함수입니다.

```python
def solve_sfft_practice(a, b, constant):
    """
    Solves mn + am + bn = constant for positive integers m, n
    Equation becomes: (m + b)(n + a) = constant + ab
    """
    target = constant + a * b
    solutions = []

    # Find factors of target
    for i in range(1, int(target**0.5) + 1):
        if target % i == 0:
            factor1 = i
            factor2 = target // i

            # Case 1: m+b = factor1, n+a = factor2
            m1 = factor1 - b
            n1 = factor2 - a
            if m1 > 0 and n1 > 0:
                solutions.append((m1, n1))

            # Case 2: m+b = factor2, n+a = factor1 (if different)
            if factor1 != factor2:
                m2 = factor2 - b
                n2 = factor1 - a
                if m2 > 0 and n2 > 0:
                    solutions.append((m2, n2))

    return solutions

def solve_difference_squares(target):
    """
    Solves x^2 - y^2 = target for natural numbers x, y
    (x-y)(x+y) = target
    """
    solutions = []
    for i in range(1, int(target**0.5) + 1):
        if target % i == 0:
            diff = i        # x - y
            summ = target // i # x + y

            # 2x = summ + diff
            if (summ + diff) % 2 == 0:
                x = (summ + diff) // 2
                y = (summ - diff) // 2
                if y > 0: # Natural numbers
                    solutions.append((x, y))
    return solutions

# 테스트
print(f"SFFT (mn + 5m + 3n = 30): {solve_sfft_practice(5, 3, 30)}")
# Expected: [(2, 4)] -> m=2, n=4 (2*4 + 10 + 12 = 8+22=30)

print(f"x^2 - y^2 = 45: {solve_difference_squares(45)}")
# Expected: [(23, 22), (9, 6), (7, 2)]
```

## 5. 요약 (Summary)

- **SFFT:** $xy+ax+by=c \rightarrow (x+b)(y+a) = c+ab$.
- **합차 공식:** $x^2-y^2=k \rightarrow (x-y)(x+y)=k$. (홀짝성 일치 확인)
- **세제곱 공식:** $a^3 \pm b^3$ 인수분해를 통해 식을 약분.

# 21.1 이차식의 인수분해 (Quadratic Factorizations)

## 1. 개요 (Overview)

인수분해(Factorization)는 전개(Expansion)의 역연산으로, 다항식을 두 개 이상의 다항식의 곱으로 나타내는 것입니다. 특히 이차식($ax^2 + bx + c$)의 인수분해는 이차방정식을 풀거나 대수적 식을 간단히 하는 데 필수적인 기술입니다.

## 2. 주요 공식 및 패턴 (Key Formulas & Patterns)

### 2.1 공통인수 묶기 (Common Factor)

가장 먼저 해야 할 단계입니다. 각 항에 공통된 인수가 있다면 묶어냅니다.
$$ ma + mb = m(a + b) $$

### 2.2 합차 공식 (Difference of Squares)

제곱의 차는 합과 차의 곱으로 인수분해됩니다.
$$ a^2 - b^2 = (a + b)(a - b) $$

### 2.3 완전제곱식 (Perfect Square Trinomials)

$$ a^2 + 2ab + b^2 = (a + b)^2 $$
$$ a^2 - 2ab + b^2 = (a - b)^2 $$

### 2.4 일반적인 이차식 (General Quadratics)

$x^2 + bx + c$ 꼴을 $(x + p)(x + q)$로 인수분해하려면 다음 조건을 만족하는 두 수 $p, q$를 찾아야 합니다.

1.  **합 (Sum):** $p + q = b$
2.  **곱 (Product):** $p \times q = c$

$ax^2 + bx + c$ ($a \neq 1$) 꼴은 '크로스 방법(Cross Method)'이나 $ac$ 방법(Grouping)을 사용합니다.

## 3. 예제 (Examples)

### 예제 1: 합차 공식

$x^2 - 16$을 인수분해하시오.

- $16 = 4^2$이므로 $x^2 - 4^2$.
- **정답:** $(x + 4)(x - 4)$.

### 예제 2: 합과 곱 찾기

$x^2 + 7x + 12$를 인수분해하시오.

- 곱해서 12, 더해서 7이 되는 두 수 찾기.
- $(1, 12) \rightarrow 13$
- $(2, 6) \rightarrow 8$
- $(3, 4) \rightarrow 7$ (찾음!)
- **정답:** $(x + 3)(x + 4)$.

### 예제 3: 계수가 있는 이차식

$2x^2 + 7x + 3$을 인수분해하시오.

- 곱해서 $2x^2$이 되는 식: $(2x, x)$.
- 곱해서 $3$이 되는 수: $(1, 3)$ 또는 $(3, 1)$.
- 크로스 확인:
  - $(2x + 1)(x + 3) = 2x^2 + 6x + x + 3 = 2x^2 + 7x + 3$. (성공)
- **정답:** $(2x + 1)(x + 3)$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 $x^2 + bx + c$ 형태의 이차식을 인수분해하는 함수입니다. (판별식을 활용하여 정수 계수 인수를 찾습니다.)

```python
import math

def factor_simple_quadratic(b, c):
    """
    x^2 + bx + c = (x + p)(x + q) 형태의 인수분해
    p, q를 반환 (p <= q)
    """
    # 판별식 D = b^2 - 4c
    D = b**2 - 4*c

    if D < 0:
        return None # 실수 범위에서 인수분해 불가

    sqrt_D = int(math.isqrt(D))
    if sqrt_D * sqrt_D != D:
        return None # 정수 범위에서 인수분해 불가 (유리수/무리수 해)

    # 근의 공식 변형을 통해 p, q 도출
    # (x+p)(x+q) = x^2 + (p+q)x + pq
    # p, q는 방정식 t^2 - bt + c = 0 의 해와 관련됨 (부호 주의)
    # 여기서는 p+q=b, pq=c를 만족하는 p, q를 찾음

    num1 = b - sqrt_D
    num2 = b + sqrt_D

    if num1 % 2 != 0 or num2 % 2 != 0:
        return None # 정수 계수 아님

    p = num1 // 2
    q = num2 // 2

    return p, q

# 테스트
print(f"x^2 + 7x + 12: {factor_simple_quadratic(7, 12)}") # Expected: (3, 4) -> (x+3)(x+4)
print(f"x^2 - 16 (b=0, c=-16): {factor_simple_quadratic(0, -16)}") # Expected: (-4, 4) -> (x-4)(x+4)
print(f"x^2 + 2x + 1: {factor_simple_quadratic(2, 1)}") # Expected: (1, 1) -> (x+1)^2
```

## 5. 요약 (Summary)

- **공통인수:** 가장 먼저 확인한다.
- **공식:** 합차($a^2-b^2$), 완전제곱($a^2 \pm 2ab + b^2$) 패턴을 암기한다.
- **일반형:** 합과 곱이 되는 두 수를 찾는다.

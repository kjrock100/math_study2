# 21.3 흥미로운 예제들 (Interesting Examples)

## 1. 개요 (Overview)

이 장에서는 앞서 배운 인수분해 공식(합차 공식, 완전제곱식, SFFT 등)을 활용하여 복잡한 수의 계산이나 대수적 식의 값을 구하는 창의적인 문제들을 다룹니다. 숫자를 문자로 치환하여 식을 간단히 하거나, 식의 구조를 파악하는 것이 핵심입니다.

## 2. 주요 예제 (Key Examples)

### 예제 1: 수의 계산 (Arithmetic using Algebra)

$20^2 - 19^2 + 18^2 - 17^2 + \dots + 2^2 - 1^2$ 의 값을 구하시오.

- **전략:** 합차 공식 $a^2 - b^2 = (a-b)(a+b)$를 두 항씩 묶어서 적용합니다.
- $(20-19)(20+19) + (18-17)(18+17) + \dots + (2-1)(2+1)$
- $= 1 \times 39 + 1 \times 35 + \dots + 1 \times 3$
- $= 39 + 35 + 31 + \dots + 3$.
- 이는 등차수열의 합입니다. 항의 개수는 10개.
- 합: $\frac{10(39 + 3)}{2} = 5 \times 42 = 210$.

### 예제 2: 식의 값 구하기 (Algebraic Manipulation)

$x + \frac{1}{x} = 5$ 일 때, $x^2 + \frac{1}{x^2}$ 의 값은?

- **전략:** 양변을 제곱합니다.
- $(x + \frac{1}{x})^2 = x^2 + 2(x)(\frac{1}{x}) + \frac{1}{x^2} = x^2 + 2 + \frac{1}{x^2}$.
- $5^2 = 25$.
- $x^2 + 2 + \frac{1}{x^2} = 25 \implies x^2 + \frac{1}{x^2} = 23$.

### 예제 3: 소수와 제곱의 차 (Primes and Squares)

$x^2 - y^2 = 31$ 을 만족하는 자연수 $x, y$를 구하시오.

- **전략:** 좌변을 인수분해합니다.
- $(x - y)(x + y) = 31$.
- 31은 소수이므로 약수는 1과 31뿐입니다.
- $x, y$가 자연수이므로 $x+y > x-y$.
- $x - y = 1$, $x + y = 31$.
- 두 식을 더하면 $2x = 32 \implies x = 16$.
- $y = 15$.

## 3. 코드 구현 (Code Implementation)

Python을 사용하여 위와 같은 유형의 문제를 해결하는 함수입니다.

```python
def solve_alternating_squares(n):
    """
    Calculates n^2 - (n-1)^2 + ... + 2^2 - 1^2
    Assumes n is even.
    """
    total = 0
    for i in range(n, 0, -2):
        # i^2 - (i-1)^2 = (i - (i-1)) * (i + i - 1) = 1 * (2i - 1)
        term = (i**2) - ((i-1)**2)
        total += term
    return total

def solve_reciprocal_square(val):
    """
    Given x + 1/x = val, returns x^2 + 1/x^2
    Formula: (x + 1/x)^2 - 2
    """
    return val**2 - 2

def solve_difference_of_squares_prime(p):
    """
    Solves x^2 - y^2 = p for prime p
    Returns (x, y)
    """
    # (x-y)(x+y) = p * 1
    # x+y = p, x-y = 1
    # 2x = p+1 => x = (p+1)/2
    if p % 2 == 0: return None # 2 is prime but x, y must be integers -> 2x=3 impossible
    x = (p + 1) // 2
    y = (p - 1) // 2
    return x, y

# 테스트
print(f"Alternating squares sum (n=20): {solve_alternating_squares(20)}") # Expected: 210
print(f"x^2 + 1/x^2 given x + 1/x = 5: {solve_reciprocal_square(5)}") # Expected: 23
print(f"Solution for x^2 - y^2 = 31: {solve_difference_of_squares_prime(31)}") # Expected: (16, 15)
```

## 4. 요약 (Summary)

- **수 계산:** $a^2 - b^2$ 꼴이 보이면 합차 공식을 먼저 떠올립니다.
- **식 변형:** 곱셈 공식의 변형($a^2+b^2 = (a+b)^2 - 2ab$)을 활용합니다.
- **정수 조건:** 소수의 약수 성질(1과 자기 자신)을 이용하여 부정방정식을 풉니다.

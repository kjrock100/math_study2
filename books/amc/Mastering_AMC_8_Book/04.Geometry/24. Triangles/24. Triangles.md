# 24. 삼각형 (Triangles)

## 1. 개요 (Overview)

삼각형(Triangle)은 기하학에서 가장 기본적이면서도 중요한 도형입니다. AMC 8에서는 삼각형의 변의 길이 조건(부등식), 넓이 공식, 피타고라스 정리, 그리고 특수한 삼각형(이등변, 정삼각형)의 성질을 묻는 문제가 자주 출제됩니다.

## 2. 주요 개념 (Key Concepts)

### 2.1 삼각형의 성립 조건 (Triangle Inequality Theorem)

삼각형의 세 변의 길이가 $a, b, c$일 때, **가장 긴 변의 길이는 나머지 두 변의 길이의 합보다 작아야 합니다.**
$$ a + b > c $$
(단, $c$가 가장 긴 변일 때)

- 일반적으로: 임의의 두 변의 합은 나머지 한 변보다 큽니다.
- 차의 관계: 임의의 두 변의 차는 나머지 한 변보다 작습니다. ($|a - b| < c$)

### 2.2 넓이 (Area)

$$ \text{Area} = \frac{1}{2} \times \text{base} \times \text{height} = \frac{1}{2}bh $$

- 밑변(base)과 높이(height)는 서로 수직이어야 합니다.

### 2.3 피타고라스 정리 (Pythagorean Theorem)

직각삼각형에서 직각을 낀 두 변의 길이를 $a, b$, 빗변(Hypotenuse)의 길이를 $c$라고 할 때:
$$ a^2 + b^2 = c^2 $$

- **피타고라스 수 (Pythagorean Triples):** $(3, 4, 5), (5, 12, 13), (8, 15, 17)$ 등 정수비를 외워두면 유용합니다.

### 2.4 특수한 삼각형 (Special Triangles)

- **이등변삼각형 (Isosceles):** 두 변의 길이가 같고, 두 밑각의 크기가 같습니다. 꼭지각의 이등분선은 밑변을 수직이등분합니다.
- **정삼각형 (Equilateral):** 세 변의 길이가 같고, 세 내각이 모두 $60^\circ$입니다.

## 3. 예제 (Examples)

### 예제 1: 삼각형 만들기

세 변의 길이가 $3, 5, x$인 삼각형이 존재하기 위한 정수 $x$의 개수는?

- 조건: $|5 - 3| < x < 5 + 3 \implies 2 < x < 8$.
- 가능한 정수: $3, 4, 5, 6, 7$.
- **정답:** 5개.

### 예제 2: 넓이 구하기

직각삼각형의 두 직각변이 $6, 8$일 때, 빗변의 길이와 넓이는?

- 빗변: $\sqrt{6^2 + 8^2} = \sqrt{36 + 64} = \sqrt{100} = 10$.
- 넓이: $\frac{1}{2} \times 6 \times 8 = 24$.

### 예제 3: 피타고라스 정리 응용

한 변의 길이가 10인 정삼각형의 높이는?

- 꼭지점에서 수선을 내리면 밑변을 이등분하므로 밑변의 절반은 5.
- 높이 $h = \sqrt{10^2 - 5^2} = \sqrt{100 - 25} = \sqrt{75} = 5\sqrt{3}$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 삼각형 성립 조건을 확인하고, 넓이와 빗변을 계산하는 함수입니다.

```python
import math

def is_valid_triangle(a, b, c):
    """삼각형 성립 조건 확인"""
    sides = sorted([a, b, c])
    # 가장 긴 변 < 나머지 두 변의 합
    return sides[2] < sides[0] + sides[1]

def triangle_area(base, height):
    """삼각형 넓이 계산"""
    return 0.5 * base * height

def calculate_hypotenuse(a, b):
    """피타고라스 정리를 이용한 빗변 계산"""
    return math.sqrt(a**2 + b**2)

def heron_formula(a, b, c):
    """헤론의 공식: 세 변의 길이를 알 때 넓이 구하기"""
    if not is_valid_triangle(a, b, c):
        return None
    s = (a + b + c) / 2
    area = math.sqrt(s * (s - a) * (s - b) * (s - c))
    return area

# 테스트
print(f"Can 3, 4, 7 form a triangle? {is_valid_triangle(3, 4, 7)}") # Expected: False
print(f"Can 3, 4, 5 form a triangle? {is_valid_triangle(3, 4, 5)}") # Expected: True

print(f"Hypotenuse of 6, 8: {calculate_hypotenuse(6, 8)}") # Expected: 10.0
print(f"Area of triangle (sides 3, 4, 5): {heron_formula(3, 4, 5)}") # Expected: 6.0
```

## 5. 요약 (Summary)

- **부등식:** 가장 긴 변은 나머지 합보다 작아야 한다.
- **넓이:** $\frac{1}{2}bh$.
- **피타고라스:** $a^2 + b^2 = c^2$ (직각삼각형).

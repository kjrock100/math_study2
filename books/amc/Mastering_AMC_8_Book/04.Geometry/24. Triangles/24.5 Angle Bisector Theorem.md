# 24.5 각의 이등분선 정리 (Angle Bisector Theorem)

## 1. 개요 (Overview)

각의 이등분선 정리(Angle Bisector Theorem)는 삼각형의 한 내각을 이등분하는 선이 대변을 나누는 비율에 관한 정리입니다. 기하학 문제에서 길이의 비를 구하거나, 특정 변의 길이를 계산할 때 매우 강력한 도구로 사용됩니다.

## 2. 내각의 이등분선 정리 (Internal Angle Bisector Theorem)

### 2.1 공식

삼각형 $ABC$에서 $\angle A$의 이등분선이 변 $BC$와 만나는 점을 $D$라고 할 때:
$$ AB : AC = BD : DC $$
또는
$$ \frac{AB}{AC} = \frac{BD}{DC} $$

### 2.2 직관적 이해 (Area Approach)

$\triangle ABD$와 $\triangle ACD$는 높이가 같습니다 (점 $A$에서 $BC$에 내린 수선).
따라서 넓이의 비는 밑변의 비와 같습니다.
$$ \frac{\text{Area}(\triangle ABD)}{\text{Area}(\triangle ACD)} = \frac{BD}{DC} $$
또한, 각의 이등분선 위의 점 $D$에서 변 $AB$와 $AC$에 이르는 거리는 같습니다 (높이).
따라서 넓이의 비는 $AB : AC$와도 같습니다.
결국 $AB : AC = BD : DC$가 성립합니다.

## 3. 외각의 이등분선 정리 (External Angle Bisector Theorem)

삼각형 $ABC$의 외각의 이등분선이 변 $BC$의 연장선과 만나는 점을 $E$라고 할 때:
$$ AB : AC = BE : CE $$
(단, $AB \neq AC$)

- "나이키(Nike) 모양" 또는 "하트 뒤집은 모양"으로 기억하기도 합니다.

## 4. 예제 (Examples)

### 예제 1: 변의 길이 구하기

$\triangle ABC$에서 $AB=10, AC=15, BC=20$이다. $\angle A$의 이등분선이 $BC$와 만나는 점을 $D$라 할 때, $BD$의 길이는?

- $BD : DC = AB : AC = 10 : 15 = 2 : 3$.
- $BD$는 전체 $BC$를 $2:3$으로 비례배분한 것 중 $2$에 해당합니다.
- $BD = 20 \times \frac{2}{2+3} = 20 \times \frac{2}{5} = 8$.

### 예제 2: 비율 문제

$\triangle ABC$에서 $AB=6, AC=8$이다. $\angle A$의 이등분선이 $BC$와 만나는 점을 $D$라 하자. $\triangle ABD$의 넓이가 12일 때, $\triangle ACD$의 넓이는?

- 넓이의 비는 밑변의 비 $BD : DC$와 같습니다.
- $BD : DC = AB : AC = 6 : 8 = 3 : 4$.
- $\text{Area}(\triangle ABD) : \text{Area}(\triangle ACD) = 3 : 4$.
- $12 : x = 3 : 4 \implies 3x = 48 \implies x = 16$.
- **정답:** 16.

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 내각 및 외각의 이등분선 정리를 계산하는 함수입니다.

```python
def solve_internal_angle_bisector(side_b, side_c, side_a):
    """
    내각의 이등분선 정리
    입력: AB(c), AC(b), BC(a)
    출력: (BD, DC) 길이 튜플
    공식: BD = a * (c / (b + c)), DC = a * (b / (b + c))
    """
    ratio_bd = side_c / (side_b + side_c)
    bd = side_a * ratio_bd
    dc = side_a - bd
    return bd, dc

def solve_external_angle_bisector(side_b, side_c, side_a):
    """
    외각의 이등분선 정리
    입력: AB(c), AC(b), BC(a) (단, c != b)
    출력: BE (외분점까지의 거리)
    공식: AB:AC = BE:CE => c:b = x:(x-a) (if E is on C side)
    c/b = x/(x-a) => c(x-a) = bx => cx - ca = bx => x(c-b) = ca => x = ca / (c-b)
    """
    if side_b == side_c:
        return None # 이등변삼각형의 외각 이등분선은 밑변과 평행

    # BE 길이 계산 (E가 C쪽에 있다고 가정, 즉 AB > AC)
    # 만약 AC > AB라면 E는 B쪽에 있음. 절대값으로 처리.
    be = abs(side_a * side_c / (side_c - side_b))
    return be

# 테스트
# 예제 1: AB=10, AC=15, BC=20
bd, dc = solve_internal_angle_bisector(15, 10, 20)
print(f"Internal Bisector (10, 15, 20) -> BD={bd}, DC={dc}") # Expected: BD=8.0, DC=12.0

# 외각 이등분선 테스트: AB=10, AC=6, BC=5
# 10:6 = BE:CE => 5:3 = x:(x-5) => 5x-25=3x => 2x=25 => x=12.5
be = solve_external_angle_bisector(6, 10, 5)
print(f"External Bisector (10, 6, 5) -> BE={be}") # Expected: 12.5
```

## 6. 요약 (Summary)

- **내각:** $AB:AC = BD:DC$ (내분).
- **외각:** $AB:AC = BE:CE$ (외분).
- **넓이:** 밑변의 비는 넓이의 비와 같다.

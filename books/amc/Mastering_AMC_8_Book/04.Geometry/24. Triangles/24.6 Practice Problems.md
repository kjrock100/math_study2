# 24.6 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 24.1장부터 24.5장까지 학습한 삼각형의 넓이, 피타고라스 정리, 특수 삼각형, 그리고 삼각형의 오심과 각의 이등분선 정리 등을 종합하여 실전 문제를 해결합니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **넓이와 길이 (Area & Length):** 헤론의 공식이나 특수 삼각형의 비를 이용하여 넓이와 변의 길이를 구합니다.
2.  **피타고라스 정리 응용:** 입체도형의 대각선이나 좌표평면 위의 거리를 구합니다.
3.  **비율과 닮음 (Ratios):** 각의 이등분선 정리나 무게중심의 성질을 이용하여 선분의 비율을 구합니다.

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: 이등변삼각형의 넓이

세 변의 길이가 $10, 10, 12$인 이등변삼각형의 넓이는?

- **전략:** 꼭짓점에서 밑변(12)에 수선을 내립니다.
- 수선은 밑변을 이등분하므로 밑변의 절반은 6입니다.
- 높이 $h = \sqrt{10^2 - 6^2} = \sqrt{100 - 36} = \sqrt{64} = 8$. (피타고라스 정리)
- 넓이 $A = \frac{1}{2} \times 12 \times 8 = 48$.
- **정답:** 48.

### 문제 2: 특수 직각삼각형

빗변의 길이가 20인 $30^\circ-60^\circ-90^\circ$ 직각삼각형에서 가장 짧은 변의 길이는?

- **비율:** $1 : \sqrt{3} : 2$ (짧은 변 : 긴 변 : 빗변).
- 짧은 변은 빗변의 절반입니다.
- $20 \div 2 = 10$.
- **정답:** 10.

### 문제 3: 각의 이등분선 정리

$\triangle ABC$에서 $AB=12, AC=18, BC=15$이다. $\angle A$의 이등분선이 $BC$와 만나는 점을 $D$라 할 때, $BD$의 길이는?

- **공식:** $AB : AC = BD : DC$.
- $12 : 18 = 2 : 3$.
- $BD$는 전체 $BC(15)$를 $2:3$으로 나눈 것 중 $2$에 해당합니다.
- $BD = 15 \times \frac{2}{2+3} = 15 \times \frac{2}{5} = 6$.
- **정답:** 6.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 위 문제들을 검증하는 함수입니다.

```python
import math

def isosceles_triangle_area(equal_side, base):
    """
    이등변삼각형의 넓이 구하기
    높이 = sqrt(equal_side^2 - (base/2)^2)
    """
    half_base = base / 2
    height = math.sqrt(equal_side**2 - half_base**2)
    return 0.5 * base * height

def solve_30_60_90_from_hypotenuse(hypotenuse):
    """
    빗변이 주어졌을 때 30-60-90 삼각형의 변 길이 반환
    (short, long, hypotenuse)
    """
    short_leg = hypotenuse / 2
    long_leg = short_leg * math.sqrt(3)
    return short_leg, long_leg, hypotenuse

def angle_bisector_segment(c, b, a):
    """
    AB=c, AC=b, BC=a 일 때 각 A의 이등분선이 BC를 나누는 길이 BD 구하기
    BD = a * (c / (c + b))
    """
    return a * (c / (c + b))

# 테스트
print(f"Area of Isosceles (10, 10, 12): {isosceles_triangle_area(10, 12)}") # Expected: 48.0
print(f"30-60-90 sides (hyp=20): {solve_30_60_90_from_hypotenuse(20)}") # Expected: (10.0, 17.32..., 20)
print(f"Angle Bisector BD (AB=12, AC=18, BC=15): {angle_bisector_segment(12, 18, 15)}") # Expected: 6.0
```

## 5. 요약 (Summary)

- **넓이:** 높이를 구할 수 있다면 $\frac{1}{2}bh$, 세 변을 안다면 헤론의 공식을 사용합니다.
- **특수각:** $30^\circ, 45^\circ, 60^\circ$가 보이면 특수 삼각형의 비를 떠올립니다.
- **비례식:** 각의 이등분선이나 닮음 조건에서 비례식을 세워 길이를 구합니다.

# 24.3 피타고라스 정리 (Pythagorean Theorem)

## 1. 개요 (Overview)

피타고라스 정리(Pythagorean Theorem)는 직각삼각형의 세 변 사이의 관계를 설명하는 기하학의 가장 유명한 정리입니다. 이 장에서는 정리의 심화 응용, **피타고라스 수(Triples)**, **거리 공식**, 그리고 **입체도형에서의 대각선** 길이를 구하는 방법을 다룹니다.

## 2. 주요 개념 (Key Concepts)

### 2.1 피타고라스 수 (Pythagorean Triples)

$a^2 + b^2 = c^2$를 만족하는 자연수 쌍 $(a, b, c)$입니다. 이를 암기하면 계산 속도가 매우 빨라집니다.

- **기본 쌍 (Primitives):**
  - $(3, 4, 5)$
  - $(5, 12, 13)$
  - $(8, 15, 17)$
  - $(7, 24, 25)$
  - $(9, 40, 41)$
- **확장:** 기본 쌍에 정수 $k$를 곱해도 성립합니다. 예: $(6, 8, 10)$.

### 2.2 삼각형의 판별 (Converse of Pythagorean Theorem)

가장 긴 변의 길이를 $c$, 나머지 두 변을 $a, b$라고 할 때:

- $c^2 = a^2 + b^2 \implies$ **직각삼각형 (Right)**
- $c^2 < a^2 + b^2 \implies$ **예각삼각형 (Acute)**
- $c^2 > a^2 + b^2 \implies$ **둔각삼각형 (Obtuse)**

### 2.3 거리 공식 (Distance Formula)

좌표평면 위의 두 점 $(x_1, y_1)$과 $(x_2, y_2)$ 사이의 거리는 피타고라스 정리를 이용하여 구합니다.
$$ d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2} $$

### 2.4 입체도형의 대각선 (Space Diagonal)

직육면체(가로 $l$, 세로 $w$, 높이 $h$)의 대각선 길이 $D$는 두 번의 피타고라스 정리를 통해 유도됩니다.
$$ D = \sqrt{l^2 + w^2 + h^2} $$

## 3. 예제 (Examples)

### 예제 1: 피타고라스 수 활용

직각삼각형의 빗변이 26이고 한 변이 10일 때, 다른 한 변의 길이는?

- $(5, 12, 13)$의 2배인 $(10, 24, 26)$을 떠올립니다.
- **정답:** 24.

### 예제 2: 삼각형 판별

세 변의 길이가 $4, 5, 7$인 삼각형의 종류는?

- 가장 긴 변 $c=7$.
- $c^2 = 49$.
- $a^2 + b^2 = 16 + 25 = 41$.
- $49 > 41$ 이므로 **둔각삼각형**.

### 예제 3: 두 점 사이의 거리

점 $(1, 2)$와 $(4, 6)$ 사이의 거리는?

- 가로 차이: $4 - 1 = 3$.
- 세로 차이: $6 - 2 = 4$.
- 거리: $\sqrt{3^2 + 4^2} = \sqrt{9 + 16} = 5$. (3-4-5 삼각형)

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 삼각형을 판별하고, 두 점 사이의 거리를 계산하는 함수입니다.

```python
import math

def classify_triangle_type(a, b, c):
    """
    세 변의 길이를 받아 삼각형의 종류(예각, 직각, 둔각)를 판별
    """
    sides = sorted([a, b, c])
    x, y, z = sides[0], sides[1], sides[2]

    if z >= x + y:
        return "Not a Triangle"

    z_sq = z**2
    xy_sq = x**2 + y**2

    if z_sq == xy_sq:
        return "Right Triangle"
    elif z_sq < xy_sq:
        return "Acute Triangle"
    else:
        return "Obtuse Triangle"

def distance_between_points(p1, p2):
    """
    두 점 p1=(x1, y1), p2=(x2, y2) 사이의 거리 계산
    """
    return math.sqrt((p2[0] - p1[0])**2 + (p2[1] - p1[1])**2)

def space_diagonal(l, w, h):
    """직육면체의 대각선 길이"""
    return math.sqrt(l**2 + w**2 + h**2)

# 테스트
print(f"Triangle (4, 5, 7): {classify_triangle_type(4, 5, 7)}") # Expected: Obtuse
print(f"Triangle (3, 4, 5): {classify_triangle_type(3, 4, 5)}") # Expected: Right
print(f"Distance (1, 2) to (4, 6): {distance_between_points((1, 2), (4, 6))}") # Expected: 5.0
print(f"Space Diagonal (3, 4, 12): {space_diagonal(3, 4, 12)}") # Expected: 13.0 (sqrt(9+16+144)=sqrt(169)=13)
```

## 5. 요약 (Summary)

- **피타고라스 수:** $(3,4,5), (5,12,13)$ 등을 암기하여 계산 시간을 단축합니다.
- **판별:** 가장 긴 변의 제곱이 나머지 합보다 크면 둔각, 작으면 예각입니다.
- **거리:** 좌표평면에서는 $x$차이와 $y$차이를 직각변으로 하는 빗변의 길이입니다.

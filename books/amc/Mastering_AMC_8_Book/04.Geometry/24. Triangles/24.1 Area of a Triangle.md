# 24.1 삼각형의 넓이 (Area of a Triangle)

## 1. 개요 (Overview)

삼각형의 넓이를 구하는 것은 기하학 문제 해결의 핵심입니다. 가장 기본적인 "밑변 $\times$ 높이 $\div 2$" 공식뿐만 아니라, 정삼각형이나 세 변의 길이가 주어졌을 때 등 다양한 상황에 맞는 공식을 적절히 선택하여 사용하는 것이 중요합니다.

## 2. 주요 공식 (Key Formulas)

### 2.1 기본 공식 (Standard Formula)

가장 널리 쓰이는 공식입니다.
$$ \text{Area} = \frac{1}{2} \times b \times h $$

- **주의:** 높이($h$)는 반드시 밑변($b$)과 **수직(Perpendicular)**이어야 합니다.
- **둔각삼각형 (Obtuse Triangle):** 높이가 삼각형의 **외부**에 존재할 수 있습니다. 밑변의 연장선에 수선을 그어 높이를 잰다는 점을 기억해야 합니다.

### 2.2 정삼각형 (Equilateral Triangle)

한 변의 길이가 $s$인 정삼각형의 넓이 공식은 암기해두면 매우 유용합니다.
$$ \text{Area} = \frac{\sqrt{3}}{4}s^2 $$

- **유도:** 꼭짓점에서 수선을 내리면 밑변을 이등분($s/2$)하고, 피타고라스 정리에 의해 높이는 $\frac{\sqrt{3}}{2}s$가 됩니다.

### 2.3 헤론의 공식 (Heron's Formula)

높이를 모르고 **세 변의 길이($a, b, c$)만 알 때** 사용합니다.

1.  반둘레(Semi-perimeter) $s$를 구합니다: $s = \frac{a + b + c}{2}$
2.  넓이를 계산합니다:
    $$ \text{Area} = \sqrt{s(s-a)(s-b)(s-c)} $$

## 3. 예제 (Examples)

### 예제 1: 둔각삼각형의 넓이

밑변의 길이가 10이고, 그 밑변의 연장선에 내린 수선의 길이(높이)가 6인 둔각삼각형의 넓이는?

- 모양에 상관없이 밑변과 높이가 주어졌으므로 기본 공식을 사용합니다.
- $\text{Area} = \frac{1}{2} \times 10 \times 6 = 30$.

### 예제 2: 정삼각형의 넓이

한 변의 길이가 4인 정삼각형의 넓이는?

- 공식 대입: $\frac{\sqrt{3}}{4} \times 4^2 = \frac{\sqrt{3}}{4} \times 16 = 4\sqrt{3}$.

### 예제 3: 헤론의 공식 활용

세 변의 길이가 $13, 14, 15$인 삼각형의 넓이는?

1.  $s = \frac{13 + 14 + 15}{2} = \frac{42}{2} = 21$.
2.  $\text{Area} = \sqrt{21(21-13)(21-14)(21-15)}$
    $= \sqrt{21 \times 8 \times 7 \times 6}$
    $= \sqrt{(3 \times 7) \times (2^3) \times 7 \times (2 \times 3)}$
    $= \sqrt{2^4 \times 3^2 \times 7^2}$
    $= 2^2 \times 3 \times 7 = 4 \times 21 = 84$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 다양한 방법으로 삼각형의 넓이를 계산하는 함수입니다.

```python
import math

def area_standard(base, height):
    """기본 공식: 1/2 * base * height"""
    return 0.5 * base * height

def area_equilateral(side):
    """정삼각형 넓이 공식: (sqrt(3)/4) * side^2"""
    return (math.sqrt(3) / 4) * (side ** 2)

def area_heron(a, b, c):
    """헤론의 공식: 세 변의 길이를 알 때"""
    # 삼각형 성립 조건 확인
    sides = sorted([a, b, c])
    if sides[2] >= sides[0] + sides[1]:
        return None # 삼각형이 만들어지지 않음

    s = (a + b + c) / 2
    try:
        area = math.sqrt(s * (s - a) * (s - b) * (s - c))
        return area
    except ValueError:
        return None

# 테스트
print(f"Standard Area (b=10, h=6): {area_standard(10, 6)}") # Expected: 30.0

side_eq = 4
print(f"Equilateral Area (s={side_eq}): {area_equilateral(side_eq):.4f}")
# Expected: 4 * sqrt(3) ≈ 6.9282

sides_heron = (13, 14, 15)
print(f"Heron's Area {sides_heron}: {area_heron(*sides_heron)}") # Expected: 84.0

# 잘못된 삼각형 테스트
print(f"Invalid Triangle (1, 2, 10): {area_heron(1, 2, 10)}") # Expected: None
```

## 5. 요약 (Summary)

- **기본:** 높이는 밑변과 수직이어야 한다. (둔각삼각형 주의)
- **정삼각형:** $\frac{\sqrt{3}}{4}s^2$ 공식을 외우면 계산 시간을 단축할 수 있다.
- **헤론:** 높이를 구하기 귀찮거나 어려울 때, 세 변의 길이만으로 넓이를 구할 수 있다.

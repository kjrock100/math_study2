# 24.2 특수 삼각형 (Special Triangles)

## 1. 개요 (Overview)

기하학 문제, 특히 AMC 8에서는 매번 피타고라스 정리를 계산하는 대신 **특수 삼각형의 길이 비(Ratio)**를 이용하여 즉각적으로 변의 길이를 구해야 하는 경우가 많습니다. 이 장에서는 가장 중요한 두 가지 특수 직각삼각형과 정삼각형의 성질을 다룹니다.

## 2. 주요 개념 (Key Concepts)

### 2.1 $45^\circ - 45^\circ - 90^\circ$ 삼각형 (Isosceles Right Triangle)

직각이등변삼각형입니다. 정사각형을 대각선으로 잘랐을 때 만들어집니다.

- **길이 비:** $1 : 1 : \sqrt{2}$
- **공식:**
  - $\text{빗변} = \text{직각변} \times \sqrt{2}$
  - $\text{직각변} = \text{빗변} \div \sqrt{2}$

### 2.2 $30^\circ - 60^\circ - 90^\circ$ 삼각형

정삼각형을 반으로 잘랐을 때 만들어지는 삼각형입니다.

- **길이 비:** $1 : \sqrt{3} : 2$
  - $1$: $30^\circ$의 대변 (가장 짧은 변)
  - $\sqrt{3}$: $60^\circ$의 대변 (중간 변)
  - $2$: $90^\circ$의 대변 (빗변, 가장 긴 변)
- **관계:** 빗변의 길이는 가장 짧은 변의 **2배**입니다.

### 2.3 정삼각형 (Equilateral Triangle)

세 변의 길이가 같고, 세 각이 모두 $60^\circ$입니다.

- 한 변의 길이가 $s$일 때:
  - **높이:** $h = \frac{\sqrt{3}}{2}s$ ($30-60-90$ 삼각형 원리 적용)
  - **넓이:** $A = \frac{\sqrt{3}}{4}s^2$

## 3. 예제 (Examples)

### 예제 1: 정사각형의 대각선

한 변의 길이가 5인 정사각형의 대각선 길이는?

- $45-45-90$ 삼각형의 빗변에 해당합니다.
- $5 \times \sqrt{2} = 5\sqrt{2}$.

### 예제 2: 30-60-90 삼각형

직각삼각형의 한 예각이 $30^\circ$이고 빗변의 길이가 12일 때, 나머지 두 변의 길이는?

- 가장 짧은 변 ($30^\circ$ 대변) = 빗변 $\div 2 = 12 \div 2 = 6$.
- 나머지 변 ($60^\circ$ 대변) = 짧은 변 $\times \sqrt{3} = 6\sqrt{3}$.
- **정답:** $6, 6\sqrt{3}$.

### 예제 3: 정삼각형의 높이

한 변의 길이가 10인 정삼각형의 높이는?

- 꼭짓점에서 수선을 내리면 밑변을 이등분하여 길이가 5가 됩니다.
- 높이는 $30-60-90$ 삼각형의 $\sqrt{3}$에 해당하는 변이므로 $5\sqrt{3}$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 특수 삼각형의 변의 길이를 계산하는 함수입니다.

```python
import math

def solve_45_45_90(leg=None, hypotenuse=None):
    """
    45-45-90 삼각형 변 계산
    leg: 직각변, hypotenuse: 빗변
    """
    if leg:
        return leg, leg, leg * math.sqrt(2)
    elif hypotenuse:
        leg = hypotenuse / math.sqrt(2)
        return leg, leg, hypotenuse
    return None

def solve_30_60_90(short_leg=None, long_leg=None, hypotenuse=None):
    """
    30-60-90 삼각형 변 계산 (1 : sqrt(3) : 2)
    """
    if short_leg:
        return short_leg, short_leg * math.sqrt(3), short_leg * 2
    elif long_leg:
        short = long_leg / math.sqrt(3)
        return short, long_leg, short * 2
    elif hypotenuse:
        short = hypotenuse / 2
        return short, short * math.sqrt(3), hypotenuse
    return None

# 테스트
print(f"Square diagonal (side 5): {solve_45_45_90(leg=5)[2]:.4f}") # Expected: 5*sqrt(2) ≈ 7.071

sides = solve_30_60_90(hypotenuse=12)
print(f"30-60-90 (hypotenuse 12): Short={sides[0]}, Long={sides[1]:.4f}")
# Expected: Short=6, Long=6*sqrt(3) ≈ 10.392
```

## 5. 요약 (Summary)

- **$45-45-90$:** $1 : 1 : \sqrt{2}$. (정사각형 대각선)
- **$30-60-90$:** $1 : \sqrt{3} : 2$. (정삼각형 반쪽)
- **팁:** 빗변이 가장 길다는 것을 항상 기억하여 비례식을 세웁니다.

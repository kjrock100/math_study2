# 23. 각도 구하기 (Angle Chasing)

## 1. 개요 (Overview)

각도 구하기(Angle Chasing)는 기하학 문제에서 주어진 각도들과 도형의 성질(평행선, 삼각형, 다각형 등)을 이용하여 미지의 각도를 찾아가는 과정을 말합니다. AMC 8 기하 영역에서 가장 기초적이면서도 빈번하게 출제되는 유형입니다.

## 2. 주요 개념 (Key Concepts)

### 2.1 직선과 각 (Lines and Angles)

- **맞꼭지각 (Vertical Angles):** 두 직선이 교차할 때 마주 보는 각의 크기는 같습니다.
- **평각 (Straight Angle):** 일직선 위의 각도는 $180^\circ$입니다.
- **보각 (Supplementary Angles):** 합해서 $180^\circ$가 되는 두 각입니다.
- **여각 (Complementary Angles):** 합해서 $90^\circ$가 되는 두 각입니다.

### 2.2 평행선 (Parallel Lines)

평행한 두 직선을 가로지르는 선(Transversal)이 있을 때:

- **동위각 (Corresponding Angles):** 같은 위치에 있는 각은 같습니다.
- **엇각 (Alternate Interior Angles):** 엇갈린 위치에 있는 안쪽 각은 같습니다.
- **동측내각 (Consecutive Interior Angles):** 같은 쪽에 있는 안쪽 각의 합은 $180^\circ$입니다.

### 2.3 삼각형 (Triangles)

- **내각의 합:** 모든 삼각형의 내각의 합은 $180^\circ$입니다.
- **외각의 성질 (Exterior Angle Theorem):** 삼각형의 한 외각의 크기는 이웃하지 않는 두 내각의 합과 같습니다.
  $$ \angle \text{Exterior} = \angle A + \angle B $$
- **이등변삼각형 (Isosceles Triangle):** 두 변의 길이가 같으면, 그 대각(밑각)의 크기도 같습니다.

### 2.4 다각형 (Polygons)

$n$각형에 대하여:

- **내각의 합:** $(n-2) \times 180^\circ$.
- **외각의 합:** 항상 $360^\circ$ (볼록 다각형일 때).
- **정다각형의 한 내각:** $\frac{(n-2) \times 180}{n}$.

## 3. 예제 (Examples)

### 예제 1: 이등변삼각형과 외각

$\triangle ABC$에서 $AB=AC$이고 $\angle A = 40^\circ$일 때, $\angle B$의 외각의 크기는?

- 이등변삼각형이므로 $\angle B = \angle C = (180 - 40) / 2 = 70^\circ$.
- $\angle B$의 외각 = $180 - 70 = 110^\circ$.
- 또는 외각 성질 이용: $\angle A + \angle C = 40 + 70 = 110^\circ$.
- **정답:** $110^\circ$.

### 예제 2: 정다각형

정십각형(Regular Decagon)의 한 내각의 크기는?

- $n=10$.
- 내각의 합: $(10-2) \times 180 = 8 \times 180 = 1440^\circ$.
- 한 내각: $1440 \div 10 = 144^\circ$.
- (다른 방법) 외각의 합은 $360^\circ$. 한 외각은 $360 \div 10 = 36^\circ$. 내각은 $180 - 36 = 144^\circ$.
- **정답:** $144^\circ$.

### 예제 3: 평행선 사이의 꺾인 선

두 평행선 사이에 $M$자 모양으로 꺾인 선이 있다. 꺾인 점의 각도를 $x$, 평행선 쪽의 내부 각도를 각각 $30^\circ, 40^\circ$라고 할 때 $x$는?

- **전략:** 꺾인 점을 지나며 평행한 보조선을 긋습니다.
- 엇각의 성질에 의해 $x$는 위쪽 엇각($30^\circ$)과 아래쪽 엇각($40^\circ$)의 합이 됩니다.
- $x = 30 + 40 = 70^\circ$.
- **정답:** $70^\circ$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 다각형의 내각/외각을 계산하고, 삼각형의 제3각을 구하는 함수입니다.

```python
def polygon_properties(n):
    """
    n각형의 내각의 합, 한 내각(정다각형), 한 외각(정다각형) 반환
    """
    if n < 3:
        return None

    sum_interior = (n - 2) * 180
    one_interior = sum_interior / n
    one_exterior = 360 / n

    return sum_interior, one_interior, one_exterior

def solve_triangle_angle(angle1, angle2):
    """
    삼각형의 두 각이 주어졌을 때 나머지 한 각 구하기
    """
    if angle1 + angle2 >= 180:
        return None # 삼각형 성립 불가
    return 180 - (angle1 + angle2)

def solve_isosceles_base_angle(vertex_angle):
    """
    이등변삼각형의 꼭지각이 주어졌을 때 밑각 구하기
    """
    if vertex_angle >= 180:
        return None
    return (180 - vertex_angle) / 2

# 테스트
n = 10
props = polygon_properties(n)
print(f"Regular {n}-gon: Sum={props[0]}, Interior={props[1]}, Exterior={props[2]}")
# Expected: Sum=1440, Interior=144.0, Exterior=36.0

print(f"Triangle third angle (40, 70): {solve_triangle_angle(40, 70)}") # Expected: 70

print(f"Isosceles base angle (vertex=40): {solve_isosceles_base_angle(40)}") # Expected: 70.0
```

## 5. 요약 (Summary)

- **보조선:** 평행선 문제에서 꺾인 점이 나오면 평행한 보조선을 긋습니다.
- **외각:** 삼각형 문제에서 외각 성질($\text{Ext} = \text{Int}_1 + \text{Int}_2$)을 적극 활용하면 계산이 빨라집니다.
- **다각형:** 외각의 합은 항상 $360^\circ$임을 기억하면 정다각형 문제를 쉽게 풀 수 있습니다.

# 23.4 심화 원주각 정리 (Advanced Circle Angle Chasing Theorems)

## 1. 개요 (Overview)

이 장에서는 기본적인 원주각 정리를 확장하여, 원의 접선(Tangent)과 할선(Secant), 현(Chord)들이 만들어내는 다양한 각도를 구하는 심화 정리들을 다룹니다. 이 공식들은 복잡한 원 기하 문제를 빠르고 정확하게 해결하는 데 필수적입니다.

## 2. 주요 정리 (Key Theorems)

### 2.1 접현각 정리 (Tangent-Chord Theorem)

원의 접선과 그 접점을 지나는 현이 이루는 각의 크기는, 그 각의 내부에 있는 호(Arc)에 대한 원주각의 크기와 같습니다.
$$ \angle (\text{Tangent, Chord}) = \frac{1}{2} \times \text{Intercepted Arc} $$

- 즉, 현이 만드는 원주각과 접선 사이의 각도는 같습니다.

### 2.2 원 내부의 교각 (Interior Intersection Theorem)

원 내부에서 두 현이 교차할 때 생기는 각(교각)의 크기는, 그 각과 맞꼭지각이 끼고 있는 **두 호의 합의 절반**입니다.
$$ \angle \text{Interior} = \frac{\text{Arc}\_1 + \text{Arc}\_2}{2} $$

- (두 호의 평균)

### 2.3 원 외부의 교각 (Exterior Intersection Theorem)

원 외부의 한 점에서 원에 그은 두 할선(또는 접선)이 이루는 각의 크기는, 두 할선 사이에 끼인 **두 호의 차의 절반**입니다.
$$ \angle \text{Exterior} = \frac{|\text{Major Arc} - \text{Minor Arc}|}{2} $$

- (큰 호 - 작은 호) / 2

## 3. 예제 (Examples)

### 예제 1: 접현각

원 위의 점 $T$에서의 접선과 현 $TA$가 이루는 각이 $50^\circ$이다. 호 $TA$의 원주각의 크기는?

- 접현각 정리에 의해 접선과 현이 이루는 각은 그 호에 대한 원주각과 같습니다.
- **정답:** $50^\circ$. (호의 중심각은 $100^\circ$)

### 예제 2: 내부 교각

원 내부에서 두 현이 만난다. 끼인 두 호의 중심각이 각각 $40^\circ$와 $80^\circ$일 때, 교각의 크기는?

- 공식: $\frac{40 + 80}{2} = \frac{120}{2} = 60^\circ$.
- **정답:** $60^\circ$.

### 예제 3: 외부 교각

원 밖의 점 $P$에서 원에 두 할선을 그었다. 두 할선 사이에 끼인 큰 호가 $100^\circ$, 작은 호가 $30^\circ$일 때, $\angle P$의 크기는?

- 공식: $\frac{100 - 30}{2} = \frac{70}{2} = 35^\circ$.
- **정답:** $35^\circ$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 접현각, 내부 교각, 외부 교각을 계산하는 함수입니다.

```python
def tangent_chord_angle(arc_degree):
    """
    접현각 계산: 호의 각도(중심각)가 주어졌을 때
    Angle = 1/2 * Arc
    """
    return arc_degree / 2

def interior_intersection_angle(arc1, arc2):
    """
    원 내부 교각 계산: (호1 + 호2) / 2
    """
    return (arc1 + arc2) / 2

def exterior_intersection_angle(major_arc, minor_arc):
    """
    원 외부 교각 계산: (큰 호 - 작은 호) / 2
    """
    return abs(major_arc - minor_arc) / 2

# 테스트
print(f"Tangent-Chord angle for arc 100: {tangent_chord_angle(100)}")
# Expected: 50.0

print(f"Interior intersection angle (arcs 40, 80): {interior_intersection_angle(40, 80)}")
# Expected: 60.0

print(f"Exterior intersection angle (arcs 100, 30): {exterior_intersection_angle(100, 30)}")
# Expected: 35.0
```

## 5. 요약 (Summary)

| 위치                    | 공식                                        | 키워드         |
| :---------------------- | :------------------------------------------ | :------------- |
| **원주 위 (On Circle)** | $\frac{1}{2} \times \text{Arc}$             | 원주각, 접현각 |
| **원 내부 (Inside)**    | $\frac{1}{2} (\text{Arc}_1 + \text{Arc}_2)$ | 합의 반 (평균) |
| **원 외부 (Outside)**   | $\frac{1}{2} (\text{Major} - \text{Minor})$ | 차의 반        |

- **접현각:** 원주각과 원리가 같다.
- **내부:** 더해서 반.
- **외부:** 빼서 반.

# 23.2 원주각 (Inscribed Angles)

## 1. 개요 (Overview)

원주각(Inscribed Angle)은 원 위의 한 점을 꼭짓점으로 하고, 원의 두 현을 변으로 하는 각을 말합니다. 원의 중심에서 만들어지는 중심각(Central Angle)과의 관계는 기하학 문제, 특히 원과 관련된 문제에서 가장 핵심적인 도구입니다.

## 2. 주요 정리 (Key Theorems)

### 2.1 원주각과 중심각 (Inscribed Angle Theorem)

같은 호(Arc)를 공유하는 원주각의 크기는 중심각의 크기의 절반입니다.
$$ \angle \text{Inscribed} = \frac{1}{2} \times \angle \text{Central} $$
$$ \angle \text{Central} = 2 \times \angle \text{Inscribed} $$

### 2.2 같은 호에 대한 원주각

같은 호를 공유하는 모든 원주각의 크기는 서로 같습니다.

- 이 성질을 이용해 복잡한 그림에서 같은 각도를 찾아낼 수 있습니다 ("나비 넥타이" 모양).

### 2.3 탈레스의 정리 (Thales's Theorem)

지름(Diameter)을 한 변으로 하고, 맞은편 꼭짓점이 원 위에 있는 삼각형은 항상 **직각삼각형**입니다.

- 즉, 반원(Semicircle)에 대한 원주각은 항상 $90^\circ$입니다.

### 2.4 내접사각형 (Cyclic Quadrilaterals)

원에 내접하는 사각형의 마주 보는 두 각(대각)의 합은 $180^\circ$입니다.
$$ \angle A + \angle C = 180^\circ $$
$$ \angle B + \angle D = 180^\circ $$

## 3. 예제 (Examples)

### 예제 1: 기본 정리

원 $O$에서 호 $AB$에 대한 중심각 $\angle AOB = 100^\circ$일 때, 원주각 $\angle ACB$의 크기는?

- $\angle ACB = \frac{1}{2} \times \angle AOB = \frac{1}{2} \times 100 = 50^\circ$.

### 예제 2: 지름과 원주각

원 $O$의 지름이 $AB$이고, 원 위의 점 $C$에 대하여 $\angle CAB = 30^\circ$일 때, $\angle ABC$는?

- 지름에 대한 원주각 $\angle ACB = 90^\circ$.
- 삼각형 내각의 합: $180 - (90 + 30) = 60^\circ$.
- **정답:** $60^\circ$.

### 예제 3: 내접사각형

원에 내접하는 사각형 $ABCD$에서 $\angle A = 80^\circ$일 때, $\angle C$는?

- 마주 보는 각의 합은 $180^\circ$.
- $\angle C = 180 - 80 = 100^\circ$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 원주각과 중심각의 관계, 내접사각형의 성질을 계산하는 함수입니다.

```python
def inscribed_angle_from_central(central_angle):
    """중심각이 주어졌을 때 원주각 계산"""
    return central_angle / 2

def central_angle_from_inscribed(inscribed_angle):
    """원주각이 주어졌을 때 중심각 계산"""
    return inscribed_angle * 2

def solve_cyclic_quadrilateral(angle):
    """내접사각형에서 마주 보는 각 구하기"""
    if angle >= 180 or angle <= 0:
        return None
    return 180 - angle

def is_right_triangle_in_circle(angle_on_circle):
    """원주각이 90도인지 확인 (지름을 끼고 있는지)"""
    return angle_on_circle == 90

# 테스트
print(f"Inscribed angle for central 100: {inscribed_angle_from_central(100)}") # Expected: 50.0
print(f"Opposite angle of 80 in cyclic quad: {solve_cyclic_quadrilateral(80)}") # Expected: 100
print(f"Is angle 90 a right angle (Thales)? {is_right_triangle_in_circle(90)}") # Expected: True
```

## 5. 요약 (Summary)

- **원주각 = 1/2 중심각.**
- **지름 $\rightarrow$ 90도.**
- **내접사각형 $\rightarrow$ 대각의 합 180도.**

# 23.3 다각형 (Polygons)

## 1. 개요 (Overview)

다각형(Polygon)은 여러 개의 선분으로 둘러싸인 평면도형입니다. 삼각형, 사각형을 넘어 오각형, 육각형 등 변의 개수가 많은 다각형의 각도 문제를 해결하기 위해서는 내각과 외각의 합에 대한 공식을 이해하는 것이 필수적입니다.

## 2. 주요 공식 (Key Formulas)

### 2.1 내각의 합 (Sum of Interior Angles)

$n$각형의 내각의 총합은 다음과 같습니다.
$$ \text{Sum} = (n-2) \times 180^\circ $$

- **원리:** 다각형의 한 꼭짓점에서 대각선을 그으면 $n-2$개의 삼각형으로 나눌 수 있습니다.

### 2.2 외각의 합 (Sum of Exterior Angles)

모든 볼록 다각형의 외각의 총합은 변의 개수와 상관없이 항상 같습니다.
$$ \text{Sum} = 360^\circ $$

### 2.3 정다각형 (Regular Polygons)

모든 변의 길이와 모든 내각의 크기가 같은 다각형입니다.

- **한 내각의 크기:**
  $$ \frac{(n-2) \times 180^\circ}{n} $$
- **한 외각의 크기:**
  $$ \frac{360^\circ}{n} $$
  > **Tip:** 정다각형의 내각을 구할 때, 외각을 먼저 구한 뒤($360/n$) $180^\circ$에서 빼는 것이 더 빠를 수 있습니다.

## 3. 예제 (Examples)

### 예제 1: 내각의 합

팔각형(Octagon)의 내각의 합은?

- $n=8$.
- 합 = $(8-2) \times 180 = 6 \times 180 = 1080^\circ$.

### 예제 2: 정다각형의 한 내각

정구각형(Regular Nonagon)의 한 내각의 크기는?

- $n=9$.
- 외각: $360 \div 9 = 40^\circ$.
- 내각: $180 - 40 = 140^\circ$.

### 예제 3: 역추적

한 내각의 크기가 $150^\circ$인 정다각형은 몇 각형인가?

- 한 외각의 크기: $180 - 150 = 30^\circ$.
- 변의 개수 $n = 360 \div (\text{한 외각}) = 360 \div 30 = 12$.
- **정답:** 십이각형 (Dodecagon).

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 다각형의 각도 관련 속성을 계산하는 함수입니다.

```python
def polygon_angle_properties(n):
    """
    n각형의 각도 속성 계산
    Returns: (내각의 합, 정다각형의 한 내각, 정다각형의 한 외각)
    """
    if n < 3:
        return None, None, None

    sum_interior = (n - 2) * 180
    regular_interior = sum_interior / n
    regular_exterior = 360 / n

    return sum_interior, regular_interior, regular_exterior

def find_sides_from_interior_angle(angle):
    """
    정다각형의 한 내각 크기로부터 변의 개수 찾기
    """
    if angle >= 180:
        return None

    exterior_angle = 180 - angle

    # 360이 외각으로 나누어 떨어져야 정다각형이 성립
    if 360 % exterior_angle == 0:
        return 360 // exterior_angle
    else:
        return "Not a valid regular polygon"

# 테스트
sum_int, one_int, one_ext = polygon_angle_properties(8)
print(f"Octagon (n=8): Sum={sum_int}, Interior={one_int}, Exterior={one_ext}")
# Expected: Sum=1080, Interior=135.0, Exterior=45.0

sides = find_sides_from_interior_angle(150)
print(f"A regular polygon with an interior angle of 150 has {sides} sides.")
# Expected: 12
```

## 5. 요약 (Summary)

- **내각의 합:** $(n-2) \times 180^\circ$.
- **외각의 합:** 항상 $360^\circ$.
- **정다각형:** 외각($360/n$)을 이용하면 계산이 간편하다.

# 23.5 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 23.1장부터 23.4장까지 학습한 기하학의 각도 구하기(Angle Chasing) 개념들을 종합하여 실전 문제를 해결합니다. **다각형의 내각과 외각**, **원주각과 중심각**, **접현각 및 원 내/외부의 교각** 등 다양한 정리를 적재적소에 활용하는 능력을 기릅니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **다각형 (Polygons):** 정다각형의 한 내각 구하기, 내각의 합을 이용해 미지수 찾기.
2.  **원주각 (Inscribed Angles):** 원주각과 중심각의 관계, 지름을 낀 삼각형(직각), 내접사각형의 성질 활용.
3.  **심화 원주각 (Advanced Circle Theorems):** 접현각, 원 내부/외부에서 만나는 선들이 이루는 각 구하기.
4.  **복합 도형:** 원과 다각형, 또는 평행선이 결합된 복잡한 도형에서 각도 추적하기.

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: 정다각형의 각도

정십이각형(Regular Dodecagon)의 한 내각의 크기는?

- **전략:** 외각을 먼저 구하는 것이 빠릅니다.
- 외각의 합은 항상 $360^\circ$.
- 한 외각: $360 \div 12 = 30^\circ$.
- 한 내각: $180 - 30 = 150^\circ$.
- **정답:** $150^\circ$.

### 문제 2: 내접사각형

원에 내접하는 사각형 $ABCD$에서 $\angle A = 75^\circ$, $\angle B = 110^\circ$일 때, $\angle C$와 $\angle D$의 크기는?

- **성질:** 내접사각형의 대각의 합은 $180^\circ$입니다.
- $\angle C = 180 - \angle A = 180 - 75 = 105^\circ$.
- $\angle D = 180 - \angle B = 180 - 110 = 70^\circ$.
- **정답:** $\angle C = 105^\circ, \angle D = 70^\circ$.

### 문제 3: 원 외부의 교각

원 밖의 점 $P$에서 두 할선을 그었다. 두 할선이 원과 만나는 점들에 의해 생긴 두 호(Arc)의 크기가 각각 $110^\circ$와 $50^\circ$일 때, $\angle P$의 크기는?

- **공식:** $\frac{\text{Major Arc} - \text{Minor Arc}}{2}$.
- **계산:** $\frac{110 - 50}{2} = \frac{60}{2} = 30^\circ$.
- **정답:** $30^\circ$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 다각형, 내접사각형, 원 외부 교각 문제를 해결하는 함수입니다.

```python
def regular_polygon_interior_angle(n):
    """정n각형의 한 내각의 크기"""
    if n < 3: return None
    exterior = 360 / n
    return 180 - exterior

def solve_cyclic_quad_opposite_angle(angle):
    """내접사각형의 대각 구하기"""
    return 180 - angle

def circle_exterior_angle(major_arc, minor_arc):
    """원 외부의 교각 구하기: (큰 호 - 작은 호) / 2"""
    return (major_arc - minor_arc) / 2

# 테스트
print(f"Interior angle of Regular 12-gon: {regular_polygon_interior_angle(12)}")
# Expected: 150.0

print(f"Opposite of 75 in cyclic quad: {solve_cyclic_quad_opposite_angle(75)}")
# Expected: 105

print(f"Exterior angle (arcs 110, 50): {circle_exterior_angle(110, 50)}")
# Expected: 30.0
```

## 5. 요약 (Summary)

- **다각형:** 외각($360/n$)을 활용하면 계산이 간편합니다.
- **원:** 중심각은 원주각의 2배, 내접사각형 대각 합은 180도입니다.
- **심화:** 교각은 두 호의 차(외부) 또는 합(내부)의 절반입니다.

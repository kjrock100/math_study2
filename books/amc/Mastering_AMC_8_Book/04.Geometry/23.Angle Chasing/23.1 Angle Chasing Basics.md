# 23.1 각도 구하기 기초 (Angle Chasing Basics)

## 1. 개요 (Overview)

각도 구하기(Angle Chasing)는 기하학에서 가장 기본이 되는 기술입니다. 복잡한 도형 문제도 결국은 직선, 평행선, 삼각형의 기본 성질을 단계적으로 적용하여 해결합니다. 이 장에서는 각도를 구하는 데 필요한 핵심 도구들을 정리합니다.

## 2. 핵심 도구 (Key Tools)

### 2.1 직선과 점 (Lines and Points)

- **평각 (Angles on a Line):** 직선 위의 각의 합은 $180^\circ$입니다.
- **한 점에서의 각 (Angles at a Point):** 한 점을 둘러싼 각의 합은 $360^\circ$입니다.
- **맞꼭지각 (Vertical Angles):** 두 직선이 교차할 때 마주 보는 각은 서로 같습니다.

### 2.2 평행선 (Parallel Lines)

평행한 두 직선을 다른 직선(Transversal)이 가로지를 때:

- **엇각 (Alternate Interior Angles):** 'Z'자 모양. 서로 같습니다.
- **동위각 (Corresponding Angles):** 'F'자 모양. 서로 같습니다.
- **동측내각 (Consecutive Interior Angles):** 'C'자 모양. 합이 $180^\circ$입니다.

### 2.3 삼각형 (Triangles)

- **내각의 합:** $\angle A + \angle B + \angle C = 180^\circ$.
- **외각 정리:** 한 외각은 이웃하지 않는 두 내각의 합과 같습니다. ($\text{Ext} = \angle A + \angle B$)

## 3. 예제 (Examples)

### 예제 1: 평행선과 엇각

두 평행선 $l, m$이 있고, 이를 가로지르는 선이 있다. 교각 중 하나가 $120^\circ$일 때, 엇각의 크기는?

- 엇각은 서로 같으므로 $120^\circ$.
- 동측내각은 $180 - 120 = 60^\circ$.

### 예제 2: 삼각형의 각도

삼각형의 두 각이 $50^\circ, 60^\circ$일 때, 나머지 한 각과 그 외각은?

- 나머지 내각: $180 - (50 + 60) = 70^\circ$.
- 외각: $180 - 70 = 110^\circ$ (또는 $50 + 60 = 110^\circ$).

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 기본적인 각도 관계를 계산하는 함수입니다.

```python
def find_supplementary_angle(angle):
    """보각 구하기 (직선 위의 각)"""
    if angle > 180 or angle < 0:
        return None
    return 180 - angle

def solve_parallel_lines(angle, relation_type):
    """
    평행선 관계에서 각도 구하기
    relation_type: 'alternate'(엇각), 'corresponding'(동위각), 'consecutive'(동측내각)
    """
    if relation_type in ['alternate', 'corresponding']:
        return angle
    elif relation_type == 'consecutive':
        return 180 - angle
    return None

def triangle_solver(angle1, angle2):
    """나머지 내각과 외각 구하기"""
    angle3 = 180 - (angle1 + angle2)
    exterior_angle3 = 180 - angle3
    return angle3, exterior_angle3

# 테스트
print(f"Supplement of 120: {find_supplementary_angle(120)}") # Expected: 60
print(f"Alternate interior to 120: {solve_parallel_lines(120, 'alternate')}") # Expected: 120
print(f"Consecutive interior to 120: {solve_parallel_lines(120, 'consecutive')}") # Expected: 60
print(f"Triangle (50, 60) -> 3rd angle & exterior: {triangle_solver(50, 60)}") # Expected: (70, 110)
```

## 5. 요약 (Summary)

- **직선:** 합은 180도.
- **평행선:** Z(엇각), F(동위각)는 같고, C(동측내각)는 합이 180도.
- **삼각형:** 내각 합 180도, 외각은 두 내각의 합.

# 25.7 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 25.1장부터 25.6장까지 학습한 사각형(Quadrilaterals)의 성질과 넓이 공식을 종합하여 실전 문제를 해결합니다. 평행사변형, 마름모, 사다리꼴 등 특수한 사각형의 성질을 이용하거나, 불규칙한 사각형을 삼각형으로 나누어 넓이를 구하는 전략을 연습합니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **각도 구하기 (Angle Chasing):** 사각형의 내각의 합($360^\circ$)과 평행선의 성질(동측내각 합 $180^\circ$)을 이용합니다.
2.  **넓이 계산 (Area Calculation):** 기본 공식(밑변 $\times$ 높이 등)을 사용하거나, 대각선을 이용해 삼각형으로 분할하여 구합니다.
3.  **대각선의 성질 (Diagonals):** 마름모(수직이등분), 직사각형(길이 같음), 평행사변형(이등분)의 대각선 성질을 활용합니다.
4.  **좌표평면 위의 사각형:** 좌표가 주어졌을 때 신발끈 공식(Shoelace Formula)이나 사각형을 포함하는 직사각형(Bounding Box)에서 빼는 방법을 사용합니다.

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: 등변사다리꼴의 각도

등변사다리꼴 $ABCD$ ($AD \parallel BC$)에서 $\angle A = 110^\circ$일 때, $\angle C$의 크기는?

- **성질:** 등변사다리꼴은 밑각의 크기가 같습니다 ($\angle B = \angle C$).
- 평행선 사이의 동측내각의 합은 $180^\circ$이므로 $\angle A + \angle B = 180^\circ$.
- $\angle B = 180 - 110 = 70^\circ$.
- 따라서 $\angle C = 70^\circ$.
- **정답:** $70^\circ$.

### 문제 2: 마름모의 넓이

한 변의 길이가 5이고 한 대각선의 길이가 6인 마름모의 넓이는?

- **전략:** 마름모의 대각선은 서로 수직이등분합니다.
- 주어진 대각선의 절반은 3. 직각삼각형의 빗변은 5.
- 피타고라스 정리: 나머지 대각선의 절반 $x = \sqrt{5^2 - 3^2} = \sqrt{16} = 4$.
- 나머지 대각선의 전체 길이 = 8.
- 넓이 = $\frac{6 \times 8}{2} = 24$.
- **정답:** 24.

### 문제 3: 좌표평면 위의 사각형 넓이

네 점 $(0,0), (5,0), (5,4), (2,6)$을 꼭짓점으로 하는 사각형의 넓이는?

- **분할법:** $x=2$에서 수선을 내려 사다리꼴과 직각삼각형으로 나누거나, 전체 큰 직사각형에서 빈 부분을 뺍니다.
- **방법 1 (사다리꼴 + 직각삼각형):**
  - 오른쪽 사다리꼴: $(2,0), (5,0), (5,4), (2,6)$. 윗변 6, 아랫변 4, 높이 3. 넓이 $\frac{6+4}{2} \times 3 = 15$.
  - 왼쪽 직각삼각형: $(0,0), (2,0), (2,6)$. 밑변 2, 높이 6. 넓이 $\frac{2 \times 6}{2} = 6$.
  - 총 넓이: $15 + 6 = 21$.
- **방법 2 (신발끈 공식):**
  - $(0,0) \to (5,0) \to (5,4) \to (2,6) \to (0,0)$.
  - $0 + 20 + 30 + 0 = 50$.
  - $0 + 0 + 8 + 0 = 8$.
  - Area = $\frac{|50 - 8|}{2} = 21$.
- **정답:** 21.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 마름모 넓이와 좌표평면 위 다각형의 넓이를 계산하는 함수입니다.

```python
import math

def rhombus_area_from_side_and_diagonal(side, diagonal1):
    """
    한 변과 한 대각선이 주어졌을 때 마름모 넓이 구하기
    """
    # 피타고라스 정리로 다른 대각선의 절반 구하기
    half_d1 = diagonal1 / 2
    if side <= half_d1:
        return None # 불가능한 삼각형

    half_d2 = math.sqrt(side**2 - half_d1**2)
    diagonal2 = 2 * half_d2

    return (diagonal1 * diagonal2) / 2

def polygon_area_shoelace(coords):
    """
    신발끈 공식을 이용한 다각형 넓이 계산
    coords: [(x1, y1), (x2, y2), ...]
    """
    n = len(coords)
    area = 0.0
    for i in range(n):
        j = (i + 1) % n
        area += coords[i][0] * coords[j][1]
        area -= coords[j][0] * coords[i][1]
    return abs(area) / 2.0

def isosceles_trapezoid_angles(angle_A):
    """
    등변사다리꼴(AD // BC)에서 각 A가 주어졌을 때 나머지 각 구하기
    A, D는 위쪽, B, C는 아래쪽이라 가정 (또는 그 반대)
    """
    angle_D = angle_A
    angle_B = 180 - angle_A
    angle_C = 180 - angle_A
    return {"A": angle_A, "B": angle_B, "C": angle_C, "D": angle_D}

# 테스트
print(f"Rhombus Area (side=5, d1=6): {rhombus_area_from_side_and_diagonal(5, 6)}") # Expected: 24.0

points = [(0,0), (5,0), (5,4), (2,6)]
print(f"Polygon Area (Shoelace): {polygon_area_shoelace(points)}") # Expected: 21.0

print(f"Isosceles Trapezoid Angles (A=110): {isosceles_trapezoid_angles(110)}")
# Expected: C=70
```

## 5. 요약 (Summary)

- **성질 활용:** 평행사변형 계열(마름모, 직사각형)의 대각선 성질을 기억합니다.
- **분할:** 복잡한 사각형은 대각선을 긋거나 수선을 내려 삼각형과 사각형으로 쪼갭니다.
- **좌표:** 꼭짓점 좌표가 주어지면 신발끈 공식이 가장 빠르고 정확합니다.

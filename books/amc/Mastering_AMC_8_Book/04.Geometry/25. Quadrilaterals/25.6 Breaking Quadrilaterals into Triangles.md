# 25.6 사각형을 삼각형으로 나누기 (Breaking Quadrilaterals into Triangles)

## 1. 개요 (Overview)

모든 사각형은 대각선을 그어 두 개의 삼각형으로 나눌 수 있습니다. 이 방법은 불규칙한 사각형(Irregular Quadrilateral)의 넓이를 구하거나, 사다리꼴, 마름모 등의 넓이 공식을 유도할 때 매우 유용하게 사용됩니다. "복잡한 문제는 단순한 문제(삼각형)로 쪼개서 푼다"는 수학적 사고방식의 핵심입니다.

## 2. 주요 방법 (Key Methods)

### 2.1 대각선을 이용한 분할 (Splitting by Diagonal)

사각형 $ABCD$에서 대각선 $AC$를 그으면 $\triangle ABC$와 $\triangle ADC$로 나뉩니다.

- **넓이:** $\text{Area}(ABCD) = \text{Area}(\triangle ABC) + \text{Area}(\triangle ADC)$.
- **공식:** 대각선의 길이를 $d$, 두 나머지 꼭짓점($B, D$)에서 대각선에 내린 수선의 길이(높이)를 각각 $h_1, h_2$라고 하면:
  $$ \text{Area} = \frac{1}{2} \times d \times h_1 + \frac{1}{2} \times d \times h_2 = \frac{1}{2} d (h_1 + h_2) $$

### 2.2 사다리꼴의 분할 (Trapezoid Decomposition)

사다리꼴은 대각선을 그어 높이가 같은 두 삼각형으로 나눌 수 있습니다.

- 윗변 $a$, 아랫변 $b$, 높이 $h$인 사다리꼴.
- 삼각형 1: 밑변 $a$, 높이 $h \rightarrow \text{Area} = \frac{1}{2}ah$.
- 삼각형 2: 밑변 $b$, 높이 $h \rightarrow \text{Area} = \frac{1}{2}bh$.
- 합: $\frac{1}{2}ah + \frac{1}{2}bh = \frac{1}{2}(a+b)h$. (사다리꼴 넓이 공식 유도)

## 3. 예제 (Examples)

### 예제 1: 대각선과 수선

사각형의 대각선 길이가 10이고, 나머지 두 꼭짓점에서 이 대각선에 내린 수선의 길이가 각각 3과 4일 때, 사각형의 넓이는?

- **전략:** 두 삼각형의 넓이의 합으로 구합니다.
- $\text{Area} = \frac{1}{2} \times 10 \times 3 + \frac{1}{2} \times 10 \times 4 = 15 + 20 = 35$.
- 또는 공식: $\frac{1}{2} \times 10 \times (3 + 4) = 5 \times 7 = 35$.
- **정답:** 35.

### 예제 2: 좌표평면 위의 사각형

네 점 $(0,0), (4,0), (4,3), (0,5)$를 꼭짓점으로 하는 사각형의 넓이는?

- **분석:** 이 도형은 $x=0$과 $x=4$가 평행한 사다리꼴입니다. (변 $(0,0)-(0,5)$와 $(4,0)-(4,3)$은 평행하지 않음. $y$축과 $x=4$ 직선이 평행하므로 세로선이 평행한 사다리꼴)
  - 밑변(세로): $5$ (왼쪽), $3$ (오른쪽).
  - 높이(가로): $4$.
  - 넓이: $\frac{(5+3) \times 4}{2} = 16$.
- **삼각형 분할법:** 대각선 $(0,0)-(4,3)$을 긋습니다.
  - 아래쪽 삼각형: 밑변 4, 높이 0? (직각삼각형 $(0,0)-(4,0)-(4,3)$). 넓이 $\frac{1}{2} \times 4 \times 3 = 6$.
  - 위쪽 삼각형: 밑변 5 (y축), 높이 4. 넓이 $\frac{1}{2} \times 5 \times 4 = 10$.
  - 합: $6 + 10 = 16$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 대각선과 높이를 이용해 사각형의 넓이를 계산하는 함수입니다.

```python
def area_quadrilateral_split(diagonal, h1, h2):
    """
    대각선과 두 높이를 이용한 사각형 넓이 계산
    Area = 1/2 * d * (h1 + h2)
    """
    return 0.5 * diagonal * (h1 + h2)

def area_irregular_polygon_coords(coords):
    """
    좌표가 주어졌을 때 신발끈 공식(Shoelace Formula)을 이용한 넓이 계산
    (삼각형 분할의 일반화된 형태)
    coords: [(x1, y1), (x2, y2), ..., (xn, yn)]
    """
    n = len(coords)
    area = 0.0
    for i in range(n):
        j = (i + 1) % n
        area += coords[i][0] * coords[j][1]
        area -= coords[j][0] * coords[i][1]
    return abs(area) / 2.0

# 테스트
# 예제 1: d=10, h1=3, h2=4
print(f"Quad Area (d=10, h=3,4): {area_quadrilateral_split(10, 3, 4)}") # Expected: 35.0

# 예제 2: (0,0), (4,0), (4,3), (0,5)
points = [(0,0), (4,0), (4,3), (0,5)]
print(f"Polygon Area (Shoelace): {area_irregular_polygon_coords(points)}") # Expected: 16.0
```

## 5. 요약 (Summary)

- **기본 전략:** 공식을 모르는 사각형은 대각선을 그어 삼각형 2개로 쪼갠다.
- **공식:** $\text{Area} = \frac{1}{2} d (h_1 + h_2)$.
- **좌표:** 좌표가 주어지면 축에 평행한 선을 긋거나 신발끈 공식을 사용한다.

# 25. 사각형 (Quadrilaterals)

## 1. 개요 (Overview)

사각형(Quadrilateral)은 4개의 변과 4개의 꼭짓점을 가진 다각형입니다. 모든 사각형의 내각의 합은 $360^\circ$입니다. AMC 8에서는 평행사변형, 직사각형, 마름모, 정사각형, 사다리꼴 등 특수한 사각형들의 성질과 넓이 공식이 주로 출제됩니다.

## 2. 주요 사각형의 종류 및 성질 (Types & Properties)

### 2.1 평행사변형 (Parallelogram)

두 쌍의 대변이 각각 평행한 사각형입니다.

- **성질:** 대변의 길이가 같고, 대각의 크기가 같습니다. 대각선은 서로를 이등분합니다.
- **넓이:** $\text{Area} = \text{base} \times \text{height}$.

### 2.2 직사각형 (Rectangle)

네 내각의 크기가 모두 $90^\circ$인 평행사변형입니다.

- **성질:** 대각선의 길이가 서로 같습니다.
- **넓이:** $\text{Area} = \text{length} \times \text{width}$.

### 2.3 마름모 (Rhombus)

네 변의 길이가 모두 같은 평행사변형입니다.

- **성질:** 대각선이 서로 **수직이등분**합니다.
- **넓이:** $\text{Area} = \frac{d_1 \times d_2}{2}$ (대각선의 곱 나누기 2).

### 2.4 정사각형 (Square)

네 변의 길이가 같고, 네 내각이 모두 $90^\circ$인 사각형입니다. (직사각형이자 마름모)

- **넓이:** $s^2$ 또는 $\frac{d^2}{2}$.

### 2.5 사다리꼴 (Trapezoid)

한 쌍의 대변이 평행한 사각형입니다.

- **넓이:** $\text{Area} = \frac{(\text{base}_1 + \text{base}_2) \times \text{height}}{2}$.
- **등변사다리꼴 (Isosceles Trapezoid):** 평행하지 않은 두 변의 길이가 같고, 밑각의 크기가 같습니다.

### 2.6 연 (Kite)

이웃하는 두 변의 길이가 각각 같은 사각형입니다.

- **성질:** 대각선이 서로 수직입니다.
- **넓이:** $\frac{d_1 \times d_2}{2}$.

## 3. 예제 (Examples)

### 예제 1: 평행사변형의 각도

평행사변형 $ABCD$에서 $\angle A = 60^\circ$일 때, $\angle B$의 크기는?

- 이웃하는 두 각의 합은 $180^\circ$입니다 (동측내각).
- $\angle B = 180 - 60 = 120^\circ$.

### 예제 2: 마름모의 넓이

대각선의 길이가 각각 10과 24인 마름모의 한 변의 길이는?

- 대각선은 서로 수직이등분하므로, 직각삼각형의 두 변은 5와 12입니다.
- 피타고라스 정리: $\sqrt{5^2 + 12^2} = \sqrt{25 + 144} = 13$.
- 넓이: $\frac{10 \times 24}{2} = 120$.

### 예제 3: 사다리꼴 넓이

윗변이 4, 아랫변이 10, 높이가 5인 사다리꼴의 넓이는?

- $\frac{(4 + 10) \times 5}{2} = \frac{14 \times 5}{2} = 35$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 다양한 사각형의 넓이를 계산하는 함수입니다.

```python
def area_parallelogram(base, height):
    return base * height

def area_trapezoid(base1, base2, height):
    return (base1 + base2) * height / 2

def area_rhombus_or_kite(d1, d2):
    """마름모 또는 연의 넓이 (대각선 이용)"""
    return (d1 * d2) / 2

def rhombus_side_length(d1, d2):
    """마름모의 대각선이 주어졌을 때 한 변의 길이 (피타고라스)"""
    import math
    return math.sqrt((d1/2)**2 + (d2/2)**2)

# 테스트
print(f"Parallelogram Area (b=10, h=5): {area_parallelogram(10, 5)}") # Expected: 50
print(f"Trapezoid Area (4, 10, 5): {area_trapezoid(4, 10, 5)}") # Expected: 35.0
print(f"Rhombus Area (d1=10, d2=24): {area_rhombus_or_kite(10, 24)}") # Expected: 120.0
print(f"Rhombus Side (d1=10, d2=24): {rhombus_side_length(10, 24)}") # Expected: 13.0
```

## 5. 요약 (Summary)

- **포함 관계:** 정사각형 $\subset$ 직사각형/마름모 $\subset$ 평행사변형 $\subset$ 사다리꼴.
- **대각선:** 마름모와 연은 대각선이 수직이므로 넓이는 $d_1 d_2 / 2$.
- **사다리꼴:** (윗변+아랫변) $\times$ 높이 / 2.

# 12.2 고급 방정식 풀이 기법 (Advanced Equation Solving Techniques)

## 1. 개요 (Overview)

기본적인 대입법과 가감법으로 풀기 어려운 복잡한 형태의 연립방정식을 다룹니다. 식의 구조적 특징(대칭성, 순환성)을 파악하여 효율적으로 해결하는 것이 핵심입니다. 무작정 계산하기보다 식 전체를 관찰하는 통찰력이 필요합니다.

## 2. 주요 기법 (Key Techniques)

### 2.1 치환을 이용한 풀이 (Substitution)

복잡한 식이나 반복되는 부분을 하나의 문자(예: $A, B$)로 치환하여 간단한 일차방정식으로 만듭니다.

- **역수 형태:** $\frac{1}{x}, \frac{1}{y}$ 등이 나올 때 $A = \frac{1}{x}, B = \frac{1}{y}$로 치환합니다.

### 2.2 순환형 연립방정식 (Cyclic Systems)

변수들이 꼬리에 꼬리를 물고 이어지는 형태입니다.

- **합의 꼴:** $x+y=a, y+z=b, z+x=c$
  - **전략:** 모든 식을 변변 더합니다. $2(x+y+z) = a+b+c$를 이용해 전체 합을 먼저 구합니다.
- **곱의 꼴:** $xy=a, yz=b, zx=c$
  - **전략:** 모든 식을 변변 곱합니다. $(xyz)^2 = abc$를 이용해 전체 곱을 구합니다.

### 2.3 대칭식 (Symmetric Equations)

$x$와 $y$를 바꾸어도 식이 변하지 않는 경우입니다.

- **전략:** 합 $S = x+y$와 곱 $P = xy$를 새로운 미지수로 두고 풉니다.
- 이차방정식의 근과 계수의 관계를 활용할 수도 있습니다.

## 3. 예제 (Examples)

### 예제 1: 순환형 (합)

$$
\begin{cases}
x + y = 3 \\
y + z = 4 \\
z + x = 5
\end{cases}
$$

1.  세 식을 모두 더함: $2(x+y+z) = 3+4+5 = 12 \implies x+y+z = 6$.
2.  $z$ 구하기: $(x+y+z) - (x+y) = 6 - 3 \implies z = 3$.
3.  $x$ 구하기: $(x+y+z) - (y+z) = 6 - 4 \implies x = 2$.
4.  $y$ 구하기: $(x+y+z) - (z+x) = 6 - 5 \implies y = 1$.
5.  **해:** $(2, 1, 3)$

### 예제 2: 역수 형태

$$
\begin{cases}
\frac{2}{x} + \frac{3}{y} = 13 \\
\frac{1}{x} - \frac{1}{y} = -1
\end{cases}
$$

1.  $A = \frac{1}{x}, B = \frac{1}{y}$로 치환.
2.  $2A + 3B = 13, A - B = -1$.
3.  두 번째 식에서 $A = B - 1$. 대입하면 $2(B-1) + 3B = 13 \implies 5B = 15 \implies B = 3$.
4.  $A = 2$.
5.  원래 변수로 복귀: $x = \frac{1}{A} = \frac{1}{2}, y = \frac{1}{B} = \frac{1}{3}$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 순환형 연립방정식($x+y, y+z, z+x$)을 푸는 함수입니다.

```python
def solve_cyclic_sum_system(sum_xy, sum_yz, sum_zx):
    """
    x + y = A
    y + z = B
    z + x = C
    형태의 연립방정식 풀이
    """
    # 1. 전체 합 구하기: 2(x+y+z) = A + B + C
    total_sum_2 = sum_xy + sum_yz + sum_zx
    total_sum = total_sum_2 / 2

    # 2. 각 변수 구하기
    # z = (x+y+z) - (x+y)
    z = total_sum - sum_xy
    x = total_sum - sum_yz
    y = total_sum - sum_zx

    return x, y, z

# 테스트: 예제 1번
sol = solve_cyclic_sum_system(3, 4, 5)
print(f"Solution: x={sol[0]}, y={sol[1]}, z={sol[2]}") # Expected: 2.0, 1.0, 3.0
```

## 5. 요약 (Summary)

- **구조 파악:** 무작정 대입하기 전에 식 전체를 더하거나 곱해본다.
- **치환:** 복잡한 부분(역수 등)은 문자로 바꿔서 단순화한다.
- **대칭성:** 합과 곱을 이용한다.
- **순환형:** 전체의 합(또는 곱)을 구하는 것이 지름길이다.

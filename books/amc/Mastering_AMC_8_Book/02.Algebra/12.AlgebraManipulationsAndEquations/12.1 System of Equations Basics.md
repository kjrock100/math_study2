# 12.1 연립방정식의 기초 (System of Equations Basics)

## 1. 개요 (Overview)

연립방정식(System of Equations)은 두 개 이상의 미지수를 포함하는 방정식들이 묶여 있는 형태입니다. 이 방정식들을 동시에 만족시키는 미지수의 값(해)을 찾는 것이 목표입니다. 기하학적으로는 두 직선의 **교점(Intersection)**을 찾는 것과 같습니다.

## 2. 풀이 방법 (Solving Methods)

### 2.1 대입법 (Substitution Method)

한 방정식을 한 변수에 대해 정리한 후, 이를 다른 방정식에 대입하여 미지수를 줄이는 방법입니다.

- **언제 사용하나?** 한 변수의 계수가 1이나 -1일 때 가장 효율적입니다.
- **절차:**
  1.  식 하나를 $y = \dots$ 또는 $x = \dots$ 꼴로 정리합니다.
  2.  다른 식의 해당 변수 자리에 대입합니다.
  3.  일차방정식을 풉니다.

### 2.2 가감법 (Elimination Method)

두 식을 더하거나 빼서 한 변수를 소거(Eliminate)하는 방법입니다.

- **언제 사용하나?** 변수의 계수가 같거나, 정수배를 통해 쉽게 맞출 수 있을 때 유용합니다.
- **절차:**
  1.  소거하려는 변수의 계수의 절댓값이 같아지도록 적절한 수를 곱합니다.
  2.  두 식을 더하거나 빼서 변수 하나를 없앱니다.
  3.  남은 변수의 값을 구하고, 이를 원래 식에 대입하여 나머지 변수를 구합니다.

## 3. 예제 (Examples)

### 예제 1: 대입법

$$
\begin{cases}
y = 2x + 1 \\
3x + y = 11
\end{cases}
$$

1.  첫 번째 식($y$)을 두 번째 식에 대입: $3x + (2x + 1) = 11$
2.  정리: $5x + 1 = 11 \implies 5x = 10 \implies x = 2$
3.  $x=2$를 첫 번째 식에 대입: $y = 2(2) + 1 = 5$
4.  **해:** $(2, 5)$

### 예제 2: 가감법

$$
\begin{cases}
2x + 3y = 8 \\
2x - y = 4
\end{cases}
$$

1.  $x$의 계수가 같으므로 위 식에서 아래 식을 뺍니다.
    $$ (2x + 3y) - (2x - y) = 8 - 4 $$
2.  정리: $4y = 4 \implies y = 1$
3.  $y=1$을 두 번째 식에 대입: $2x - 1 = 4 \implies 2x = 5 \implies x = 2.5$
4.  **해:** $(2.5, 1)$

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 2원 1차 연립방정식($ax+by=e, cx+dy=f$)을 크래머 공식(Cramer's Rule)으로 해결하는 함수입니다.

```python
def solve_linear_system(eq1, eq2):
    """
    eq1: (a, b, e) -> ax + by = e
    eq2: (c, d, f) -> cx + dy = f
    해 (x, y)를 반환
    """
    a, b, e = eq1
    c, d, f = eq2

    # 행렬식(Determinant) 계산: ad - bc
    det = a * d - b * c

    if det == 0:
        return None # 해가 없거나 무수히 많음 (평행 또는 일치)

    # 크래머 공식 적용
    x = (e * d - b * f) / det
    y = (a * f - e * c) / det

    return x, y

# 테스트: 예제 2번 (2x + 3y = 8, 2x - y = 4)
solution = solve_linear_system((2, 3, 8), (2, -1, 4))
print(f"Solution: x={solution[0]}, y={solution[1]}") # Expected: x=2.5, y=1.0
```

## 5. 요약 (Summary)

- **대입법:** $y=$ 꼴로 정리하여 넣기.
- **가감법:** 계수를 맞춰서 더하거나 빼기.
- **해의 의미:** 두 그래프의 교점.

# 12.4 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 12.1장부터 12.3장까지 다룬 연립방정식의 기초, 고급 풀이 기법, 그리고 문장제 문제 해결 능력을 종합하여 실전 문제를 다룹니다. 특히 미지수의 개수보다 식의 개수가 적어 정수 조건이 필요한 **부정방정식**이나, 식을 조작하여 한 번에 답을 구하는 **대수적 테크닉**이 요구되는 문제들을 집중적으로 살펴봅니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **가치와 개수 혼합 (Value & Count):** 동전(Coins)이나 티켓(Tickets) 문제처럼 개수의 합과 가치의 합이 주어지는 유형.
2.  **순환형 시스템 (Cyclic Systems):** $x+y, y+z, z+x$와 같이 변수들이 순환하며 주어지는 경우.
3.  **부정방정식 (Diophantine Equations):** 식은 하나인데 미지수가 2개인 경우, "양의 정수"라는 조건을 이용하여 해를 찾는 유형.

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: 동전 문제 (Coins)

주머니에 10센트(Dimes)와 25센트(Quarters)가 총 15개 들어있다. 총액이 $2.55일 때, 25센트 동전은 몇 개인가?

- **식 세우기:**
  1.  개수: $d + q = 15$
  2.  가치: $10d + 25q = 255$ (단위를 센트로 통일)
- **풀이 (가감법):**
  - 첫 식에 10을 곱함: $10d + 10q = 150$
  - 두 식을 뺌: $(10d + 25q) - (10d + 10q) = 255 - 150$
  - $15q = 105 \implies q = 7$
- **정답:** 7개.

### 문제 2: 3변수 순환형 (Cyclic Sums)

세 수 $a, b, c$에 대하여 $a+b=10, b+c=14, c+a=16$일 때, $abc$의 값은?

- **전략:** 세 식을 모두 더합니다.
  - $2(a+b+c) = 10 + 14 + 16 = 40$
  - $a+b+c = 20$
- **개별 변수 구하기:**
  - $c = (a+b+c) - (a+b) = 20 - 10 = 10$
  - $a = (a+b+c) - (b+c) = 20 - 14 = 6$
  - $b = (a+b+c) - (c+a) = 20 - 16 = 4$
- **정답:** $abc = 6 \times 4 \times 10 = 240$.

### 문제 3: 부정방정식 (Diophantine Equation)

사과($x$)는 3달러, 오렌지($y$)는 5달러이다. 과일을 섞어서 정확히 50달러어치를 샀을 때, 가능한 사과의 최대 개수는? (단, $x, y$는 1 이상의 정수)

- **식:** $3x + 5y = 50$
- **전략:** $y$에 작은 정수를 대입하며 $x$가 정수가 되는지 확인하거나, 모듈러 연산을 사용합니다.
  - $x = \frac{50 - 5y}{3}$. $50 - 5y$가 3의 배수여야 함.
  - $y=1 \implies 50-5=45$ (3의 배수 O) $\rightarrow x=15$.
  - $y=2 \implies 50-10=40$ (X)
  - $y=3 \implies 50-15=35$ (X)
  - $y=4 \implies 50-20=30$ (O) $\rightarrow x=10$.
- **최대 개수:** $x$가 최대가 되려면 $y$가 최소여야 하므로, $y=1$일 때 $x=15$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 부정방정식 $ax + by = c$의 모든 비음수 정수 해를 찾는 함수입니다.

```python
def solve_diophantine_bruteforce(a, b, c):
    """
    ax + by = c 를 만족하는 비음수 정수 (x, y) 쌍을 모두 찾음.
    a, b, c는 양의 정수라고 가정.
    """
    solutions = []
    # x의 가능한 최대 범위: 0부터 c/a 까지
    max_x = c // a

    for x in range(max_x + 1):
        remainder = c - (a * x)

        # 남은 값이 b로 나누어 떨어지는지 확인
        if remainder >= 0 and remainder % b == 0:
            y = remainder // b
            solutions.append((x, y))

    return solutions

def solve_cyclic_system(ab, bc, ca):
    # a+b=ab, b+c=bc, c+a=ca
    total_sum = (ab + bc + ca) / 2
    c = total_sum - ab
    a = total_sum - bc
    b = total_sum - ca
    return a, b, c
```

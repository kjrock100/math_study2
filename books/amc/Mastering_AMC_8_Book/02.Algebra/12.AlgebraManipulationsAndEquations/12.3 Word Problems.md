# 12.3 실전 응용 문제 (Word Problems)

## 1. 개요 (Overview)

연립방정식의 가장 큰 목적은 실생활의 문제나 문장제 문제(Word Problems)를 해결하는 것입니다. 이 장에서는 문장으로 된 조건을 수학적 식(Equation)으로 번역하는 능력과 유형별 접근 전략을 다룹니다.

## 2. 주요 유형 및 전략 (Key Types & Strategies)

### 2.1 나이 문제 (Age Problems)

- **변수 설정:** 현재 나이를 $x, y$로 둡니다.
- **시간 이동:** "$n$년 후"는 $+n$, "$n$년 전"은 $-n$을 합니다.
- **주의:** "A는 B보다 3배 많다" $\rightarrow A = 3B$.

### 2.2 자릿수 문제 (Digit Problems)

- **표현:** 십의 자리 숫자가 $t$, 일의 자리 숫자가 $u$인 두 자리 수의 **값(Value)**은 $10t + u$입니다.
- **조건:** "자릿수의 합" $\rightarrow t+u$, "자릿수를 바꾼 수" $\rightarrow 10u + t$.

### 2.3 다리와 머리 문제 (Legs and Heads)

동물 농장 문제나 동전 문제처럼 **개수(Count)**의 합과 **가치(Value/Legs)**의 합이 주어지는 유형입니다.

- 식 1: $x + y = \text{Total Heads}$
- 식 2: $ax + by = \text{Total Legs}$ ($a, b$는 각각의 다리 수)

## 3. 예제 (Examples)

### 예제 1: 나이 문제

현재 아버지는 아들보다 4배 나이가 많습니다. 20년 후에는 아버지가 아들의 2배가 됩니다. 현재 아들의 나이는?

1.  현재: $F = 4S$
2.  20년 후: $(F + 20) = 2(S + 20)$
3.  대입: $(4S + 20) = 2S + 40$
4.  풀이: $2S = 20 \implies S = 10$.
5.  **정답:** 10세.

### 예제 2: 닭과 토끼 (Chickens and Rabbits)

농장에 닭과 토끼가 있다. 머리의 수는 35개이고, 다리의 수는 94개이다. 토끼는 몇 마리인가?

1.  변수: 닭 $c$, 토끼 $r$.
2.  머리 식: $c + r = 35$
3.  다리 식: $2c + 4r = 94$
4.  풀이 (가감법):
    - 첫 식에 2를 곱함: $2c + 2r = 70$
    - 다리 식에서 뺌: $(2c + 4r) - (2c + 2r) = 94 - 70 \implies 2r = 24 \implies r = 12$.
5.  **정답:** 12마리.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 "머리와 다리" 유형의 연립방정식을 푸는 일반화된 함수입니다.

```python
def solve_heads_legs(total_heads, total_legs, legs_type1=2, legs_type2=4):
    """
    머리 수와 다리 수가 주어졌을 때 두 동물의 마리 수를 구함.
    기본값: 닭(2)과 토끼(4)
    식:
    x + y = heads
    L1*x + L2*y = legs
    """
    # 크래머 공식 또는 대입법 유도
    # y = heads - x
    # L1*x + L2*(heads - x) = legs
    # L1*x + L2*heads - L2*x = legs
    # x(L1 - L2) = legs - L2*heads
    # x = (legs - L2*heads) / (L1 - L2)

    if legs_type1 == legs_type2:
        return None # 다리 수가 같으면 구별 불가

    numerator = total_legs - (legs_type2 * total_heads)
    denominator = legs_type1 - legs_type2

    x = numerator / denominator
    y = total_heads - x

    # 정수 해인지 확인 (음수이거나 소수점이면 불가능한 경우)
    if x < 0 or y < 0 or not x.is_integer():
        return "No valid integer solution"

    return int(x), int(y)

# 테스트: 예제 2번
chickens, rabbits = solve_heads_legs(35, 94)
print(f"Chickens: {chickens}, Rabbits: {rabbits}") # Expected: 23, 12
```

## 5. 요약 (Summary)

- **문장 번역:** "Is"는 $=$, "More than"은 $+$, "Times"는 $\times$.
- **단위 통일:** 센트/달러, 년/월 등 단위가 섞여 있으면 하나로 통일해야 합니다.
- **검산:** 구한 답이 문제의 뜻(음수 나이 불가, 정수 마리 수 등)에 맞는지 확인합니다.

# 14.5 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 14.1장부터 14.4장까지 학습한 등차수열, 등비수열, 특수 수열(거듭제곱의 합, 망원 급수, 몇 급수)의 개념을 종합하여 실전 문제를 해결합니다. 단순한 공식 적용을 넘어, 수열의 규칙을 발견하고 적절한 합 공식을 선택하여 적용하는 능력을 기릅니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **규칙 찾기 (Pattern Recognition):** 복잡해 보이는 수열에서 주기성이나 그룹(군수열) 규칙을 찾아내는 문제.
2.  **망원 급수 응용 (Telescoping Series):** 부분분수 분해를 통해 중간 항들을 소거하여 합을 구하는 문제.
3.  **무한등비급수 활용 (Infinite Geometric Series):** 공이 튀어 오르는 문제나 프랙탈 도형의 넓이/둘레 구하기.
4.  **몇 급수 계산:** 등차수열과 등비수열이 곱해진 형태의 합 구하기.

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: 망원 급수 ($\frac{1}{n(n+2)}$ 형태)

다음 급수의 합을 구하시오.
$$ \frac{1}{1 \times 3} + \frac{1}{3 \times 5} + \frac{1}{5 \times 7} + \dots + \frac{1}{99 \times 101} $$

- **전략:** 부분분수 공식 $\frac{1}{AB} = \frac{1}{B-A}(\frac{1}{A} - \frac{1}{B})$를 사용합니다. 여기서 $B-A=2$입니다.
- **변형:** $\frac{1}{2} \left[ (\frac{1}{1} - \frac{1}{3}) + (\frac{1}{3} - \frac{1}{5}) + \dots + (\frac{1}{99} - \frac{1}{101}) \right]$
- **소거:** 중간 항들이 모두 사라지고 $\frac{1}{2} (1 - \frac{1}{101})$만 남습니다.
- **계산:** $\frac{1}{2} \times \frac{100}{101} = \frac{50}{101}$.

### 문제 2: 공 튀기기 (Bouncing Ball)

높이 10m에서 떨어뜨린 공이 바닥에 닿을 때마다 이전 높이의 $\frac{1}{2}$만큼 튀어 오른다. 공이 멈출 때까지 움직인 총 거리는?

- **전략:** 내려가는 거리와 올라가는 거리를 구분합니다.
- **내려감:** $10 + 5 + 2.5 + \dots$ (첫항 10, 공비 1/2)
- **올라감:** $5 + 2.5 + 1.25 + \dots$ (첫항 5, 공비 1/2)
- **공식:** $S = \frac{a}{1-r}$.
  - Down: $\frac{10}{1-0.5} = 20$.
  - Up: $\frac{5}{1-0.5} = 10$.
- **총 거리:** $20 + 10 = 30$m.
- **다른 방법:** $10 + 2 \times (5 + 2.5 + \dots) = 10 + 2 \times 10 = 30$.

### 문제 3: 군수열 (Grouped Sequence)

수열 $\frac{1}{1}, \frac{1}{2}, \frac{2}{2}, \frac{1}{3}, \frac{2}{3}, \frac{3}{3}, \frac{1}{4}, \dots$ 에서 50번째 항은?

- **규칙:** 분모가 $n$인 항이 $n$개씩 묶여 있습니다. (1군: 1개, 2군: 2개, ...)
- **누적 항 수:** $1 + 2 + 3 + \dots + k \approx 50$.
  - $k=9$일 때 합: $\frac{9 \times 10}{2} = 45$.
  - $k=10$일 때 합: 55.
- **위치:** 50번째 항은 10군의 5번째 항입니다 ($50 - 45 = 5$).
- **10군의 형태:** $\frac{1}{10}, \frac{2}{10}, \dots, \frac{10}{10}$.
- **정답:** $\frac{5}{10}$ (또는 $\frac{1}{2}$).

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 망원 급수, 공 튀기기, 군수열 문제를 해결하는 함수입니다.

```python
def solve_telescoping_series(n_start, n_end, step):
    """
    Sum of 1/(n*(n+step)) type series
    Using formula: 1/step * (1/n_start - 1/(n_end + step))
    Note: This simple formula applies when terms cancel out perfectly leaving only first and last parts.
    For 1/(1*3) + ... + 1/(99*101), start=1, end=99, step=2.
    """
    # 실제 항들을 더해서 검증
    total = 0
    current = n_start
    while current <= n_end:
        total += 1 / (current * (current + step))
        current += step
    return total

def solve_bouncing_ball(initial_height, bounce_ratio):
    """
    Total distance = h + 2 * (h*r + h*r^2 + ...)
    = h + 2 * h*r / (1-r)
    """
    if bounce_ratio >= 1:
        return float('inf')

    up_distance = (initial_height * bounce_ratio) / (1 - bounce_ratio)
    total_distance = initial_height + 2 * up_distance
    return total_distance

def find_nth_term_group_sequence(n):
    """
    1/1, 1/2, 2/2, 1/3, 2/3, 3/3 ... 에서 n번째 항 찾기
    """
    count = 0
    group = 1
    while count + group < n:
        count += group
        group += 1

    # group군의 (n - count)번째 항
    numerator = n - count
    denominator = group
    return numerator, denominator

# 테스트
print(f"문제 1 (Telescoping): {solve_telescoping_series(1, 99, 2):.4f}") # Expected: 50/101 approx 0.4950
print(f"문제 2 (Bouncing Ball): {solve_bouncing_ball(10, 0.5)}")       # Expected: 30.0
print(f"문제 3 (50th term): {find_nth_term_group_sequence(50)}")       # Expected: (5, 10)
```

## 5. 요약 (Summary)

- **망원 급수:** $\frac{1}{AB} = \frac{1}{B-A}(\frac{1}{A}-\frac{1}{B})$ 공식을 기억하세요.
- **무한등비급수:** 공이 튀어 오를 때는 처음 높이는 한 번, 튀어 오르는 높이는 왕복(두 번)임을 주의하세요.
- **군수열:** 몇 번째 그룹에 속하는지 파악하기 위해 $\frac{n(n+1)}{2}$ 합 공식을 활용하세요.

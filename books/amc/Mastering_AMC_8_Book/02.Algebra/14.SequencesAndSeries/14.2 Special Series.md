# 14.2 특수 수열과 급수 (Special Series)

## 1. 개요 (Overview)

등차수열 외에 AMC 8에서 자주 등장하는 **등비수열(Geometric Sequence)**과, 자연수의 거듭제곱의 합, 그리고 항들이 서로 지워지는 **망원 급수(Telescoping Series)** 등을 다룹니다. 이러한 특수 수열들의 규칙과 합 공식을 이해하는 것은 복잡한 계산 문제를 빠르게 해결하는 열쇠입니다.

## 2. 등비수열 (Geometric Sequences)

연속하는 두 항의 비(Ratio)가 일정한 수열입니다.

### 2.1 핵심 요소

- **공비 (Common Ratio, $r$):** 곱해지는 일정한 수. ($r = a_{n+1} / a_n$)
- **일반항:** $a_n = a_1 \cdot r^{n-1}$

### 2.2 등비수열의 합 ($S_n$)

첫째항이 $a_1$, 공비가 $r$일 때 $n$항까지의 합입니다.
$$ S_n = \frac{a_1(r^n - 1)}{r - 1} = \frac{a_1(1 - r^n)}{1 - r} \quad (\text{단, } r \neq 1) $$

- 만약 $r=1$이면, $S_n = n \cdot a_1$입니다.

## 3. 거듭제곱의 합 (Sum of Powers)

자연수 $1$부터 $n$까지의 합, 제곱의 합, 세제곱의 합 공식입니다.

1.  **자연수의 합 (Triangular Numbers):**
    $$ \sum\_{k=1}^n k = \frac{n(n+1)}{2} $$
2.  **제곱의 합 (Sum of Squares):**
    $$ \sum\_{k=1}^n k^2 = \frac{n(n+1)(2n+1)}{6} $$
3.  **세제곱의 합 (Sum of Cubes):**
    $$ \sum\_{k=1}^n k^3 = \left[ \frac{n(n+1)}{2} \right]^2 $$

## 4. 망원 급수 (Telescoping Series)

이웃한 항들이 서로 상쇄되어(Cancel out) 처음과 끝만 남는 형태의 급수입니다. **부분분수 분해(Partial Fraction Decomposition)**가 대표적입니다.

- **공식:**
  $$ \frac{1}{k(k+1)} = \frac{1}{k} - \frac{1}{k+1} $$
- **전개:**
  $$ \sum\_{k=1}^n \frac{1}{k(k+1)} = \left(1 - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \dots + \left(\frac{1}{n} - \frac{1}{n+1}\right) = 1 - \frac{1}{n+1} $$

## 5. 예제 (Examples)

### 예제 1: 등비수열의 합

$1, 2, 4, 8, \dots, 512$의 합은?

- $a_1 = 1, r = 2$.
- 마지막 항 $512 = 2^9$이므로, 항의 개수 $n = 10$ ($2^0$부터 $2^9$).
- $S_{10} = \frac{1(2^{10} - 1)}{2 - 1} = 1024 - 1 = 1023$.

### 예제 2: 제곱의 합

$1^2 + 2^2 + \dots + 10^2$의 값은?

- $n=10$.
- Sum $= \frac{10(11)(21)}{6} = \frac{2310}{6} = 385$.

### 예제 3: 부분분수

$\frac{1}{1 \times 2} + \frac{1}{2 \times 3} + \dots + \frac{1}{99 \times 100}$의 값은?

- $1 - \frac{1}{100} = \frac{99}{100}$.

## 6. 코드 구현 (Code Implementation)

Python을 사용하여 등비수열의 합, 거듭제곱의 합, 망원 급수를 계산하는 함수입니다.

```python
def sum_geometric_sequence(a1, r, n):
    """등비수열의 합 (Geometric Series Sum)"""
    if r == 1:
        return a1 * n
    return a1 * (r**n - 1) / (r - 1)

def sum_squares(n):
    """1^2 + ... + n^2"""
    return n * (n + 1) * (2 * n + 1) // 6

def sum_cubes(n):
    """1^3 + ... + n^3"""
    sum_n = n * (n + 1) // 2
    return sum_n ** 2

def sum_telescoping(n):
    """Sum of 1/(k(k+1)) from k=1 to n"""
    # 결과는 n / (n + 1)
    return n / (n + 1)

# 테스트
print(f"Geometric Sum (1, 2, ..., 512): {sum_geometric_sequence(1, 2, 10)}") # Expected: 1023.0
print(f"Sum of Squares (1 to 10): {sum_squares(10)}") # Expected: 385
print(f"Sum of Cubes (1 to 10): {sum_cubes(10)}")     # Expected: 3025 (55^2)
print(f"Telescoping Sum (n=99): {sum_telescoping(99)}") # Expected: 0.99
```

## 7. 요약 (Summary)

- **등비수열 합:** $\frac{a(r^n-1)}{r-1}$.
- **제곱의 합:** $\frac{n(n+1)(2n+1)}{6}$.
- **망원 급수:** 중간 항들이 모두 지워지고 양 끝만 남는다.

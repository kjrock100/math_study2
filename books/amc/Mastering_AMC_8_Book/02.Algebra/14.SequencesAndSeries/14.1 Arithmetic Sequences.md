# 14.1 등차수열 (Arithmetic Sequences)

## 1. 개요 (Overview)

등차수열(Arithmetic Sequence)은 연속하는 두 항의 차이가 일정한 수열을 말합니다. 이 일정한 차이를 **공차(Common Difference, $d$)**라고 합니다. AMC 8에서는 수열의 규칙 찾기, 특정 항 구하기, 그리고 수열의 합을 구하는 문제가 자주 출제됩니다.

## 2. 핵심 공식 (Key Formulas)

### 2.1 일반항 (The $n$-th Term)

첫째항이 $a_1$이고 공차가 $d$일 때, $n$번째 항 $a_n$은 다음과 같습니다.
$$ a_n = a_1 + (n-1)d $$

### 2.2 항의 개수 (Number of Terms)

수열의 시작($a_1$)과 끝($a_n$), 그리고 공차($d$)를 알 때 항의 개수 $n$을 구하는 공식입니다. (가장 많이 실수하는 부분이니 주의하세요!)
$$ n = \frac{a_n - a_1}{d} + 1 $$

- **팁:** "차이를 공차로 나누고 1을 더한다"로 기억하세요.

### 2.3 등차수열의 합 (Sum of Arithmetic Series)

첫째항부터 $n$항까지의 합 $S_n$은 **(첫 항 + 끝 항) $\times$ 항의 개수 $\div$ 2** 입니다.
$$ S_n = \frac{n(a_1 + a_n)}{2} $$

## 3. 등차중항 (Arithmetic Mean)

세 수 $a, b, c$가 순서대로 등차수열을 이룰 때, 가운데 항 $b$는 양옆 항의 평균입니다.
$$ b = \frac{a+c}{2} \quad \text{또는} \quad 2b = a+c $$

## 4. 예제 (Examples)

### 예제 1: $n$번째 항 구하기

수열 $2, 5, 8, 11, \dots$ 의 50번째 항은?

- $a_1 = 2$, $d = 3$.
- $a_{50} = 2 + (50-1) \times 3 = 2 + 49 \times 3 = 2 + 147 = 149$.

### 예제 2: 항의 개수 구하기

수열 $3, 7, 11, \dots, 99$ 의 항의 개수는?

- $a_1 = 3$, $a_n = 99$, $d = 4$.
- $n = \frac{99 - 3}{4} + 1 = \frac{96}{4} + 1 = 24 + 1 = 25$개.

### 예제 3: 수열의 합 구하기

$1$부터 $100$까지의 자연수의 합은? (가우스의 일화)

- $a_1 = 1$, $a_{100} = 100$, $n = 100$.
- $S_{100} = \frac{100(1 + 100)}{2} = 50 \times 101 = 5050$.

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 등차수열의 $n$번째 항, 항의 개수, 합을 계산하는 함수입니다.

```python
def get_nth_term(a1, d, n):
    """등차수열의 n번째 항 반환"""
    return a1 + (n - 1) * d

def count_terms(a1, an, d):
    """등차수열의 항의 개수 반환"""
    if (an - a1) % d != 0:
        return None # 등차수열이 성립하지 않는 경우
    return (an - a1) // d + 1

def sum_arithmetic_sequence(a1, an, n=None, d=None):
    """
    등차수열의 합 반환.
    n(개수)이 주어지면 바로 계산, 없으면 d(공차)를 이용해 n을 구함.
    """
    if n is None:
        if d is None:
            raise ValueError("n 또는 d 중 하나는 반드시 필요합니다.")
        n = count_terms(a1, an, d)

    return n * (a1 + an) / 2

# 테스트
# 예제 1: 2, 5, 8... 의 50번째 항
print(f"50th term: {get_nth_term(2, 3, 50)}") # Expected: 149

# 예제 2: 3, 7, ..., 99 의 항의 개수
cnt = count_terms(3, 99, 4)
print(f"Number of terms: {cnt}") # Expected: 25

# 예제 3: 1 ~ 100 합
total = sum_arithmetic_sequence(1, 100, n=100)
print(f"Sum 1 to 100: {total}") # Expected: 5050.0
```

## 6. 요약 (Summary)

- **일반항:** $a_n = a_1 + (n-1)d$.
- **개수:** $\frac{\text{끝} - \text{처음}}{\text{공차}} + 1$. (가장 중요!)
- **합:** $\frac{\text{개수} \times (\text{처음} + \text{끝})}{2}$.
- **원리:** 평균값 $\times$ 개수.

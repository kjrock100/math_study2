# 15.1 평균, 중앙값, 최빈값의 기초 (Mean, Median, Mode Fundamentals)

## 1. 개요 (Overview)

통계학에서 자료의 특징을 하나의 수로 요약하여 나타내는 값을 **대푯값(Measures of Central Tendency)**이라고 합니다. AMC 8에서는 가장 대표적인 대푯값인 평균(Mean), 중앙값(Median), 최빈값(Mode)의 정의와 계산 방법, 그리고 이들의 성질을 묻는 문제가 자주 출제됩니다.

## 2. 핵심 개념 (Core Concepts)

### 2.1 평균 (Mean / Arithmetic Mean)

자료의 총합을 자료의 개수로 나눈 값입니다.
$$ \text{Mean} = \frac{\text{Sum of terms}}{\text{Number of terms}} $$

- **특징:** 모든 자료의 값을 반영하므로 극단적인 값(Outlier)에 영향을 많이 받습니다.
- **응용:** (평균) $\times$ (개수) = (총합) 공식을 자주 사용합니다.

### 2.2 중앙값 (Median)

자료를 **크기 순서대로 나열했을 때** 정중앙에 위치하는 값입니다.

- **자료의 개수($n$)가 홀수일 때:** 정확히 가운데 있는 수. ($\frac{n+1}{2}$번째 수)
- **자료의 개수($n$)가 짝수일 때:** 가운데 있는 두 수의 평균. ($\frac{n}{2}$번째와 $\frac{n}{2}+1$번째의 평균)
- **특징:** 극단적인 값에 영향을 받지 않습니다.

### 2.3 최빈값 (Mode)

자료 중에서 가장 자주 나타나는(빈도수가 가장 높은) 값입니다.

- **특징:** 하나일 수도 있고, 여러 개일 수도 있으며, 없을 수도 있습니다(모든 값이 한 번씩만 나오면).

### 2.4 범위 (Range)

자료의 흩어진 정도를 나타내는 산포도의 일종으로, 최댓값과 최솟값의 차이입니다.
$$ \text{Range} = \text{Max} - \text{Min} $$

## 3. 예제 (Examples)

### 예제 1: 기본 계산

자료 집합 $\{2, 5, 9, 2, 7\}$에 대하여 평균, 중앙값, 최빈값을 구하시오.

1.  **정렬:** $2, 2, 5, 7, 9$
2.  **평균:** $\frac{2+2+5+7+9}{5} = \frac{25}{5} = 5$.
3.  **중앙값:** 5개 중 3번째 값 $\rightarrow 5$.
4.  **최빈값:** 2가 두 번 나옴 $\rightarrow 2$.

### 예제 2: 짝수 개의 중앙값

자료 집합 $\{10, 20, 30, 40\}$의 중앙값은?

- 가운데 두 수: 20, 30.
- 중앙값: $\frac{20+30}{2} = 25$.

### 예제 3: 미지수 찾기

5개의 수 $3, 8, 10, x, 15$의 평균이 10일 때, $x$의 값은?

- 총합 = 평균 $\times$ 개수 = $10 \times 5 = 50$.
- $3 + 8 + 10 + x + 15 = 36 + x$.
- $36 + x = 50 \implies x = 14$.

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 리스트의 평균, 중앙값, 최빈값을 계산하는 함수입니다.

```python
from collections import Counter

def calculate_statistics(data):
    """
    주어진 리스트의 평균, 중앙값, 최빈값을 계산
    """
    if not data:
        return None, None, None

    # 1. 평균 (Mean)
    mean_val = sum(data) / len(data)

    # 2. 중앙값 (Median)
    sorted_data = sorted(data)
    n = len(data)
    if n % 2 == 1:
        median_val = sorted_data[n // 2]
    else:
        # 짝수 개일 경우 가운데 두 수의 평균
        mid1 = sorted_data[n // 2 - 1]
        mid2 = sorted_data[n // 2]
        median_val = (mid1 + mid2) / 2

    # 3. 최빈값 (Mode)
    counts = Counter(data)
    max_freq = max(counts.values())
    # 최빈값이 여러 개일 수 있음
    modes = [k for k, v in counts.items() if v == max_freq]

    return mean_val, median_val, modes

# 테스트
data1 = [2, 5, 9, 2, 7]
mean1, med1, mode1 = calculate_statistics(data1)
print(f"Data: {data1}")
print(f"Mean: {mean1}, Median: {med1}, Mode: {mode1}")
# Expected: Mean 5.0, Median 5, Mode [2]

data2 = [10, 20, 30, 40]
mean2, med2, mode2 = calculate_statistics(data2)
print(f"Data: {data2}")
print(f"Mean: {mean2}, Median: {med2}, Mode: {mode2}")
# Expected: Mean 25.0, Median 25.0, Mode [10, 20, 30, 40] (모두 1번씩 등장)
```

## 5. 요약 (Summary)

- **평균:** $\frac{\text{합}}{\text{개수}}$. 총합을 구할 때 유용함.
- **중앙값:** 순서대로 나열했을 때 가운데 값. (짝수 개면 평균).
- **최빈값:** 가장 많이 나온 값.
- **팁:** 중앙값을 구할 때는 반드시 **정렬(Sorting)**부터 해야 합니다.

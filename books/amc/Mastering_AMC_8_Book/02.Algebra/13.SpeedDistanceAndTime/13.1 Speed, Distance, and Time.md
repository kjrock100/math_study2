# 13.1 속력, 거리, 시간 (Speed, Distance, and Time)

## 1. 개요 (Overview)

속력, 거리, 시간 문제는 AMC 8에서 가장 변별력 있는 유형 중 하나입니다. 11장에서 다룬 기본 공식($d=rt$)을 바탕으로, 두 물체가 움직이는 **상대 속도(Relative Speed)**, 물체의 길이를 고려해야 하는 **기차 문제**, 그리고 유속이 있는 **강물 문제** 등 심화된 상황을 다룹니다.

## 2. 상대 속도 (Relative Speed)

두 물체가 동시에 움직일 때, 서로에게 느껴지는 속도입니다.

### 2.1 마주 보고 이동 (Opposite Direction)

서로를 향해 다가오거나, 서로 멀어질 때입니다.

- **상대 속도:** $V_{\text{rel}} = V_1 + V_2$
- **만나는 시간:** $t = \frac{\text{Distance}}{V_1 + V_2}$

### 2.2 같은 방향 이동 (Same Direction)

뒤에서 추격하거나, 앞서 나갈 때입니다.

- **상대 속도:** $V_{\text{rel}} = |V_1 - V_2|$
- **따라잡는 시간:** $t = \frac{\text{Distance Gap}}{|V_1 - V_2|}$

## 3. 기차 문제 (Train Problems)

기차처럼 길이가 긴 물체가 이동할 때는 **자신의 길이**를 이동 거리에 포함해야 합니다.

- **전봇대/사람 통과:** 이동 거리 = 기차 길이 ($L_{\text{train}}$)
- **다리/터널 통과:** 이동 거리 = 기차 길이 + 다리 길이 ($L_{\text{train}} + L_{\text{bridge}}$)
- **다른 기차 통과:** 이동 거리 = 두 기차 길이의 합 ($L_1 + L_2$)

## 4. 강물과 배 (Boats and Currents)

강물의 흐름이 배의 속력에 영향을 줍니다.

- **배의 자체 속력:** $b$ (정지한 물에서의 속력)
- **강물의 속력:** $c$
- **하류로 갈 때 (Downstream):** 물이 밀어주므로 빨라집니다. ($b + c$)
- **상류로 갈 때 (Upstream):** 물을 거슬러 가므로 느려집니다. ($b - c$)

## 5. 예제 (Examples)

### 예제 1: 추격 문제

도둑이 60km/h로 도망가고 1시간 뒤에 경찰이 90km/h로 쫓아갑니다. 경찰이 도둑을 잡는 데 걸리는 시간은?

1.  **초기 거리 차이:** 도둑이 1시간 동안 간 거리 = 60km.
2.  **상대 속도:** $90 - 60 = 30$ km/h.
3.  **시간:** $\frac{60}{30} = 2$시간.

### 예제 2: 기차와 터널

길이 200m인 기차가 800m 길이의 터널을 완전히 통과하는 데 50초가 걸렸다. 기차의 속력은?

1.  **총 이동 거리:** $200 + 800 = 1000$m.
2.  **시간:** 50초.
3.  **속력:** $\frac{1000}{50} = 20$ m/s.

### 예제 3: 강물 문제

배가 강을 거슬러 올라갈 때(Upstream) 속력은 10mph, 내려올 때(Downstream) 속력은 20mph이다. 강물의 속력은?

1.  $b - c = 10$
2.  $b + c = 20$
3.  두 식을 뺌: $(b+c) - (b-c) = 20 - 10 \implies 2c = 10 \implies c = 5$ mph.

## 6. 코드 구현 (Code Implementation)

Python을 사용하여 추격 문제와 기차 통과 문제를 해결하는 함수입니다.

```python
def solve_catch_up_time(speed_chaser, speed_runner, head_start_distance):
    """
    추격 문제: 뒤에 있는 사람이 앞선 사람을 잡는 시간 계산
    """
    if speed_chaser <= speed_runner:
        return float('inf') # 절대 못 잡음

    relative_speed = speed_chaser - speed_runner
    time = head_start_distance / relative_speed
    return time

def solve_train_crossing(train_length, object_length, speed_mps):
    """
    기차가 물체(다리, 터널 등)를 통과하는 시간 계산
    단위: 미터(m), 초(s)
    """
    total_distance = train_length + object_length
    time = total_distance / speed_mps
    return time

# 테스트
print(f"추격 시간 (경찰 90, 도둑 60, 거리차 60): {solve_catch_up_time(90, 60, 60)} hours") # Expected: 2.0
print(f"터널 통과 (기차 200m, 터널 800m, 속도 20m/s): {solve_train_crossing(200, 800, 20)} seconds") # Expected: 50.0
```

## 7. 요약 (Summary)

- **상대 속도:** 반대 방향은 합($+$), 같은 방향은 차($-$).
- **기차:** 이동 거리에 **기차 길이**를 반드시 더한다.
- **강물:** 흐르는 방향이면 더하고($+$), 거스르면 뺀다($-$).

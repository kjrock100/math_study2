# 11.2 속력과 일 (Rate and Work)

## 1. 개요 (Overview)

속력(Speed/Rate)과 일(Work) 문제는 AMC 8에서 매우 빈번하게 출제되는 유형입니다. 두 유형은 물리적 상황은 다르지만, 수학적으로는 **$Quantity = Rate \times Time$**이라는 동일한 기본 구조를 공유합니다.

## 2. 거리, 속력, 시간 (Distance, Speed, Time)

### 2.1 기본 공식

$$ d = r \times t $$

- **$d$ (Distance):** 이동 거리
- **$r$ (Rate):** 속력 (Speed)
- **$t$ (Time):** 걸린 시간

### 2.2 평균 속력 (Average Speed)

학생들이 가장 많이 실수하는 개념입니다. 평균 속력은 속력들의 단순 평균(산술 평균)이 아닙니다.

$$ \text{Average Speed} = \frac{\text{Total Distance}}{\text{Total Time}} $$

- 예: 갈 때 $v_1$, 올 때 $v_2$로 왕복한 경우 평균 속력은 $\frac{v_1 + v_2}{2}$가 아니라, 조화 평균인 $\frac{2v_1v_2}{v_1+v_2}$가 됩니다.

## 3. 일률 문제 (Work Problems)

### 3.1 기본 공식

$$ w = r \times t $$

- **$w$ (Work):** 일의 양. 문제에서 구체적인 양이 주어지지 않으면 보통 **1**로 둡니다.
- **$r$ (Rate):** 일률. 단위 시간당 하는 일의 양입니다. ($r = \frac{1}{t}$)
- **$t$ (Time):** 일을 끝내는 데 걸리는 시간.

### 3.2 협동 작업 (Combined Work)

여러 사람이 함께 일할 때는 **일률(Rate)을 더합니다**. (시간을 더하면 안 됩니다!)

$$ r*{\text{combined}} = r_1 + r_2 + \dots $$
$$ \frac{1}{t*{\text{total}}} = \frac{1}{t_1} + \frac{1}{t_2} + \dots $$

## 4. 예제 (Examples)

### 예제 1: 평균 속력 (Average Speed)

어떤 사람이 산을 오를 때는 3km/h, 내려올 때는 같은 길을 6km/h로 이동했습니다. 전체 평균 속력은?

- 거리를 $d$라고 가정합니다.
- 올라갈 때 시간: $t_1 = \frac{d}{3}$
- 내려올 때 시간: $t_2 = \frac{d}{6}$
- 총 거리: $2d$
- 총 시간: $\frac{d}{3} + \frac{d}{6} = \frac{2d}{6} + \frac{d}{6} = \frac{3d}{6} = \frac{d}{2}$
- **평균 속력:** $\frac{2d}{d/2} = 4$ km/h. (3과 6의 평균인 4.5가 아님에 주의)

### 예제 2: 함께 일하기 (Working Together)

페인트칠을 하는 데 A는 4시간, B는 12시간이 걸립니다. 둘이 함께 칠하면 몇 시간이 걸립니까?

- A의 일률: $\frac{1}{4}$ (시간당)
- B의 일률: $\frac{1}{12}$ (시간당)
- 함께 일할 때 일률: $\frac{1}{4} + \frac{1}{12} = \frac{3}{12} + \frac{1}{12} = \frac{4}{12} = \frac{1}{3}$
- **걸리는 시간:** 일률이 $\frac{1}{3}$이므로, 역수인 **3시간**이 걸립니다.

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 평균 속력과 협동 작업 시간을 계산하는 함수입니다.

```python
def calculate_average_speed(speed1, speed2):
    """
    왕복 구간(동일 거리)에서의 평균 속력 계산 (조화 평균)
    """
    # Total Distance = 2d
    # Total Time = d/s1 + d/s2 = d(s1+s2)/(s1*s2)
    # Avg Speed = 2d / (d(s1+s2)/(s1*s2)) = 2*s1*s2 / (s1+s2)
    return (2 * speed1 * speed2) / (speed1 + speed2)

def calculate_work_together_time(time_a, time_b):
    """
    A가 혼자 할 때 time_a, B가 혼자 할 때 time_b 걸릴 경우
    함께 할 때 걸리는 시간 계산
    """
    rate_a = 1 / time_a
    rate_b = 1 / time_b
    combined_rate = rate_a + rate_b
    return 1 / combined_rate

# 테스트
print(f"Average Speed (3km/h, 6km/h): {calculate_average_speed(3, 6)} km/h") # Expected: 4.0
print(f"Work Together (4h, 12h): {calculate_work_together_time(4, 12)} hours") # Expected: 3.0
```

## 6. 요약 (Summary)

- **공식:** $d=rt$, $w=rt$.
- **평균 속력:** $\frac{\text{Total Distance}}{\text{Total Time}}$. (절대 속력끼리 더해서 나누지 말 것)
- **일률:** 함께 일할 때는 **Rate(속도)**를 더한다. ($r_{total} = r_1 + r_2$)
- **역수 관계:** 시간($t$)과 일률($r$)은 역수 관계이다.

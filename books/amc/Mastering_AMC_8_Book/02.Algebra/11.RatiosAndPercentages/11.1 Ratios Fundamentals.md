# 11.1 비율의 기초 (Ratios Fundamentals)

## 1. 개요 (Overview)

비율(Ratio)은 두 개 이상의 수나 양을 비교하는 방법입니다. 수학적으로 $a:b$ 또는 분수 형태 $\frac{a}{b}$로 표현하며, AMC 8에서는 기본적인 비례식 계산부터 복잡한 연비(Combined Ratio) 문제까지 다양하게 출제됩니다.

## 2. 핵심 개념 (Core Concepts)

### 2.1 표기법 및 정의

- **표기:** $a$ 대 $b$의 비는 $a:b$ 또는 $\frac{a}{b}$로 씁니다.
- **성질:** 비율의 각 항에 0이 아닌 같은 수를 곱하거나 나누어도 비율의 값은 변하지 않습니다. (예: $2:3 = 4:6$)

### 2.2 부분 대 부분 vs 부분 대 전체

- **Part-to-Part:** 사과가 2개, 배가 3개일 때, 사과 대 배의 비는 $2:3$.
- **Part-to-Whole:** 사과 대 전체 과일의 비는 $2:(2+3) = 2:5$.

### 2.3 비례식 (Proportions)

두 비율이 같음을 나타내는 식입니다.
$$ a:b = c:d \implies \frac{a}{b} = \frac{c}{d} \implies ad = bc $$

- **내항의 곱 = 외항의 곱** 성질을 이용하여 미지수를 구합니다.

## 3. 연비 (Combined Ratios)

두 개의 비율 $a:b$와 $b:c$가 주어졌을 때, 이를 하나로 합쳐 $a:b:c$를 만드는 것입니다.

- 공통된 항($b$)의 값을 최소공배수(LCM)로 통일하여 계산합니다.

## 4. 예제 (Examples)

### 예제 1: 비율 간단히 하기

$12:18$을 가장 간단한 자연수의 비로 나타내시오.

- 최대공약수(GCD)인 6으로 나눕니다.
- $12 \div 6 : 18 \div 6 = 2:3$

### 예제 2: 비례식 풀기

$3:5 = x:20$ 일 때 $x$의 값은?

- 내항의 곱 = 외항의 곱: $5x = 3 \times 20$
- $5x = 60 \rightarrow x = 12$
- 또는 배수 관계 이용: $5 \times 4 = 20$ 이므로 $3 \times 4 = 12$.

### 예제 3: 연비 구하기

$A:B = 2:3$이고 $B:C = 4:5$일 때, $A:B:C$는?

- 공통인 $B$를 통일합니다. 3과 4의 최소공배수는 12입니다.
- $A:B = 2:3 = 8:12$ ($\times 4$)
- $B:C = 4:5 = 12:15$ ($\times 3$)
- 결과: $A:B:C = 8:12:15$

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 비율을 약분하고, 연비를 계산하는 함수입니다.

```python
import math

def simplify_ratio(a, b):
    """두 수의 비율을 가장 간단한 형태로 약분"""
    common = math.gcd(a, b)
    return a // common, b // common

def combine_ratios(a_b, b_c):
    """
    a:b = n1:d1, b:c = n2:d2 일 때 a:b:c 반환
    입력: a_b=(n1, d1), b_c=(n2, d2)
    """
    n1, d1 = a_b # A:B
    n2, d2 = b_c # B:C

    # 공통 항인 B(d1과 n2)의 최소공배수 구하기
    lcm_b = math.lcm(d1, n2)

    # 비율 조정
    factor1 = lcm_b // d1
    factor2 = lcm_b // n2

    new_a = n1 * factor1
    new_b = lcm_b # d1 * factor1 == n2 * factor2
    new_c = d2 * factor2

    return new_a, new_b, new_c

# 테스트
print(f"Simplify 12:18 -> {simplify_ratio(12, 18)}") # Expected: (2, 3)
print(f"Combine A:B=2:3, B:C=4:5 -> {combine_ratios((2,3), (4,5))}") # Expected: (8, 12, 15)
```

## 6. 요약 (Summary)

- **비율:** 두 수의 나눗셈 관계 ($a:b$).
- **비례식:** $a:b = c:d \iff ad=bc$.
- **연비:** 공통된 항을 최소공배수로 맞춰서 연결한다.

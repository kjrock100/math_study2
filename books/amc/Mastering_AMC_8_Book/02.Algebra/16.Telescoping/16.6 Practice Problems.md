# 16.6 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 16.1장부터 16.5장까지 학습한 망원 급수(Telescoping Series)와 망원 곱(Telescoping Products)의 개념을 종합하여 실전 문제를 해결합니다. 부분분수 분해, 무리식의 유리화, 로그의 성질, 팩토리얼의 변형 등 다양한 기법을 적재적소에 활용하여 복잡한 급수를 간단히 계산하는 능력을 기릅니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **팩토리얼 급수 (Factorial Series):** $k \cdot k!$ 또는 $\frac{k}{(k+1)!}$ 형태를 변형하여 소거하는 유형.
2.  **로그 급수 (Logarithmic Series):** 로그의 덧셈을 진수의 곱셈으로 바꾸어 약분하거나, $\log a - \log b$ 꼴로 변형하는 유형.
3.  **복잡한 부분분수 (Complex Partial Fractions):** 분모에 항이 3개 있거나($\frac{1}{n(n+1)(n+2)}$), 항의 차이가 큰 경우($\frac{1}{n(n+3)}$).

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: 팩토리얼 분수 합

$$ S = \sum\_{k=1}^{99} \frac{k}{(k+1)!} $$

- **전략:** 분자 $k$를 $(k+1) - 1$로 변형합니다.
- **변형:** $\frac{k+1 - 1}{(k+1)!} = \frac{k+1}{(k+1)!} - \frac{1}{(k+1)!} = \frac{1}{k!} - \frac{1}{(k+1)!}$
- **나열:** $(\frac{1}{1!} - \frac{1}{2!}) + (\frac{1}{2!} - \frac{1}{3!}) + \dots + (\frac{1}{99!} - \frac{1}{100!})$
- **소거:** 중간 항들이 모두 사라지고 $\frac{1}{1!} - \frac{1}{100!}$만 남습니다.
- **정답:** $1 - \frac{1}{100!}$

### 문제 2: 로그 급수

$$ S = \sum\_{k=1}^{n} \log \left( \frac{k}{k+1} \right) $$

- **전략:** $\log(\frac{A}{B}) = \log A - \log B$ 성질 이용.
- **변형:** $\sum (\log k - \log(k+1))$
- **나열:** $(\log 1 - \log 2) + (\log 2 - \log 3) + \dots + (\log n - \log(n+1))$
- **소거:** $\log 1 - \log(n+1) = 0 - \log(n+1) = -\log(n+1)$
- **정답:** $-\log(n+1)$ (또는 $\log \frac{1}{n+1}$)

### 문제 3: 항이 3개인 부분분수

$$ S = \sum\_{k=1}^{n} \frac{1}{k(k+1)(k+2)} $$

- **전략:** $\frac{1}{AB} = \frac{1}{B-A}(\frac{1}{A} - \frac{1}{B})$ 공식을 응용합니다. $A=k, B=k+2$로 보면 가운데 $(k+1)$을 포함하여 쪼갤 수 있습니다.
- **변형:** $\frac{1}{k(k+1)(k+2)} = \frac{1}{2} \left( \frac{1}{k(k+1)} - \frac{1}{(k+1)(k+2)} \right)$
  - 확인: $\frac{1}{2} (\frac{k+2 - k}{k(k+1)(k+2)}) = \frac{1}{2} (\frac{2}{k(k+1)(k+2)}) = \text{원식}$
- **치환:** $b_k = \frac{1}{k(k+1)}$이라 하면, 식은 $\frac{1}{2} (b_k - b_{k+1})$
- **결과:** $\frac{1}{2} (b_1 - b_{n+1}) = \frac{1}{2} \left( \frac{1}{1 \cdot 2} - \frac{1}{(n+1)(n+2)} \right)$
- **정답:** $\frac{1}{4} - \frac{1}{2(n+1)(n+2)}$

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 위 문제들의 합을 계산하고 공식을 검증하는 함수입니다.

```python
import math

def solve_factorial_fraction_sum(n):
    """
    Sum of k / (k+1)! from k=1 to n
    Formula: 1 - 1/(n+1)!
    """
    # 실제 합 계산
    total = 0
    for k in range(1, n + 1):
        total += k / math.factorial(k + 1)

    # 공식 값
    formula_val = 1 - 1 / math.factorial(n + 1)

    return total, formula_val

def solve_log_sum(n):
    """
    Sum of log(k/(k+1)) from k=1 to n
    Formula: -log(n+1)
    """
    total = 0
    for k in range(1, n + 1):
        total += math.log(k / (k + 1))

    formula_val = -math.log(n + 1)
    return total, formula_val

def solve_three_term_partial_fraction(n):
    """
    Sum of 1/(k(k+1)(k+2)) from k=1 to n
    Formula: 1/4 - 1/(2(n+1)(n+2))
    """
    total = 0
    for k in range(1, n + 1):
        total += 1 / (k * (k + 1) * (k + 2))

    formula_val = 0.25 - 1 / (2 * (n + 1) * (n + 2))
    return total, formula_val

# 테스트
n_val = 10
print(f"Factorial Sum (n={n_val}): {solve_factorial_fraction_sum(n_val)}")
print(f"Log Sum (n={n_val}): {solve_log_sum(n_val)}")
print(f"3-Term Fraction (n={n_val}): {solve_three_term_partial_fraction(n_val)}")
```

## 5. 요약 (Summary)

- **팩토리얼:** $\frac{k}{(k+1)!} = \frac{1}{k!} - \frac{1}{(k+1)!}$
- **로그:** $\sum \log(\frac{a_k}{a_{k+1}}) = \log a_1 - \log a_{n+1}$
- **3항 부분분수:** 양 끝을 잡고 쪼개면($\frac{1}{n(n+2)}$ 처럼) 가운데 항이 공통으로 들어가며 망원 급수가 됩니다.

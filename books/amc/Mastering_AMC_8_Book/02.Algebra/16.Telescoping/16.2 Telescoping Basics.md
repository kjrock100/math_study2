# 16.2 망원 급수의 기초 (Telescoping Basics)

## 1. 개요 (Overview)

망원 급수(Telescoping Series)는 급수의 각 항이 이웃한 항과 상쇄되어(Cancel out) 중간 항들이 모두 사라지고, **처음과 끝의 일부 항만 남는** 형태의 급수를 말합니다. 마치 망원경(Telescope)을 접으면 길이가 줄어드는 것처럼, 긴 급수가 간단한 식으로 축소된다고 하여 붙여진 이름입니다.

## 2. 기본 원리 (Basic Principle)

일반항 $a_n$을 두 항의 차 $b_n - b_{n+1}$ (또는 $b_{n+1} - b_n$) 형태로 변형할 수 있다면, 급수의 합은 다음과 같이 계산됩니다.

$$ \sum*{k=1}^n (b_k - b*{k+1}) = (b*1 - b_2) + (b_2 - b_3) + \dots + (b_n - b*{n+1}) $$
$$ = b*1 - b*{n+1} $$

중간의 $-b_2$와 $+b_2$, $-b_3$와 $+b_3$ 등이 서로 지워지기 때문입니다.

## 3. 주요 유형 (Common Types)

### 3.1 부분분수 분해 (Partial Fractions)

분모가 두 수의 곱으로 되어 있는 경우입니다.
$$ \frac{1}{n(n+k)} = \frac{1}{k} \left( \frac{1}{n} - \frac{1}{n+k} \right) $$
가장 흔한 형태는 $k=1$인 경우입니다: $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$.

### 3.2 무리식 (Radicals)

분모의 유리화(Rationalization)를 통해 차의 꼴로 만듭니다.
$$ \frac{1}{\sqrt{n} + \sqrt{n+1}} = \sqrt{n+1} - \sqrt{n} $$

### 3.3 로그 (Logarithms)

로그의 성질 $\log(\frac{a}{b}) = \log a - \log b$를 이용합니다.
$$ \log\left(1 + \frac{1}{n}\right) = \log\left(\frac{n+1}{n}\right) = \log(n+1) - \log n $$

## 4. 예제 (Examples)

### 예제 1: 부분분수

$$ S = \sum\_{k=1}^{99} \frac{1}{k(k+1)} $$

- 변형: $\sum (\frac{1}{k} - \frac{1}{k+1})$
- 나열: $(1 - \frac{1}{2}) + (\frac{1}{2} - \frac{1}{3}) + \dots + (\frac{1}{99} - \frac{1}{100})$
- 결과: $1 - \frac{1}{100} = \frac{99}{100}$

### 예제 2: 무리식

$$ S = \sum\_{k=1}^{99} \frac{1}{\sqrt{k} + \sqrt{k+1}} $$

- 유리화: $\frac{\sqrt{k+1} - \sqrt{k}}{(k+1)-k} = \sqrt{k+1} - \sqrt{k}$
- 나열: $(\sqrt{2} - \sqrt{1}) + (\sqrt{3} - \sqrt{2}) + \dots + (\sqrt{100} - \sqrt{99})$
- 소거 후 남는 항: $\sqrt{100} - \sqrt{1} = 10 - 1 = 9$

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 망원 급수의 합을 계산하고, 실제로 중간 항들이 상쇄되는지 확인하는 함수입니다.

```python
import math

def solve_telescoping_fraction(n):
    """
    Sum of 1/(k(k+1)) from k=1 to n
    Formula: 1 - 1/(n+1)
    """
    return 1 - 1/(n + 1)

def solve_telescoping_radical(n):
    """
    Sum of 1/(sqrt(k) + sqrt(k+1)) from k=1 to n
    Formula: sqrt(n+1) - sqrt(1)
    """
    return math.sqrt(n + 1) - 1

def solve_telescoping_log(n):
    """
    Sum of log(1 + 1/k) from k=1 to n
    Formula: log(n+1) - log(1) = log(n+1)
    """
    return math.log(n + 1)

# 검증
n_val = 99
print(f"Partial Fraction Sum (n={n_val}): {solve_telescoping_fraction(n_val)}") # Expected: 0.99
print(f"Radical Sum (n={n_val}): {solve_telescoping_radical(n_val)}")         # Expected: 9.0
print(f"Log Sum (n={n_val}): {solve_telescoping_log(n_val):.4f}")             # Expected: log(100) approx 4.605
```

## 6. 요약 (Summary)

- **핵심:** 일반항을 **차(Difference)**의 형태로 변형한다. ($f(k) - f(k+1)$)
- **계산:** 나열해서 지우고, **맨 앞**과 **맨 뒤**의 항만 계산한다.
- **주의:** 항이 두 칸씩 건너뛰며 지워지는 경우($\frac{1}{n(n+2)}$)에는 앞의 두 개, 뒤의 두 개가 남을 수 있다.

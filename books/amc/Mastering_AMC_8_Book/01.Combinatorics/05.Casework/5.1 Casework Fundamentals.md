# 5.1 케이스 분류의 기초 (Casework Fundamentals)

## 1. 개요 (Overview)

케이스 분류(Casework)는 복잡한 문제를 한 번에 풀기 어려울 때, 문제를 **서로 겹치지 않는 여러 개의 작은 부분(Case)**으로 나누어 해결하는 전략입니다. 공식 하나로 해결되지 않는 문제나 제약 조건이 까다로운 문제에서 가장 기본적이면서도 강력한 도구로 사용됩니다.

## 2. 핵심 원칙 (Core Principles)

성공적인 케이스 분류를 위해서는 다음 두 가지 조건을 만족해야 합니다 (MECE 원칙).

1.  **상호 배타적 (Mutually Exclusive):** 각 케이스는 서로 겹치지 않아야 합니다. (중복 방지)
2.  **전체 포괄 (Collectively Exhaustive):** 모든 케이스를 합치면 전체 경우가 되어야 합니다. (누락 방지)

## 3. 분류 기준 설정 (Setting Criteria)

케이스를 나누는 기준을 잘 잡는 것이 문제 해결의 열쇠입니다.

- **크기 순서:** 가장 큰 수나 가장 작은 수를 기준으로 분류.
- **위치:** 특정 자릿수(맨 앞자리 등)의 숫자를 기준으로 분류.
- **합 또는 차:** 숫자의 합이나 차이가 특정 값이 되는 경우로 분류.
- **기하학적 특성:** 가장 긴 변의 길이를 기준으로 분류.

## 4. 예제 (Examples)

### 예제 1: 자릿수의 합 (Digit Sum)

각 자릿수의 합이 5인 3자리 양의 정수의 개수는?

- **기준:** 백의 자리 숫자 (1부터 5까지 가능)
- **Case 1 (백의 자리가 1):** 나머지 두 자리 합이 4.
  - (0,4), (1,3), (2,2), (3,1), (4,0) $\rightarrow$ 5개
- **Case 2 (백의 자리가 2):** 나머지 두 자리 합이 3.
  - (0,3), (1,2), (2,1), (3,0) $\rightarrow$ 4개
- **Case 3 (백의 자리가 3):** 나머지 두 자리 합이 2.
  - (0,2), (1,1), (2,0) $\rightarrow$ 3개
- **Case 4 (백의 자리가 4):** 나머지 두 자리 합이 1.
  - (0,1), (1,0) $\rightarrow$ 2개
- **Case 5 (백의 자리가 5):** 나머지 두 자리 합이 0.
  - (0,0) $\rightarrow$ 1개
- **총합:** $5 + 4 + 3 + 2 + 1 = 15$개

### 예제 2: 삼각형 만들기 (Integer Triangles)

둘레의 길이가 12이고 각 변의 길이가 정수인 삼각형의 개수는? (단, 합동인 삼각형은 하나로 칩니다. $a \le b \le c$)

- **조건:** $a + b + c = 12$, $a + b > c$ (삼각형 성립 조건)
  - $a + b > c \implies a + b + c > 2c \implies 12 > 2c \implies c < 6$
  - 또한 $c$는 가장 긴 변이므로 $c \ge 12/3 = 4$.
  - 따라서 가능한 $c$는 4, 5.
- **Case 1 ($c = 5$):** $a + b = 7$. ($a \le b \le 5$)
  - (2, 5, 5), (3, 4, 5) $\rightarrow$ 2개
- **Case 2 ($c = 4$):** $a + b = 8$. ($a \le b \le 4$)
  - (4, 4, 4) $\rightarrow$ 1개
- **총합:** $2 + 1 = 3$개

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 모든 경우를 탐색(Brute Force)함으로써 케이스 분류의 결과를 검증할 수 있습니다.

```python
def solve_digit_sum_problem():
    count = 0
    # 3자리 수: 100 ~ 999
    for num in range(100, 1000):
        digits = [int(d) for d in str(num)]
        if sum(digits) == 5:
            count += 1
    print(f"자릿수 합이 5인 3자리 수 개수: {count}") # Expected: 15

def solve_triangle_problem():
    perimeter = 12
    triangles = []
    # a <= b <= c 조건을 만족하는 조합 찾기
    # c는 최소 perimeter/3, 최대 perimeter/2 미만
    for c in range(1, perimeter):
        for b in range(1, c + 1):
            for a in range(1, b + 1):
                if a + b + c == perimeter:
                    if a + b > c: # 삼각형 성립 조건
                        triangles.append((a, b, c))

    print(f"둘레가 {perimeter}인 삼각형 개수: {len(triangles)}") # Expected: 3
    print(f"목록: {triangles}")

solve_digit_sum_problem()
solve_triangle_problem()
```

## 6. 요약 (Summary)

- **체계적 접근:** 무작정 세기보다는 명확한 기준(가장 큰 수, 첫 자리 등)을 세워 분류해야 합니다.
- **중복/누락 주의:** 분류한 케이스가 겹치지 않는지, 빠진 경우는 없는지 항상 점검해야 합니다.
- **조건 확인:** 삼각형 문제처럼 숨겨진 제약 조건($a+b>c$)을 놓치지 않도록 주의합니다.

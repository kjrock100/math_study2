# 2.1 조합의 기초 (Combinations Fundamentals)

## 1. 개요 (Overview)

조합(Combination)은 서로 다른 $n$개의 원소 중에서 **순서를 고려하지 않고** $r$개를 선택하는 것을 의미합니다. 순열(Permutation)이 "선택하여 나열"하는 것이라면, 조합은 단순히 "선택(Selection)"하는 것에 초점을 맞춥니다. 즉, $\{A, B\}$와 $\{B, A\}$를 같은 경우로 취급합니다.

## 2. 정의 및 공식 (Definition & Formula)

### 2.1 정의

서로 다른 $n$개에서 $r$개를 택하는 조합의 수는 다음과 같이 표기합니다.
$$ C(n, r), \quad \_nC_r, \quad \binom{n}{r} $$
(미국 수학 교과 과정이나 AMC에서는 $\binom{n}{r}$ 표기를 주로 사용합니다.)

### 2.2 공식 유도

조합의 수는 순열의 수에서 선택된 $r$개의 원소를 나열하는 경우의 수($r!$)를 나누어 구합니다. 순서를 무시하기 위함입니다.

$$ \binom{n}{r} = \frac{P(n, r)}{r!} = \frac{n!}{r!(n-r)!} $$

- $n$: 전체 원소의 개수
- $r$: 선택할 원소의 개수 ($0 \le r \le n$)

## 3. 주요 성질 (Key Properties)

### 3.1 대칭성 (Symmetry)

$n$개 중에서 $r$개를 선택하는 것은, 선택되지 않을 $(n-r)$개를 선택하는 것과 같습니다.
$$ \binom{n}{r} = \binom{n}{n-r} $$
예: $\binom{10}{8} = \binom{10}{2}$ (계산이 훨씬 간편해짐)

### 3.2 경계값

- $\binom{n}{0} = 1$: 아무것도 선택하지 않는 방법은 1가지(공집합).
- $\binom{n}{n} = 1$: 모두 선택하는 방법은 1가지.
- $\binom{n}{1} = n$: 하나를 선택하는 방법은 $n$가지.

## 4. 예시 (Examples)

### 예제 1: 대표 선출

학생 5명(A, B, C, D, E) 중 청소 당번 2명을 뽑는 경우의 수:

- 순서가 중요하지 않음 (A, B가 뽑히나 B, A가 뽑히나 같음).
  $$ \binom{5}{2} = \frac{5 \times 4}{2 \times 1} = 10 $$

### 예제 2: 피자 토핑

5가지 토핑 중 3가지를 선택하는 방법의 수:
$$ \binom{5}{3} = \binom{5}{2} = \frac{5 \times 4}{2 \times 1} = 10 $$

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 조합의 수(값)를 계산하거나 실제 조합 리스트를 생성하는 방법입니다.

### 5.1 `math.comb` 사용 (값 계산)

Python 3.8 이상에서 지원하는 내장 함수입니다.

```python
import math

n = 5
r = 2

# 5C2 계산
count = math.comb(n, r)
print(f"C({n}, {r})의 경우의 수: {count}")
# 출력: 10
```

### 5.2 `itertools.combinations` 사용 (리스트 생성)

실제 선택된 원소들의 조합을 확인할 때 사용합니다.

```python
from itertools import combinations

items = ['A', 'B', 'C', 'D', 'E']
r = 2

# 조합 객체 생성 및 리스트 변환
comb_list = list(combinations(items, r))

print(f"생성된 조합: {comb_list}")
# 출력: [('A', 'B'), ('A', 'C'), ..., ('D', 'E')]
print(f"개수: {len(comb_list)}")
# 출력: 10
```

## 6. 요약 (Summary)

- **조합 vs 순열:** 순서가 없으면 조합($C$), 있으면 순열($P$).
- **공식:** $\binom{n}{r} = \frac{n!}{r!(n-r)!}$.
- **계산 팁:** $\binom{n}{r} = \binom{n}{n-r}$ 성질을 이용하여 $r$이 클 때는 작은 값으로 바꾸어 계산한다.

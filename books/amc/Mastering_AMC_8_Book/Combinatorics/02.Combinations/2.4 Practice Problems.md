# 2.4 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 2.1부터 2.3까지 학습한 조합(Combinations), 이항 계수(Binomial Coefficients), 그리고 다양한 제약 조건이 있는 문제들을 종합적으로 다룹니다. AMC 8 시험에서 자주 등장하는 대표적인 유형들을 통해 문제 해결 능력을 키웁니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **그룹 구성 (Forming Groups):** 특정 조건(성별, 직업 등)을 만족하는 위원회나 팀 만들기.
2.  **기하학적 개수 (Geometric Counting):** 점들을 연결하여 만들 수 있는 선분, 삼각형, 또는 다각형의 대각선 개수 구하기.
3.  **부분집합 (Subsets):** 특정 원소를 포함하거나 포함하지 않는 부분집합의 개수.
4.  **경로 찾기 (Grid Paths):** 격자무늬 길 찾기 (조합을 이용한 최단 경로).

## 3. 예제 문제 및 풀이 (Sample Problems & Solutions)

### 문제 1: 위원회 구성 (Committee Formation)

남학생 5명과 여학생 6명 중에서 남학생 2명, 여학생 2명으로 구성된 4명의 위원회를 뽑는 방법의 수는?

- **남학생 선택:** $\binom{5}{2} = 10$
- **여학생 선택:** $\binom{6}{2} = 15$
- **곱의 법칙 적용:** $10 \times 15 = 150$

### 문제 2: 다각형의 대각선 (Diagonals of a Polygon)

십각형(Decagon, 변이 10개)의 대각선 개수는?

- **전체 선분 개수:** 10개의 점 중 2개를 연결하는 모든 경우 $\binom{10}{2} = 45$
- **변(Side) 제외:** 다각형의 변(10개)은 대각선이 아님.
- **정답:** $45 - 10 = 35$
- **일반화 공식:** $\binom{n}{2} - n = \frac{n(n-3)}{2}$

### 문제 3: 피자 토핑 (Subsets)

피자 가게에서 6가지 토핑을 제공합니다. 토핑을 하나도 선택하지 않는 경우(치즈 피자)를 포함하여, 만들 수 있는 서로 다른 피자의 종류는 몇 가지인가?

- **해석:** 6개의 원소를 가진 집합의 부분집합 개수 구하기.
- **공식:** $2^n = 2^6 = 64$

## 4. 문제 해결 전략 (Problem Solving Strategies)

- **조건 분리:** "남학생 2명 AND 여학생 2명"처럼 조건이 독립적일 때는 각각 조합을 구한 후 곱합니다.
- **제외하기:** 대각선 문제처럼 전체 조합에서 조건에 맞지 않는 것(변)을 빼는 방식이 유용합니다.
- **이항 정리 활용:** "선택하거나 선택하지 않거나"의 문제는 $2^n$을 떠올립니다.

## 5. 코드 구현 (Code Implementation)

Python을 사용하여 위 문제들의 정답을 검증하는 코드입니다.

```python
import math

def nCr(n, r):
    return math.comb(n, r)

# 문제 1: 위원회 구성
men_ways = nCr(5, 2)
women_ways = nCr(6, 2)
total_committee = men_ways * women_ways
print(f"문제 1 정답: {total_committee}")  # Expected: 150

# 문제 2: 십각형 대각선
n = 10
total_lines = nCr(n, 2)
diagonals = total_lines - n
print(f"문제 2 정답: {diagonals}")  # Expected: 35

# 문제 3: 피자 토핑 (부분집합)
n_toppings = 6
total_pizzas = 2 ** n_toppings
print(f"문제 3 정답: {total_pizzas}")  # Expected: 64
```

## 6. 요약 (Summary)

- **곱의 법칙:** 서로 다른 그룹에서 각각 뽑을 때는 곱한다 ($\binom{n}{a} \times \binom{m}{b}$).
- **대각선:** $\binom{n}{2} - n$.
- **부분집합:** $2^n$.
- 조합 문제는 순서가 중요하지 않은 상황(팀, 도형, 토핑 등)에서 사용됨을 명심해야 합니다.

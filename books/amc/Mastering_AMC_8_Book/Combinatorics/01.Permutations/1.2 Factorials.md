# 1.2 팩토리얼 (Factorials) 정의 및 분석

## 1. 개요 (Overview)

팩토리얼(Factorial, 계승)은 1부터 어떤 양의 정수 $n$까지의 모든 정수를 곱한 것을 의미합니다. 순열(Permutation)과 조합(Combination) 계산의 핵심이 되는 연산이며, 기호로는 느낌표($!$)를 사용합니다.

## 2. 정의 (Definition)

$n$이 양의 정수일 때, $n$ 팩토리얼($n!$)은 다음과 같이 정의됩니다.

$$ n! = n \times (n-1) \times (n-2) \times \dots \times 2 \times 1 $$
$$ n! = \prod\_{k=1}^{n} k $$

### 0! 의 정의

$0!$은 1로 정의합니다.
$$ 0! = 1 $$
이는 순열 공식 $P(n, n) = \frac{n!}{(n-n)!} = \frac{n!}{0!}$이 $n!$이 되기 위한 논리적 일관성을 유지하기 위함이며, "아무것도 선택하지 않는 경우의 수는 1가지"라는 조합론적 해석과도 일치합니다.

## 3. 성질 (Properties)

### 3.1 재귀적 성질 (Recursive Property)

팩토리얼은 다음과 같은 재귀적 관계를 가집니다.
$$ n! = n \times (n-1)! $$
이 성질은 큰 수의 팩토리얼을 계산하거나 식을 약분할 때 유용하게 사용됩니다.

### 3.2 증가 속도

$n$이 커짐에 따라 $n!$의 값은 매우 빠르게 증가합니다.

- $3! = 6$
- $5! = 120$
- $10! = 3,628,800$

## 4. AMC 8 관련 주요 개념 (Key Concepts for AMC 8)

### 4.1 식의 약분 (Simplification)

분수 형태의 팩토리얼 계산 시, 공통된 인수를 약분하여 계산을 간단히 할 수 있습니다.
$$ \frac{8!}{6!} = \frac{8 \times 7 \times 6!}{6!} = 8 \times 7 = 56 $$

### 4.2 끝자리 0의 개수 (Trailing Zeros)

$n!$의 끝에 있는 0의 개수는 소인수분해 했을 때 인수 5의 개수에 의해 결정됩니다 (2의 개수는 항상 5보다 많으므로).
르장드르 공식(Legendre's Formula)을 활용하여 계산합니다.
$$ \text{Count} = \lfloor \frac{n}{5} \rfloor + \lfloor \frac{n}{25} \rfloor + \lfloor \frac{n}{125} \rfloor + \dots $$

## 5. 코드 구현 (Code Implementation)

### 5.1 Python `math` 모듈 사용

가장 효율적이고 권장되는 방법입니다.

```python
import math

n = 5
result = math.factorial(n)
print(f"{n}! = {result}")
# 출력: 5! = 120
```

### 5.2 재귀 함수 (Recursive Function) 구현

팩토리얼의 재귀적 정의를 코드로 표현한 방법입니다.

```python
def factorial_recursive(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial_recursive(n - 1)

print(factorial_recursive(5))
# 출력: 120
```

### 5.3 반복문 (Iterative) 구현

스택 오버플로우 위험 없이 구현하는 방법입니다.

```python
def factorial_iterative(n):
    result = 1
    for i in range(2, n + 1):
        result *= i
    return result

print(factorial_iterative(5))
# 출력: 120
```

## 6. 요약 (Summary)

- $n! = n \times (n-1) \times \dots \times 1$
- $0! = 1$
- $n! = n \times (n-1)!$
- AMC 8 문제에서는 팩토리얼의 약분과 끝자리 0의 개수를 묻는 문제가 자주 출제됩니다.

# 4.7 실전 문제 (Practice Problems)

## 1. 개요 (Overview)

이 장에서는 지금까지 학습한 확률의 모든 개념(기초 정의, 독립 vs 종속, 케이스 분류, 여사건)을 종합하여 AMC 8 수준의 심화 문제를 다룹니다. 또한, 개수를 셀 수 없는 연속적인 상황에서의 확률인 **기하학적 확률(Geometric Probability)** 개념을 도입하여 문제 해결의 폭을 넓힙니다.

## 2. 주요 문제 유형 (Key Problem Types)

1.  **복합 사건:** 독립 사건과 종속 사건이 혼합된 형태.
2.  **양말 뽑기 (Sock Drawer):** 비복원 추출 상황에서 짝을 맞추는 문제 (케이스 분류 + 종속 확률).
3.  **기하학적 확률:** 길이, 넓이, 부피의 비율을 이용한 확률 계산.
4.  **적어도 하나:** 여사건을 이용한 효율적인 계산.

## 3. 기하학적 확률 (Geometric Probability)

사건의 결과가 셀 수 있는 개수(Discrete)가 아니라, 길이(Length)나 넓이(Area)처럼 연속적인 값(Continuous)을 가질 때 사용합니다.

$$ P(E) = \frac{\text{사건 } E\text{에 해당하는 길이(또는 넓이)}}{\text{전체 길이(또는 넓이)}} $$

### 예제: 과녁 맞히기

한 변의 길이가 10인 정사각형 과녁 안에 반지름이 5인 원이 있다. 과녁을 벗어나지 않게 화살을 쏠 때, 원 안에 맞을 확률은?

- **전체 넓이:** $10 \times 10 = 100$
- **원 넓이:** $\pi \times 5^2 = 25\pi$
- **확률:** $\frac{25\pi}{100} = \frac{\pi}{4} \approx 0.785$

## 4. 실전 예제 및 풀이 (Sample Problems)

### 문제 1: 양말 짝 맞추기 (Matching Socks)

서랍 안에 빨간 양말 4짝, 파란 양말 6짝이 들어있다. (총 10짝). 보지 않고 2짝을 꺼낼 때, 같은 색 양말이 나올 확률은?

- **유형:** 비복원 추출(종속) + 케이스 분류(Casework)
- **Case 1 (빨강-빨강):** $\frac{4}{10} \times \frac{3}{9} = \frac{12}{90}$
- **Case 2 (파랑-파랑):** $\frac{6}{10} \times \frac{5}{9} = \frac{30}{90}$
- **합계:** $\frac{12 + 30}{90} = \frac{42}{90} = \frac{7}{15}$

### 문제 2: 동전 던지기 (At least one Head)

동전을 4번 던질 때, 적어도 한 번은 앞면이 나올 확률은?

- **유형:** 독립 사건 + 여사건(Complementary)
- **전체 확률:** 1
- **반대 사건(모두 뒷면):** $\frac{1}{2} \times \frac{1}{2} \times \frac{1}{2} \times \frac{1}{2} = \frac{1}{16}$
- **정답:** $1 - \frac{1}{16} = \frac{15}{16}$

## 5. 코드 구현 (Code Implementation)

Python의 `random` 모듈을 사용하여 기하학적 확률(몬테카를로 시뮬레이션)과 양말 뽑기 문제를 검증합니다.

```python
import random

def solve_geometric_probability(trials=100000):
    """
    정사각형(10x10) 안에 내접하는 원(r=5)에 점이 찍힐 확률 시뮬레이션
    """
    hits = 0
    for _ in range(trials):
        # -5 ~ 5 사이의 좌표 생성 (중심이 0,0)
        x = random.uniform(-5, 5)
        y = random.uniform(-5, 5)

        # 원의 방정식: x^2 + y^2 <= r^2
        if x**2 + y**2 <= 25:
            hits += 1

    prob = hits / trials
    print(f"[기하학적 확률] 시뮬레이션: {prob:.4f} (이론값: pi/4 ≈ 0.7854)")

def solve_sock_problem(trials=100000):
    """
    빨강 4, 파랑 6 중 2개 뽑을 때 같은 색일 확률
    """
    drawer = ['R']*4 + ['B']*6
    match_count = 0

    for _ in range(trials):
        # 비복원 추출
        picked = random.sample(drawer, 2)
        if picked[0] == picked[1]:
            match_count += 1

    prob = match_count / trials
    print(f"[양말 문제] 시뮬레이션: {prob:.4f} (이론값: 7/15 ≈ 0.4667)")

solve_geometric_probability()
solve_sock_problem()
```

# 4.1 확률의 기초 (Probability Fundamentals)

## 1. 개요 (Overview)

확률(Probability)은 어떤 사건이 일어날 가능성을 0과 1 사이의 수치로 나타낸 것입니다. AMC 8에서는 주로 동전 던지기, 주사위 굴리기, 공 뽑기 등의 상황에서 모든 결과가 일어날 가능성이 동등하다고 가정하는 **고전적 확률(Classical Probability)**을 다룹니다.

## 2. 정의 및 공식 (Definition & Formula)

### 2.1 주요 용어

- **표본 공간 (Sample Space, $S$):** 시행에서 일어날 수 있는 모든 가능한 결과들의 집합.
- **사건 (Event, $E$):** 표본 공간의 부분집합으로, 특정한 조건을 만족하는 결과들의 모임.

### 2.2 확률 공식

모든 결과가 일어날 가능성이 같을 때(Equally Likely), 사건 $E$가 일어날 확률 $P(E)$는 다음과 같습니다.

$$ P(E) = \frac{\text{사건 } E\text{가 일어나는 경우의 수}}{\text{전체 경우의 수}} = \frac{n(E)}{n(S)} $$

## 3. 주요 성질 (Key Properties)

### 3.1 확률의 범위

모든 확률은 0과 1 사이의 값을 가집니다.
$$ 0 \le P(E) \le 1 $$

- $P(E) = 0$: 절대로 일어나지 않는 사건 (Impossible Event).
- $P(E) = 1$: 반드시 일어나는 사건 (Certain Event).

### 3.2 여사건의 확률 (Complementary Probability)

사건 $E$가 일어나지 않을 확률은 1에서 사건 $E$가 일어날 확률을 뺀 것과 같습니다.
$$ P(\text{not } E) = 1 - P(E) $$
주로 "적어도 하나(At least one)"가 일어날 확률을 구할 때 유용합니다.

## 4. 예제 (Examples)

### 예제 1: 주사위 던지기

6면 주사위를 한 번 던질 때, 짝수가 나올 확률은?

- 전체 경우($S$): $\{1, 2, 3, 4, 5, 6\}$ (6가지)
- 사건($E$): $\{2, 4, 6\}$ (3가지)
- 확률: $\frac{3}{6} = \frac{1}{2}$

### 예제 2: 동전 두 개 던지기

동전 두 개를 동시에 던질 때, 적어도 하나의 앞면(Head)이 나올 확률은?

- 전체 경우: (H, H), (H, T), (T, H), (T, T) (4가지)
- **방법 1 (직접 세기):** (H, H), (H, T), (T, H) $\rightarrow$ 3가지. 확률 $\frac{3}{4}$.
- **방법 2 (여사건):** 모두 뒷면(T, T)일 확률은 $\frac{1}{4}$.
  $$ 1 - \frac{1}{4} = \frac{3}{4} $$

## 5. 코드 구현 (Code Implementation)

Python의 `fractions` 모듈을 사용하여 확률을 분수 형태로 정확하게 계산하는 방법입니다.

```python
from fractions import Fraction

def calculate_probability(favorable_outcomes, total_outcomes):
    return Fraction(favorable_outcomes, total_outcomes)

# 예제 1: 주사위 짝수 확률
total_dice = 6
even_numbers = len([2, 4, 6])
prob_dice = calculate_probability(even_numbers, total_dice)
print(f"주사위 짝수 확률: {prob_dice}") # 출력: 1/2

# 예제 2: 동전 2개 적어도 하나 앞면
# 전체 경우의 수 생성 (Cartesian Product)
outcomes = ['HH', 'HT', 'TH', 'TT']

# 조건에 맞는 경우 필터링
favorable = [outcome for outcome in outcomes if 'H' in outcome]

prob_coin = calculate_probability(len(favorable), len(outcomes))
print(f"동전 앞면 확률: {prob_coin}") # 출력: 3/4
```

## 6. 요약 (Summary)

- **확률**은 $\frac{\text{원하는 경우}}{\text{전체 경우}}$이다.
- 확률 값은 항상 **0 이상 1 이하**이다.
- **여사건($1-P$)**은 복잡한 조건을 반대로 생각하여 쉽게 풀 때 필수적이다.
- 경우의 수(Counting)를 정확히 세는 것이 확률 계산의 핵심이다.

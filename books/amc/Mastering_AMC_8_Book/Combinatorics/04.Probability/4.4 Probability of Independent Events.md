# 4.4 독립 사건의 확률 (Probability of Independent Events)

## 1. 개요 (Overview)

두 사건 $A$와 $B$가 서로의 발생 가능성에 아무런 영향을 미치지 않을 때, 이 두 사건을 **독립(Independent)**이라고 합니다. 예를 들어, 동전을 던지는 결과가 주사위를 굴리는 결과에 영향을 주지 않는 것과 같습니다.

## 2. 정의 및 공식 (Definition & Formula)

### 2.1 곱셈 법칙 (Multiplication Rule for Independent Events)

두 사건 $A$와 $B$가 독립일 때, 두 사건이 **동시에(AND)** 일어날 확률은 각 사건이 일어날 확률의 곱과 같습니다.

$$ P(A \text{ and } B) = P(A \cap B) = P(A) \times P(B) $$

### 2.2 독립 vs 종속 (Independent vs Dependent)

- **독립(Independent):** 복원 추출(Replacement)처럼 첫 번째 시행의 결과가 두 번째 시행의 확률을 변화시키지 않는 경우.
  - 예: 동전 두 번 던지기.
- **종속(Dependent):** 비복원 추출(Without Replacement)처럼 첫 번째 시행의 결과에 따라 두 번째 시행의 확률이 달라지는 경우.
  - 예: 제비뽑기에서 당첨 제비를 뽑고 다시 넣지 않은 채 다음 사람이 뽑는 경우.

## 3. 예제 (Examples)

### 예제 1: 동전과 주사위 (Coin and Die)

동전 한 개와 주사위 한 개를 동시에 던질 때, 동전은 앞면(Head)이 나오고 주사위는 5 이상의 눈이 나올 확률은?

1.  **사건 A (동전 앞면):** $P(A) = \frac{1}{2}$
2.  **사건 B (주사위 5 이상):** $\{5, 6\}$이므로 $P(B) = \frac{2}{6} = \frac{1}{3}$
3.  **독립 사건의 확률:**
    $$ P(A \cap B) = \frac{1}{2} \times \frac{1}{3} = \frac{1}{6} $$

### 예제 2: 비가 올 확률 (Probability of Rain)

내일 서울에 비가 올 확률이 $0.4$, 부산에 비가 올 확률이 $0.5$라고 하자. 두 도시의 날씨가 서로 독립적이라고 가정할 때, 두 도시 모두 비가 올 확률은?

$$ P(\text{Seoul} \cap \text{Busan}) = 0.4 \times 0.5 = 0.2 $$

## 4. 코드 구현 (Code Implementation)

Python을 사용하여 독립 사건의 확률을 계산하고, 시뮬레이션을 통해 이론값과 실제 발생 빈도를 비교해 봅니다.

```python
import random

def check_independence_simulation(trials=100000):
    # 사건 A: 동전 앞면 (확률 1/2)
    # 사건 B: 주사위 5 이상 (확률 1/3)

    count_A = 0
    count_B = 0
    count_both = 0

    for _ in range(trials):
        # 동전 던지기 ('H': 앞면, 'T': 뒷면)
        coin = random.choice(['H', 'T'])
        # 주사위 던지기 (1~6)
        die = random.randint(1, 6)

        is_A = (coin == 'H')
        is_B = (die >= 5)

        if is_A:
            count_A += 1
        if is_B:
            count_B += 1
        if is_A and is_B:
            count_both += 1

    prob_A = count_A / trials
    prob_B = count_B / trials
    prob_both = count_both / trials

    print(f"P(A) 시뮬레이션: {prob_A:.4f} (이론값: 0.5)")
    print(f"P(B) 시뮬레이션: {prob_B:.4f} (이론값: 0.3333)")
    print(f"P(A and B) 시뮬레이션: {prob_both:.4f}")
    print(f"P(A) * P(B): {prob_A * prob_B:.4f}")

check_independence_simulation()
```

## 5. 요약 (Summary)

- **독립 사건:** 서로 영향을 주지 않음.
- **공식:** $P(A \text{ and } B) = P(A) \times P(B)$.
- **주의:** 배반 사건(Mutually Exclusive, $P(A \cap B)=0$)과 혼동하지 말 것. 독립 사건은 동시에 일어날 수 있지만, 배반 사건은 동시에 일어날 수 없습니다.

# 1.5.3 Proofs about Functions 분석

이 문서는 **1.5.3 Proofs about Functions** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 이산 수학과 컴퓨터 과학에서 매우 중요한 함수의 성질인 **단사(Injective)**, **전사(Surjective)**, **전단사(Bijective)**를 증명하는 방법을 다룹니다.

## 1. 개요 (Overview)

함수에 관한 증명은 그래프를 그려서 확인하는 것이 아니라, **정의(Definition)**를 논리식으로 변환하여 대수적으로 입증해야 합니다.

## 2. 단사 함수 증명 (Proving Injectivity)

### 2.1 정의 (One-to-One)

함수 $f: X \rightarrow Y$가 단사 함수라는 것은 정의역의 서로 다른 원소가 공역의 서로 다른 원소로 대응된다는 뜻입니다.

* **논리적 정의:** $\forall x_1, x_2 \in X, (f(x_1) = f(x_2) \rightarrow x_1 = x_2)$
* **대우:** $x_1 \neq x_2 \rightarrow f(x_1) \neq f(x_2)$

### 2.2 증명 구조 (Standard Approach)

대우보다는 원래의 함축 명제를 직접 증명하는 것이 일반적입니다.

1. **가정:** "정의역의 임의의 두 원소 $x, y$에 대하여, $f(x) = f(y)$라고 가정하자."
2. **전개:** 함수의 식을 대입하여 방정식을 풉니다.
3. **결론:** "$x = y$임을 보였다."
4. **마무리:** 따라서 $f$는 단사 함수이다.

## 3. 전사 함수 증명 (Proving Surjectivity)

### 3.1 정의 (Onto)

함수 $f: X \rightarrow Y$가 전사 함수라는 것은 공역($Y$)과 치역(Range)이 같다는 뜻입니다. 즉, 공역의 모든 원소가 선택받아야 합니다.

* **논리적 정의:** $\forall y \in Y, \exists x \in X \text{ such that } f(x) = y$

### 3.2 증명 구조 (Constructive Proof)

존재성($\exists$)을 증명해야 하므로, 실제로 $x$를 찾아내거나 구성해야 합니다.

1. **가정:** "공역의 임의의 원소 $y \in Y$를 선택하자."
2. **발견:** 방정식 $f(x) = y$를 $x$에 대해 풉니다. (역연산 수행)
3. **검증:** 구한 $x$가 실제로 정의역 $X$에 속하는지 확인합니다. (이 단계가 중요합니다!)
4. **결론:** "$f(x) = y$인 $x$가 존재하므로, $f$는 전사 함수이다."

## 4. 전단사 함수 증명 (Proving Bijectivity)

### 4.1 정의 (One-to-One Correspondence)

함수 $f$가 단사 함수이면서 동시에 전사 함수인 경우입니다.

### 4.2 증명 구조

두 가지를 각각 증명해야 합니다.

1. **Part 1:** $f$가 단사임을 증명합니다.
2. **Part 2:** $f$가 전사임을 증명합니다.
3. **결론:** 두 조건을 모두 만족하므로 $f$는 전단사 함수(일대일 대응)이다.

## 5. 반례 찾기 (Disproving)

* **단사가 아님을 보일 때:** $f(x_1) = f(x_2)$이지만 $x_1 \neq x_2$인 구체적인 두 값 $x_1, x_2$를 찾습니다. (예: $f(x)=x^2$에서 $f(-1)=f(1)$)
* **전사가 아님을 보일 때:** 공역의 원소 중 어떤 $x$로도 만들어낼 수 없는 $y$를 하나 찾습니다. (예: $f(x)=x^2$에서 $y=-1$)

## 6. 요약

**1.5.3 Proofs about Functions**는 함수의 '내부 동작'을 검증하는 과정입니다.

* **단사:** 입력이 같아야 출력이 같다. ($f(x)=f(y) \implies x=y$)
* **전사:** 모든 출력은 원인이 있다. (방정식 $f(x)=y$의 해가 존재)

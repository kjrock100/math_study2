# 1.4.3 Proof by Contrapositive 분석

이 문서는 **1.4.3 Proof by Contrapositive** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 직접 증명하기 까다로운 명제를 **대우(Contrapositive)**를 이용하여 간접적으로 증명하는 방법을 다룹니다.

## 1. 개요 (Overview)

직접 증명법이 가정($P$)에서 출발하여 결론($Q$)으로 가는 정공법이라면, 대우 증명법은 **"결론이 거짓이라면 가정 또한 거짓이어야 한다"**는 논리를 사용하여 우회적으로 원명제의 참을 입증하는 방법입니다.

## 2. 논리적 배경 (Logical Background)

이 증명법의 핵심은 **논리적 동치(Logical Equivalence)**입니다.

* 원명제: $P \rightarrow Q$
* 대우 명제: $\neg Q \rightarrow \neg P$

논리학에서 이 두 명제는 항상 같은 진리값을 가집니다($P \rightarrow Q \equiv \neg Q \rightarrow \neg P$). 따라서 대우 명제가 참임을 보이면, 자동으로 원명제도 참이 됩니다.

## 3. 증명 구조 (Structure)

대우 증명은 다음과 같은 단계로 진행됩니다.

1. **대우 명제 작성:** 증명하려는 명제 "만약 $P$라면 $Q$이다"의 대우인 "만약 $Q$가 아니라면 $P$도 아니다"를 명시합니다.
2. **가정 (Assumption):** 결론의 부정($\neg Q$)을 참이라고 가정합니다.
3. **논리적 전개:** 정의와 공리를 사용하여 추론을 진행합니다.
4. **결론 도출:** 가정의 부정($\neg P$)이 참임을 이끌어냅니다.
5. **마무리:** 대우가 참이므로 원명제도 참이라고 선언합니다.

## 4. 예시 분석 (Example Analysis)

> **명제:** 정수 $n$에 대하여, 만약 $n^2$이 짝수라면, $n$도 짝수이다.

이 명제를 직접 증명하려면 $n^2 = 2k$에서 $n = \sqrt{2k}$를 다뤄야 하므로 정수론적 성질을 보이기 어렵습니다. 따라서 대우를 사용합니다.

### 증명 과정

1. **대우 명제:** 만약 $n$이 홀수라면(짝수가 아니라면), $n^2$도 홀수이다.
2. **가정:** $n$이 홀수라고 가정하자.
3. **정의:** $n = 2k + 1$ (단, $k$는 정수)
4. **전개:**
    $$n^2 = (2k + 1)^2 = 4k^2 + 4k + 1 = 2(2k^2 + 2k) + 1$$
5. **결론:** $2(2k^2 + 2k) + 1$은 홀수의 정의($2m + 1$)를 만족하므로 $n^2$은 홀수이다.
6. **최종:** 대우가 참이므로, 원명제인 "만약 $n^2$이 짝수라면 $n$도 짝수이다"는 참이다. $\blacksquare$

## 5. 언제 사용하는가? (When to use)

다음과 같은 경우에 대우 증명법이 유용합니다.

1. **부정어가 포함된 경우:** 결론($Q$)에 "not", "없다", "아니다" 등이 포함되어 있을 때, 이를 부정하면 긍정문이 되어 다루기 쉬워집니다.
    * *예: "$n$이 홀수이면..." (다루기 쉬움) vs "$n$이 짝수가 아니면..."*
2. **연산 방향:** 가정에서 출발하면 식이 복잡해지지만(예: 제곱근), 결론에서 거꾸로 가면 식이 간단해지는 경우(예: 제곱).

## 6. 요약

**1.4.3 Proof by Contrapositive**는 증명의 방향을 뒤집어 난관을 돌파하는 전략입니다. 직접 증명이 막혔을 때 가장 먼저 고려해야 할 강력한 도구입니다.

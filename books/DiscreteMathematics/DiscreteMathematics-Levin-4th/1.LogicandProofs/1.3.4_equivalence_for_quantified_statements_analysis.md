# 1.3.4 Equivalence for Quantified Statements 분석

이 문서는 **1.3.4 Equivalence for Quantified Statements** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 명제 논리의 동치 개념을 **술어 논리(Predicate Logic)**로 확장하여, 한정자($\forall, \exists$)가 포함된 명제를 조작하고 부정하는 규칙을 다룹니다.

## 1. 섹션 개요 (Overview)

단순 명제에서의 드모르간 법칙이 AND와 OR를 뒤집는 것처럼, 술어 논리에서는 **한정자의 부정**이 핵심 주제입니다. 또한 한정자가 논리 연산자($\land, \lor$)와 만났을 때 분배가 가능한지 여부를 따지는 것은 논리적 오류를 피하기 위해 매우 중요합니다.

## 2. 한정자의 부정 (Negating Quantifiers)

한정자가 포함된 명제를 부정할 때, 부정 기호($\neg$)가 한정자 안으로 들어가면서 한정자의 종류가 바뀝니다. 이를 **한정자에 대한 드모르간 법칙**이라고도 합니다.

### 2.1 전칭 한정자의 부정

* **규칙:** $\neg (\forall x P(x)) \equiv \exists x (\neg P(x))$
* **해석:** "모든 $x$가 $P(x)$인 것은 아니다"는 "$P(x)$를 만족하지 않는 $x$가 **적어도 하나 존재한다**"와 같습니다.
* **예시:**
  * 명제: "모든 새는 날 수 있다."
  * 부정: "날지 못하는 새가 존재한다." (타조, 펭귄 등)

### 2.2 존재 한정자의 부정

* **규칙:** $\neg (\exists x P(x)) \equiv \forall x (\neg P(x))$
* **해석:** "$P(x)$인 $x$가 존재하지 않는다"는 "**모든** $x$에 대하여 $P(x)$가 아니다"와 같습니다.
* **예시:**
  * 명제: "분홍색 코끼리가 존재한다."
  * 부정: "모든 코끼리는 분홍색이 아니다."

## 3. 한정자의 분배 (Distributing Quantifiers)

한정자가 논리 연산자($\land, \lor$) 전체에 걸려 있을 때, 이를 각각의 술어로 분리할 수 있는지에 대한 규칙입니다. **매우 주의해야 할 부분입니다.**

### 3.1 전칭 한정자 ($\forall$)

* **AND에 대해 분배 가능:**
  * $\forall x (P(x) \land Q(x)) \equiv \forall x P(x) \land \forall x Q(x)$
  * *이유: 모든 $x$가 두 조건을 동시에 만족해야 하므로, 각각을 모두 만족하는 것과 같습니다.*
* **OR에 대해 분배 불가능:**
  * $\forall x (P(x) \lor Q(x)) \not\equiv \forall x P(x) \lor \forall x Q(x)$
  * *반례:* "모든 사람은 남자이거나 여자이다"(참) $\neq$ "모든 사람이 남자이거나, 모든 사람이 여자이다"(거짓).

### 3.2 존재 한정자 ($\exists$)

* **OR에 대해 분배 가능:**
  * $\exists x (P(x) \lor Q(x)) \equiv \exists x P(x) \lor \exists x Q(x)$
  * *이유: 둘 중 하나라도 만족하는 $x$가 존재하면 되므로.*
* **AND에 대해 분배 불가능:**
  * $\exists x (P(x) \land Q(x)) \not\equiv \exists x P(x) \land \exists x Q(x)$
  * *반례:* "소수이면서 짝수인 수가 존재한다"(참, 2) $\neq$ "소수가 존재하고, 짝수가 존재한다"(참).
  * *설명:* 오른쪽 식은 서로 다른 $x$일 수 있지만, 왼쪽 식은 **동일한** $x$여야 합니다.

## 4. 변수의 범위와 결합 (Scope and Binding)

* **자유 변수 (Free Variable):** 한정자에 의해 범위가 지정되지 않은 변수.
* **구속 변수 (Bound Variable):** 한정자에 속한 변수.
* **예시:** $\forall x (P(x) \lor Q(y))$
  * $x$는 $\forall$에 묶여 있지만, $y$는 자유롭습니다. 이 식 전체는 $y$에 대한 술어입니다.

## 5. 핵심 분석 및 팁

* **부정의 깊이:** "모든 학생이 숙제를 한 것은 아니다"를 번역할 때 $\forall x \neg P(x)$ ("아무도 안 했다")로 착각하기 쉽습니다. 정답은 $\neg \forall x P(x)$ ("안 한 학생이 있다")입니다.
* **반례 찾기:** $\forall$ 명제가 거짓임을 보일 때는 **단 하나의 반례(Counterexample)**만 찾으면 됩니다. 이는 $\neg \forall \equiv \exists \neg$ 규칙에 근거합니다.

## 요약

**1.3.4 Equivalence for Quantified Statements**는 논리의 정밀함을 더해줍니다. 특히 "모든"과 "어떤"의 부정을 정확히 처리하는 능력은 수학적 증명뿐만 아니라, 일상적인 논쟁이나 프로그래밍 로직에서도 필수적입니다.

# 2.5.3 Coloring Edges 분석

이 문서는 **2.5.3 Coloring Edges** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 정점 채색(Vertex Coloring)과 유사하지만, 이번에는 **간선(Edge)**을 채색하는 문제를 다룹니다.

## 1. 섹션 개요 (Overview)

간선 채색의 기본 규칙은 정점 채색과 유사하지만, 대상이 다릅니다.
> **같은 정점에 연결된(Incident) 두 간선은 서로 다른 색을 가져야 한다.**

이 문제는 "여러 경기가 동시에 열릴 때, 각 팀이 하루에 한 경기만 하도록 일정을 짜려면 최소 며칠이 필요한가?"와 같은 스케줄링 문제에 응용될 수 있습니다.

## 2. 간선 채색 (Edge Coloring)

### 2.1 정의

* **k-간선-채색 (k-edge-coloring):** $k$개의 색을 사용하여 그래프의 모든 간선을 규칙에 맞게 칠하는 것.
* **간선 색상수 (Chromatic Index, $\chi'(G)$):** 그래프 $G$의 간선을 채색하는 데 필요한 **최소한의 색의 수**입니다.

### 2.2 간선 색상수의 하한 (Lower Bound)

* 그래프의 한 정점 $v$에 연결된 모든 간선들은 서로 다른 색을 가져야 합니다.
* 따라서, 필요한 색의 수는 적어도 그래프의 최대 차수($\Delta(G)$)만큼은 되어야 합니다.
    $$ \chi'(G) \ge \Delta(G) $$

## 3. 비징의 정리 (Vizing's Theorem)

간선 채색에서 가장 중요한 정리로, 간선 색상수의 상한을 매우 타이트하게 제시합니다.

> **비징의 정리:** 모든 단순 그래프(Simple Graph) $G$에 대하여, 간선 색상수는 최대 차수 $\Delta(G)$ 또는 $\Delta(G)+1$ 이다.
> $$ \Delta(G) \le \chi'(G) \le \Delta(G) + 1 $$

* **Class 1 그래프:** $\chi'(G) = \Delta(G)$인 그래프.
* **Class 2 그래프:** $\chi'(G) = \Delta(G) + 1$인 그래프.

### 3.1 예시

* **이분 그래프 (Bipartite Graphs):** 모든 이분 그래프는 Class 1입니다. 즉, $\Delta(G)$개의 색만으로 간선 채색이 가능합니다.
* **홀수 사이클 ($C_n$, n은 홀수):** $\Delta(C_n)=2$이지만, 2개의 색으로는 채색할 수 없고 3개의 색이 필요합니다. 따라서 $\chi'(C_n) = 3 = \Delta(C_n) + 1$이므로 Class 2입니다.

## 4. 응용: 라운드 로빈 토너먼트 스케줄링

* **문제:** $n$개의 팀이 서로 한 번씩 경기를 치르는 리그(Round-robin tournament)를 최소 며칠 만에 끝낼 수 있는가? (단, 한 팀은 하루에 한 경기만 할 수 있음)
* **모델링:**
  * **정점:** 각 팀
  * **간선:** 두 팀 간의 경기
  * **그래프:** 완전 그래프 $K_n$
  * **색상:** 경기 날짜 (Day)
* **해결:** 이 문제는 $K_n$의 간선 색상수($\chi'(K_n)$)를 찾는 것과 같습니다.

## 5. 요약

**2.5.3 Coloring Edges**는 정점 채색과 쌍을 이루는 중요한 문제입니다. 비징의 정리는 간선 채색에 필요한 색의 수가 최대 차수와 거의 같다는 놀라운 사실을 알려주며, 이는 정점 채색 문제보다 훨씬 더 예측 가능한 구조를 가짐을 의미합니다.

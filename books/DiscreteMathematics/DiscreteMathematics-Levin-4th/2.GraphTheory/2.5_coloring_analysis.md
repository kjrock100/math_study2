# 2.5 Coloring 분석

이 문서는 **2.5 Coloring** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 그래프의 정점(Vertex)에 색을 칠하는 **그래프 채색(Graph Coloring)** 문제를 다루며, 이는 자원 할당, 스케줄링 등 다양한 최적화 문제의 수학적 모델이 됩니다.

## 1. 섹션 개요 (Overview)

그래프 채색의 기본 규칙은 간단합니다.
> **인접한(Adjacent) 두 정점은 서로 다른 색을 가져야 한다.**

이 간단한 규칙에서 "최소 몇 개의 색이 필요한가?"라는 질문이 파생되며, 이것이 바로 그래프 채색 문제의 핵심입니다.

## 2. 정점 채색 (Vertex Coloring)

### 2.1 정의

* **k-채색 (k-coloring):** $k$개의 색을 사용하여 그래프의 모든 정점을 규칙에 맞게 칠하는 것.
* **색상수 (Chromatic Number, $\chi(G)$):** 그래프 $G$를 채색하는 데 필요한 **최소한의 색의 수**입니다.
  * $\chi(G) = k$라는 것은, $G$가 $k$-채색은 가능하지만 $(k-1)$-채색은 불가능함을 의미합니다.

### 2.2 색상수 계산

* **일반적인 경우:** 색상수를 찾는 것은 매우 어려운 문제입니다(NP-hard).
* **간단한 예시:**
  * **완전 그래프 ($K_n$):** 모든 정점이 서로 인접하므로, 모든 정점은 다른 색을 가져야 합니다. $\chi(K_n) = n$.
  * **사이클 그래프 ($C_n$):**
    * $n$이 짝수이면, 두 가지 색을 번갈아 칠할 수 있습니다. $\chi(C_n) = 2$.
    * $n$이 홀수이면, 마지막 정점에서 색이 충돌하므로 세 번째 색이 필요합니다. $\chi(C_n) = 3$.
  * **이분 그래프 (Bipartite Graph):** 두 그룹으로 나뉘므로, 각 그룹에 다른 색을 칠하면 됩니다. $\chi(G) = 2$.

## 3. 주요 정리 (Key Theorems)

### 3.1 브룩스의 정리 (Brooks' Theorem)
>
> 그래프 $G$의 최대 차수를 $\Delta(G)$라고 할 때, $\chi(G) \le \Delta(G) + 1$ 이다.

* **탐욕 알고리즘 (Greedy Algorithm):** 정점을 하나씩 순서대로 칠하되, 이웃이 사용하지 않은 가장 작은 번호의 색을 칠하는 방식입니다. 이 알고리즘은 최대 $\Delta(G)+1$개의 색을 사용함을 보장합니다.

### 3.2 4색 정리 (The Four Color Theorem)
>
> 모든 **평면 그래프(Planar Graph)**는 4가지 색으로 채색할 수 있다. ($\chi(G) \le 4$)

* **역사:** 1976년 컴퓨터를 이용하여 증명된 최초의 주요 정리로, 수학계에 큰 논쟁을 불러일으켰습니다.
* **의의:** 지도 채색 문제(인접한 국가는 다른 색으로)를 해결합니다. 모든 지도는 평면 그래프로 모델링할 수 있기 때문입니다.

## 4. 응용 (Applications)

그래프 채색은 "충돌" 또는 "간섭"을 피해야 하는 다양한 문제에 적용됩니다.

* **스케줄링:**
  * **정점:** 과목, 회의, 작업
  * **간선:** 동시에 열릴 수 없는 두 과목 (같은 학생이 수강)
  * **색상:** 강의 시간 또는 강의실
  * **목표:** 최소한의 시간 슬롯으로 모든 강의를 배정하는 것.
* **레지스터 할당 (Register Allocation):**
  * 컴파일러 최적화에서, 동시에 사용되는 변수들을 다른 레지스터에 할당하는 문제.

## 5. 요약

**2.5 Coloring**은 단순한 퍼즐처럼 보이지만, 컴퓨터 과학의 핵심 난제 중 하나인 자원 할당 문제를 수학적으로 모델링하는 강력한 도구입니다.

* **색상수($\chi(G)$):** 문제 해결에 필요한 최소 자원의 수.
* **4색 정리:** 2차원 평면이라는 제약 조건이 문제의 복잡도를 크게 낮추는 대표적인 사례.

이 섹션을 통해 우리는 추상적인 그래프 이론이 어떻게 현실 세계의 복잡한 최적화 문제를 해결하는지 이해하게 됩니다.

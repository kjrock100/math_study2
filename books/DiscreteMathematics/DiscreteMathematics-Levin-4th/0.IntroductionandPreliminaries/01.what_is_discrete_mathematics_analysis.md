# 0.1 What is Discrete Mathematics? 분석

이 문서는 "이산 수학(Discrete Mathematics)이란 무엇인가?"에 대한 개요와 핵심 개념을 분석하고 설명합니다. 이산 수학은 컴퓨터 과학의 기초가 되는 수학적 언어이자 도구입니다.

## 이산 수학의 정의

이산 수학은 **연속적이지 않은(discrete)**, 서로 구별되는 값을 다루는 수학의 한 분야입니다.

* **이산(Discrete):** 별개의, 분리된, 끊어진. (예: 정수, 논리값, 그래프의 노드, 디지털 시계의 시간)
* **연속(Continuous):** 끊어지지 않고 이어지는. (예: 실수, 미적분학의 함수, 아날로그 시계의 시간)

미적분학(Calculus)이 변화하는 연속적인 양을 다루는 반면, 이산 수학은 셀 수 있는(countable) 구조와 논리적 관계를 다룹니다.

## 이산 수학 vs 연속 수학

| 특징 | 이산 수학 (Discrete Math) | 연속 수학 (Continuous Math) |
| :--- | :--- | :--- |
| **대상** | 정수(Integers), 유한 집합, 그래프 | 실수(Real Numbers), 연속 함수 |
| **주요 도구** | 논리, 집합론, 조합론, 그래프 이론 | 미분, 적분, 극한 |
| **응용 분야** | 알고리즘, 암호학, 데이터베이스, 네트워크 | 물리학, 공학, 경제학(유체 역학 등) |

## 왜 이산 수학을 배우는가? (컴퓨터 과학에서의 중요성)

컴퓨터는 기본적으로 이산적인 기계입니다. 0과 1(비트)로 정보를 처리하고 저장합니다. 따라서 이산 수학은 컴퓨터 과학의 언어라고 할 수 있습니다.

1. **알고리즘 분석:** 알고리즘의 효율성(시간 복잡도, 공간 복잡도)을 수학적으로 증명하고 분석하는 데 필요합니다.
2. **자료 구조:** 배열, 리스트, 트리, 그래프 등 데이터를 구조화하는 방법론의 기초가 됩니다.
3. **논리 회로 및 컴퓨터 구조:** 불 대수(Boolean Algebra)는 컴퓨터 하드웨어 설계의 기본입니다.
4. **암호학:** 정수론(Number Theory)은 RSA와 같은 암호화 알고리즘의 핵심 원리입니다.
5. **문제 해결 능력:** 복잡한 문제를 수학적으로 모델링하고 논리적으로 해결하는 사고력을 기를 수 있습니다.

## 주요 연구 주제

이산 수학은 다음과 같은 다양한 하위 분야를 포함합니다.

* **논리와 증명 (Logic and Proofs):** 수학적 추론의 기초. 명제, 술어 논리, 증명 방법(귀납법 등).
* **집합, 함수, 관계 (Sets, Functions, Relations):** 객체들의 모임과 그들 사이의 관계를 정의.
* **알고리즘과 정수론 (Algorithms and Integers):** 계산 절차와 수의 성질(소수, 합동식 등).
* **세기와 확률 (Counting and Probability):** 순열, 조합, 비둘기집 원리 등 경우의 수 계산.
* **그래프 이론 (Graph Theory):** 점과 선으로 연결된 구조(네트워크) 연구. 최단 경로, 스패닝 트리 등.
* **트리 (Trees):** 계층적 구조를 모델링. 탐색, 정렬 등에 활용.

## 요약

"이산 수학이란 무엇인가?"라는 질문에 대한 답은 **"컴퓨터가 생각하고 문제를 해결하는 방식을 수학적으로 정립한 것"**이라고 할 수 있습니다. 연속적인 실세계의 문제를 컴퓨터가 처리할 수 있는 이산적인 형태로 모델링하고 해결하는 능력을 기르는 것이 이산 수학 학습의 목표입니다.

# 5.1 Sets 분석

이 문서는 **5.1 Sets** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 이산 수학의 가장 기초적인 구조인 **집합(Set)**을 엄밀한 수학적 언어로 정의하고, 이를 다루는 다양한 도구들을 소개합니다.

## 1. 섹션 개요 (Overview)

0장에서는 집합을 직관적으로 "객체들의 모임"으로 이해했습니다. 5.1장에서는 이를 형식화하여, 집합을 정의하는 방법(표기법), 집합 사이의 관계(부분집합, 상등), 그리고 새로운 집합을 만드는 연산(합집합, 교집합 등)을 체계적으로 다룹니다. 이는 이후의 증명이나 함수, 관계 등을 이해하는 필수적인 기초가 됩니다.

## 2. 표기법 (Notation) - 5.1.1

집합을 표현하는 방법과 중요한 특수 집합들을 정의합니다.

* **원소 나열법 (Roster Method):** 원소들을 중괄호 안에 나열합니다. 예: $A = \{1, 2, 3\}$. 순서는 상관없으며, 중복된 원소는 하나로 취급합니다.
* **조건 제시법 (Set-Builder Notation):** 특정 조건을 만족하는 원소들의 모임으로 정의합니다.
  * 형식: $\{x \in U : P(x)\}$ (여기서 $U$는 전체 집합, $P(x)$는 조건)
  * 예: $\{x \in \mathbb{Z} : x > 0\}$ (양의 정수 집합)
* **주요 수 집합:**
  * $\mathbb{N}$: 자연수 집합 $\{0, 1, 2, \dots\}$
  * $\mathbb{Z}$: 정수 집합
  * $\mathbb{Q}$: 유리수 집합
  * $\mathbb{R}$: 실수 집합
* **특수 집합:**
  * **공집합 (Empty Set):** 원소가 하나도 없는 집합. 기호: $\emptyset = \{\}$.
  * **멱집합 (Power Set):** 집합 $A$의 모든 부분집합들을 원소로 가지는 집합. 기호: $\mathcal{P}(A)$.
    * 특징: $|A| = n$이면 $|\mathcal{P}(A)| = 2^n$입니다.
* **크기 (Cardinality):** 집합 $A$의 원소의 개수. 기호: $|A|$.

## 3. 집합 간의 관계 (Relationships between Sets) - 5.1.2

두 집합이 서로 어떻게 연관되는지 정의합니다.

* **부분집합 (Subset):** $A$의 모든 원소가 $B$에 포함될 때, $A$는 $B$의 부분집합입니다.
  * 표기: $A \subseteq B \iff \forall x (x \in A \to x \in B)$.
* **진부분집합 (Proper Subset):** $A \subseteq B$이면서 $A \neq B$인 경우.
  * 표기: $A \subset B$.
* **상등 (Equality):** 두 집합이 정확히 같은 원소들을 가질 때.
  * 정의: $A = B \iff A \subseteq B \land B \subseteq A$. (증명에서 자주 사용되는 전략)

## 4. 집합의 연산 (Operations on Sets) - 5.1.3

기존 집합들을 결합하여 새로운 집합을 만듭니다.

* **합집합 (Union):** $A \cup B = \{x : x \in A \lor x \in B\}$.
* **교집합 (Intersection):** $A \cap B = \{x : x \in A \land x \in B\}$.
* **차집합 (Set Difference):** $A \setminus B = \{x : x \in A \land x \notin B\}$. ($A - B$로 쓰기도 함)
* **여집합 (Complement):** 전체 집합 $U$에 대하여, $\bar{A} = U \setminus A = \{x \in U : x \notin A\}$.
* **곱집합 (Cartesian Product):** $A \times B = \{(a, b) : a \in A, b \in B\}$.
  * 크기: $|A \times B| = |A| \cdot |B|$.

## 5. 벤 다이어그램 (Venn Diagrams) - 5.1.4

집합의 관계를 시각적으로 나타내는 도구입니다.

* **용도:** 집합 연산의 결과를 직관적으로 이해하거나, 복잡한 식을 단순화할 때 유용합니다.
* **한계:** 벤 다이어그램은 엄밀한 **증명(Proof)**이 될 수 없습니다. 증명은 논리적 정의나 원소 포함 관계를 통해 서술해야 합니다.

## 6. 요약

**5.1 Sets**는 수학적 논의를 위한 "어휘"를 정립하는 단계입니다.

* **핵심:** 집합은 조건(논리)으로 정의되며, 집합 연산은 논리 연산(AND, OR, NOT)과 직접적으로 대응됩니다.
* **주의:** $\in$ (원소이다)와 $\subseteq$ (부분집합이다)를 혼동하지 않도록 주의해야 합니다. 예: $\{1\} \in \{1, \{1\}\}$은 거짓이지만, $\{1\} \subseteq \{1, 2\}$는 참입니다.

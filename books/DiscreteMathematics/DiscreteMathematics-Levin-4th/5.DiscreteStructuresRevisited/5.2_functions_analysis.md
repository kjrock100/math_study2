# 5.2 Functions 분석

이 문서는 **5.2 Functions** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 이산 수학뿐만 아니라 모든 수학의 핵심 도구인 **함수(Function)**를 집합론적 관점에서 엄밀하게 정의하고 분류합니다.

## 1. 섹션 개요 (Overview)

함수는 두 집합 사이의 관계를 기술하는 규칙입니다. 미적분학에서는 주로 실수에서 실수로 가는 연속적인 함수를 다루지만, 이산 수학에서는 유한 집합 사이의 함수나 정수에서 정의된 함수 등을 다루며, 함수의 **구조적 성질(매핑 방식)**에 더 집중합니다.

## 2. 함수의 기술 (Describing Functions) - 5.2.1

### 2.1 정의와 구성 요소

함수 $f: X \to Y$는 집합 $X$의 **모든** 원소를 집합 $Y$의 **유일한** 원소에 대응시키는 규칙입니다.

* **정의역 (Domain):** 입력값들의 집합 $X$.
* **공역 (Codomain):** 가능한 출력값들의 집합 $Y$.
* **치역 (Range/Image):** 실제 함숫값들의 집합. $\{f(x) : x \in X\}$. 치역은 공역의 부분집합입니다.

### 2.2 표현 방법

* **공식:** $f(n) = n^2 + 1$
* **재귀:** $f(0)=1, f(n) = 2f(n-1)$
* **표/화살표 그림:** 유한 집합일 때 원소 간의 대응 관계를 직접 나열하거나 그림으로 표현.

## 3. 함수의 분류 (Surjections, Injections, and Bijections) - 5.2.2

함수가 원소들을 어떻게 연결하는지에 따라 세 가지로 분류합니다. 이는 집합의 크기(Cardinality) 비교와 밀접한 관련이 있습니다.

### 3.1 단사 함수 (Injections / One-to-One)

* **정의:** 서로 다른 입력은 항상 서로 다른 출력으로 갑니다.
* **논리:** $\forall x_1, x_2 \in X, f(x_1) = f(x_2) \implies x_1 = x_2$.
* **의미:** 충돌(Collision)이 없습니다. $|X| \le |Y|$임을 암시합니다.

### 3.2 전사 함수 (Surjections / Onto)

* **정의:** 공역의 모든 원소가 적어도 한 번은 선택받습니다.
* **논리:** $\forall y \in Y, \exists x \in X \text{ such that } f(x) = y$.
* **의미:** 공역과 치역이 같습니다. $|X| \ge |Y|$임을 암시합니다.

### 3.3 전단사 함수 (Bijections)

* **정의:** 단사이면서 동시에 전사인 함수입니다.
* **의미:** 두 집합의 원소들이 완벽하게 일대일로 짝지어집니다.
* **결과:** $|X| = |Y|$입니다. 역함수(Inverse Function) $f^{-1}: Y \to X$가 존재합니다.

## 4. 상과 역상 (Image and Inverse Image) - 5.2.3

함수를 개별 원소가 아닌 **집합**에 적용하는 개념입니다.

### 4.1 상 (Image)

* 정의역의 부분집합 $A \subseteq X$에 대해, $A$의 원소들이 도달하는 함숫값들의 집합입니다.
* $f(A) = \{f(x) : x \in A\} \subseteq Y$.

### 4.2 역상 (Inverse Image / Preimage)

* 공역의 부분집합 $B \subseteq Y$에 대해, $B$의 원소로 매핑되는 정의역의 원소들의 집합입니다.
* $f^{-1}(B) = \{x \in X : f(x) \in B\} \subseteq X$.
* **주의:** 여기서 $f^{-1}$는 역함수가 존재하지 않아도 정의되는 **집합 연산**입니다.

## 5. 요약

**5.2 Functions**는 집합 간의 구조적 관계를 파악하는 도구입니다.

* **단사/전사:** 비둘기집 원리나 집합의 크기 비교에 사용됩니다.
* **전단사:** 두 집합이 구조적으로 동일함(Isomorphic)을 보이거나, 조합론에서 개수를 셀 때(Combinatorial Proof) 핵심적인 역할을 합니다.
* **역상:** 위상수학이나 해석학 등 상위 수학에서 연속성을 정의할 때 필수적인 개념입니다.

# 5.2.3 Image and Inverse Image 분석

이 문서는 **5.2.3 Image and Inverse Image** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 함수를 개별 원소가 아닌 **집합(Subset)**에 적용할 때의 개념인 **상(Image)**과 **역상(Inverse Image)**을 다룹니다.

## 1. 섹션 개요 (Overview)

지금까지 우리는 함수 $f$에 하나의 원소 $x$를 넣어 $f(x)$를 얻었습니다. 이제는 정의역의 부분집합 전체를 함수에 넣거나, 공역의 부분집합에 대응되는 원래의 입력값들을 찾는 작업을 수행합니다. 이는 위상수학, 해석학, 확률론 등 상위 수학에서 매우 빈번하게 사용되는 언어입니다.

## 2. 상 (Image)

### 2.1 정의

함수 $f: X \to Y$와 정의역의 부분집합 $A \subseteq X$가 있을 때, $A$에 속한 모든 원소들의 함숫값을 모은 집합을 $A$의 **상(Image)**이라고 합니다.

* **기호:** $f(A)$
* **정의:** $f(A) = \{f(x) : x \in A\}$
* **논리적 정의:** $y \in f(A) \iff \exists x \in A \text{ such that } f(x) = y$.
* **결과:** $f(A)$는 공역 $Y$의 부분집합입니다. ($f(A) \subseteq Y$)

### 2.2 치역과의 관계

* 함수 전체의 **치역(Range)**은 정의역 전체 $X$에 대한 상, 즉 $f(X)$와 같습니다.

## 3. 역상 (Inverse Image / Preimage)

### 3.1 정의

함수 $f: X \to Y$와 공역의 부분집합 $B \subseteq Y$가 있을 때, 함숫값이 $B$에 포함되는 모든 정의역의 원소들을 모은 집합을 $B$의 **역상(Inverse Image)** 또는 **원상(Preimage)**이라고 합니다.

* **기호:** $f^{-1}(B)$
* **정의:** $f^{-1}(B) = \{x \in X : f(x) \in B\}$
* **결과:** $f^{-1}(B)$는 정의역 $X$의 부분집합입니다. ($f^{-1}(B) \subseteq X$)

### 3.2 주의사항: 역함수와의 차이

이 섹션에서 가장 중요한 개념적 구분입니다.

* 기호 $f^{-1}$를 사용하지만, 이는 **역함수(Inverse Function)와는 다릅니다.**
* 역함수는 전단사(Bijection)일 때만 존재하지만, **역상은 어떤 함수에서든 정의됩니다.**
* 역함수는 원소에서 원소로 가는 함수($Y \to X$)이지만, 역상은 집합에서 집합으로 가는 연산($\mathcal{P}(Y) \to \mathcal{P}(X)$)으로 볼 수 있습니다.
* 예를 들어, $f$가 단사함수가 아니어서 $f(1)=a, f(2)=a$일 때, 역함수 $f^{-1}(a)$는 정의되지 않지만, 집합 $\{a\}$에 대한 역상 $f^{-1}(\{a\})$는 $\{1, 2\}$로 잘 정의됩니다.

## 4. 예시 (Example)

함수 $f: \{1, 2, 3, 4, 5\} \to \{a, b, c, d\}$가 다음과 같이 정의된다고 합시다.

* $f(1)=a, f(2)=a, f(3)=b, f(4)=c, f(5)=c$

1. **상 구하기:**
    * $A = \{1, 2, 3\}$ 일 때, $f(A) = \{f(1), f(2), f(3)\} = \{a, a, b\} = \{a, b\}$.
2. **역상 구하기:**
    * $B = \{a, c\}$ 일 때, $f(x)$가 $a$ 또는 $c$가 되는 $x$를 모두 찾습니다.
    * $f^{-1}(B) = \{1, 2, 4, 5\}$.
    * $C = \{d\}$ 일 때, $d$로 가는 화살표가 없으므로 $f^{-1}(C) = \emptyset$.

## 5. 요약

**5.2.3 Image and Inverse Image**는 집합론적 관점에서 함수를 다루는 방법을 확장합니다.

* **방향:**
  * 상($f$): 정의역의 부분집합 $\to$ 공역의 부분집합.
  * 역상($f^{-1}$): 공역의 부분집합 $\to$ 정의역의 부분집합.
* **활용:** 이 개념은 "방정식 $f(x)=0$의 해집합"을 $f^{-1}(\{0\})$으로 표현하는 등, 조건에 맞는 원소들의 모임을 간결하게 나타내는 데 필수적입니다.

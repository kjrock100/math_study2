# 5.1.1 Notation 분석

이 문서는 **5.1.1 Notation** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 집합을 정의하고 표현하는 표준적인 방법들과 이산 수학에서 자주 사용되는 특수 집합들의 표기법을 다룹니다.

## 1. 섹션 개요 (Overview)

수학에서 명확한 의사소통을 위해서는 약속된 기호와 표기법을 사용하는 것이 필수적입니다. 이 섹션에서는 집합을 기술하는 두 가지 주요 방법(원소 나열법, 조건 제시법)과 수 체계, 그리고 공집합이나 멱집합 같은 특별한 집합들을 나타내는 기호들을 정의합니다.

## 2. 집합을 정의하는 방법 (Defining Sets)

### 2.1 원소 나열법 (Roster Method)

집합의 모든 원소를 중괄호 `{}` 안에 나열하는 방법입니다.

* **예시:** $A = \{a, e, i, o, u\}$
* **특징:**
  * **순서 무시:** $\{1, 2, 3\} = \{3, 1, 2\}$. 집합은 순서가 없는 모임입니다.
  * **중복 무시:** $\{1, 1, 2\} = \{1, 2\}$. 같은 원소를 여러 번 써도 하나로 취급합니다.
* **생략 기호:** 원소가 많을 때는 패턴이 명확하다면 줄임표($\dots$)를 사용할 수 있습니다. 예: $\{1, 2, \dots, 100\}$.

### 2.2 조건 제시법 (Set-Builder Notation)

원소들이 만족해야 하는 조건을 명시하여 집합을 정의하는 방법입니다.

* **형식:** $\{ \text{변수} : \text{조건} \}$ 또는 $\{ \text{변수} \mid \text{조건} \}$
  * "조건을 만족하는 모든 변수들의 집합"이라고 읽습니다.
* **예시:**
  * $\{x \in \mathbb{Z} : x > 0\}$ (양의 정수 집합)
  * $\{2n : n \in \mathbb{Z}\}$ (짝수 집합)
* **장점:** 무한 집합이나 원소를 나열하기 어려운 복잡한 집합을 명확하게 표현할 수 있습니다.

## 3. 주요 수 집합 (Standard Number Sets)

이산 수학에서 자주 등장하는 수의 집합들은 칠판 굵은 글씨체(Blackboard Bold)로 표기합니다.

* $\mathbb{N}$: **자연수 (Natural Numbers)**. $\{0, 1, 2, \dots\}$. (저자에 따라 0을 포함하지 않기도 하므로 정의를 확인해야 합니다. 이 책에서는 0을 포함합니다.)
* $\mathbb{Z}$: **정수 (Integers)**. $\{\dots, -2, -1, 0, 1, 2, \dots\}$.
* $\mathbb{Q}$: **유리수 (Rationals)**. $\{ \frac{a}{b} : a, b \in \mathbb{Z}, b \neq 0 \}$.
* $\mathbb{R}$: **실수 (Reals)**. 수직선 상의 모든 수.
* 참고: $\mathbb{Z}^+$는 양의 정수 $\{1, 2, 3, \dots\}$를 의미합니다.

## 4. 특수 집합 (Special Sets)

### 4.1 공집합 (The Empty Set)

* 원소가 하나도 없는 집합입니다.
* **기호:** $\emptyset$ 또는 $\{\}$.
* **주의:** $\{\emptyset\}$은 공집합을 원소로 가지는 집합이므로 공집합이 아닙니다. (빈 상자 vs 빈 상자가 들어있는 상자)
* 모든 집합의 부분집합입니다.

### 4.2 멱집합 (The Power Set)

* 어떤 집합 $A$의 **모든 부분집합**들을 원소로 가지는 집합입니다.
* **기호:** $\mathcal{P}(A)$.
* **예시:** $A = \{1, 2\}$ 이면, $\mathcal{P}(A) = \{ \emptyset, \{1\}, \{2\}, \{1, 2\} \}$.
* **크기:** 만약 $|A| = n$이면, $|\mathcal{P}(A)| = 2^n$입니다.

## 5. 집합의 크기 (Cardinality)

* 집합 $A$에 포함된 서로 다른 원소의 개수를 의미합니다.
* **기호:** $|A|$.
* **유한 집합:** 원소의 개수가 유한한 경우 정수 값을 가집니다. 예: $|\{a, b, c\}| = 3$.
* **무한 집합:** 자연수 집합이나 실수 집합처럼 원소가 무한히 많은 경우입니다. (이 책의 뒷부분에서 무한 집합의 크기 비교를 다룹니다.)

## 6. 요약

**5.1.1 Notation**은 수학적 언어의 기초 문법을 배우는 단계입니다.

* 집합을 정의할 때는 상황에 따라 원소 나열법과 조건 제시법 중 더 명확한 것을 선택합니다.
* $\emptyset, \in, \subseteq, \mathcal{P}(A)$ 등의 기호 의미를 정확히 파악해야 이후의 논리 전개에서 혼동을 피할 수 있습니다.

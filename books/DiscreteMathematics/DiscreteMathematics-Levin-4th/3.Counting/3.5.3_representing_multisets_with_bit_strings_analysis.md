# 3.5.3 Representing Multisets with Bit Strings 분석

이 문서는 **3.5.3 Representing Multisets with Bit Strings** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 시각적인 도구였던 "Stars and Bars"를 수학적이고 디지털적인 **비트 문자열(Bit String)**로 변환하여, 중복 조합 문제를 우리가 이미 알고 있는 조합 문제로 공식화합니다.

## 1. 섹션 개요 (Overview)

3.5.2절에서 우리는 쿠키(별)와 칸막이(막대)를 나열하여 중복 조합 문제를 해결했습니다. 이 섹션에서는 이 모델을 0과 1로 이루어진 비트 문자열로 1:1 대응(Bijection)시킴으로써, 이 문제가 3.1.3절에서 배운 **비트 문자열 세기** 문제와 완전히 동일함을 보여줍니다.

## 2. 변환 과정 (The Transformation)

Stars and Bars 모델을 비트 문자열로 바꾸는 것은 매우 직관적인 치환 과정입니다.

* **별 ($*$)** $\rightarrow$ **1** (선택된 아이템)
* **막대 ($|$)** $\rightarrow$ **0** (구분자)

### 2.1 예시

3명의 아이에게 7개의 쿠키를 나눠주는 상황 ($n=3, k=7$)을 다시 봅시다.

* **Stars and Bars:** `***|**|**`
* **Bit String:** `111011011`

이 변환을 통해, "별과 막대를 나열하는 문제"는 "1과 0을 나열하는 문제"가 됩니다.

## 3. 수학적 공식화 (Mathematical Formulation)

이제 문제는 다음과 같이 재정의됩니다.
> "1이 $k$개이고 0이 $n-1$개인 비트 문자열의 개수는?"

### 3.1 길이와 선택

* **문자열의 총 길이:** (1의 개수) + (0의 개수) = $k + (n-1)$
* **선택:** 총 $n+k-1$개의 자리(비트) 중에서, 1이 들어갈 $k$개의 자리를 선택하면 됩니다. (나머지는 자동으로 0이 됩니다.)

### 3.2 공식 유도

이항 계수의 정의에 따라:
$$ \binom{\text{총 길이}}{\text{1의 개수}} = \binom{n+k-1}{k} $$

또한, 0의 위치를 선택하는 관점에서 보면:
$$ \binom{\text{총 길이}}{\text{0의 개수}} = \binom{n+k-1}{n-1} $$

두 값은 대칭성에 의해 동일합니다.

## 4. 요약

**3.5.3 Representing Multisets with Bit Strings**는 중복 조합 공식의 수학적 근거를 완성합니다.

1. **Multiset Counting** (중복 조합)
2. $\leftrightarrow$ **Stars and Bars** (시각적 모델)
3. $\leftrightarrow$ **Bit Strings** (대수적 모델)
4. $\leftrightarrow$ **Combinations** (이항 계수 $\binom{n+k-1}{k}$)

이 연결 고리를 통해, 우리는 복잡한 중복 허용 선택 문제를 단순한 조합 공식 하나로 해결할 수 있게 됩니다.

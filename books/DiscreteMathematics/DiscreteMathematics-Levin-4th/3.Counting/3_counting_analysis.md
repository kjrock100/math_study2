# 3 Counting 분석

이 문서는 **3 Counting** 챕터의 내용을 분석하고 요약합니다. 이 장은 **조합론(Combinatorics)**의 핵심 주제인 "어떤 조건을 만족하는 경우의 수를 세는 방법"을 다룹니다. 이는 알고리즘의 복잡도 분석, 확률 계산, 데이터 보안 등 컴퓨터 과학 전반에 걸쳐 필수적인 기초 지식입니다.

## 1. 챕터 개요 (Chapter Overview)

단순히 하나씩 세는 것(Enumeration)은 대상이 많아지면 불가능합니다. 이 장에서는 **합의 법칙**과 **곱의 법칙**이라는 두 가지 기본 원리에서 시작하여, 순열, 조합, 중복 조합, 그리고 포함-배제의 원리와 같은 강력한 세기 도구들을 체계적으로 학습합니다. 또한, 수식을 대수적으로 변형하는 대신 "세는 방법"을 통해 증명하는 **조합적 증명**이라는 우아한 기법을 소개합니다.

## 2. 주요 섹션 분석

### 3.1 Pascal’s Arithmetical Triangle

* **핵심 내용:** 조합론의 가장 유명한 패턴인 파스칼의 삼각형을 통해 이항 계수(Binomial Coefficients) $\binom{n}{k}$를 소개합니다.
* **주요 개념:**
  * **격자 경로 (Lattice Paths):** (0,0)에서 (n,k)로 가는 최단 경로의 수.
  * **비트 문자열 (Bit Strings):** 길이가 $n$이고 1이 $k$개인 비트열의 수.
  * **부분집합 (Subsets):** 원소가 $n$개인 집합에서 크기가 $k$인 부분집합의 수.
  * 이 모든 것이 $\binom{n}{k}$로 귀결됨을 보여줍니다.

### 3.2 Combining Outcomes

* **핵심 내용:** 복잡한 문제를 작은 문제로 나누어 푸는 기본 원리를 다룹니다.
* **주요 원리:**
  * **합의 법칙 (Sum Principle):** 사건들이 서로 동시에 일어날 수 없을 때(배반 사건), 경우의 수를 더합니다. (OR)
  * **곱의 법칙 (Product Principle):** 사건들이 연속적으로 또는 독립적으로 일어날 때, 경우의 수를 곱합니다. (AND)

### 3.3 Non-Disjoint Outcomes

* **핵심 내용:** 사건들이 서로 겹칠 때(Non-disjoint) 정확하게 세는 방법을 다룹니다.
* **주요 개념:**
  * **포함-배제의 원리 (Principle of Inclusion-Exclusion, PIE):** 겹치는 부분을 더했다가 뺐다가 하는 과정을 일반화합니다. $|A \cup B| = |A| + |B| - |A \cap B|$.
  * 벤 다이어그램을 이용한 시각적 이해가 중요합니다.

### 3.4 Combinations and Permutations

* **핵심 내용:** 순서가 중요한 경우와 중요하지 않은 경우를 구별합니다.
* **주요 공식:**
  * **순열 (Permutations, $P(n, k)$):** 순서 O, 중복 X. $n! / (n-k)!$.
  * **조합 (Combinations, $C(n, k)$ 또는 $\binom{n}{k}$):** 순서 X, 중복 X. $n! / (k!(n-k)!)$.
  * **몫의 원리 (Quotient Principle):** 순서를 무시하기 위해 중복되는 횟수($k!$)로 나누는 개념을 설명합니다.

### 3.5 Counting Multisets

* **핵심 내용:** 중복을 허용하여 선택하는 경우(중복 조합)를 다룹니다.
* **주요 기법:**
  * **Stars and Bars (별과 칸막이):** $n$개의 종류에서 중복을 허용하여 $k$개를 선택하는 문제를, 칸막이($n-1$개)와 별($k$개)을 나열하는 문제로 변환하여 $\binom{n+k-1}{k}$ (또는 $\binom{n+k-1}{n-1}$)로 해결합니다.

### 3.6 Combinatorial Proofs

* **핵심 내용:** 복잡한 수식을 계산 없이 "스토리"를 통해 증명합니다.
* **방법:** 등호($=$)의 양쪽이 "동일한 집합을 두 가지 다른 방법으로 센 것"임을 보여줍니다.
  * *예: $\binom{n}{k} = \binom{n}{n-k}$ (k명을 뽑는 것은 남을 n-k명을 뽑는 것과 같다)*

### 3.7 Applications to Probability

* **핵심 내용:** 카운팅 기술을 이산 확률(Discrete Probability) 계산에 적용합니다.
* **주요 개념:**
  * $P(E) = \frac{|E|}{|S|}$ (사건의 경우의 수 / 전체 표본 공간의 크기).
  * 조건부 확률과 독립 사건.

### 3.8 Advanced Counting Using PIE

* **핵심 내용:** 포함-배제의 원리를 심화 응용하여 복잡한 제약 조건이 있는 문제를 해결합니다.
* **주요 문제:**
  * **교란 순열 (Derangements):** 자기 자신의 위치에 있지 않은 순열의 수 (모자 문제).
  * **전사 함수 (Surjective Functions):** 공역의 모든 원소가 선택받는 함수의 개수.

## 3. 학습 포인트

1. **문제의 구조 파악:** 주어진 문제가 "순서가 있는가?", "중복을 허용하는가?", "구별 가능한가?"를 먼저 파악해야 올바른 공식을 선택할 수 있습니다.
2. **모델링:** "쿠키를 나눠주는 문제"를 "방정식의 해의 개수"나 "Stars and Bars"로 변환하는 모델링 능력이 중요합니다.
3. **직관적 증명:** 수식을 전개하는 것보다, 그 수식이 의미하는 바를 언어로 설명하는 조합적 증명 연습이 논리적 사고력을 크게 향상시킵니다.

## 요약

**3 Counting**은 "어떻게 셀 것인가?"에 대한 답을 줍니다.

* 기본은 **합**과 **곱**입니다.
* 순서가 없으면 **나눕니다** (조합).
* 겹치면 **뺍니다** (포함-배제).
* 복잡하면 **모델을 바꿉니다** (Stars and Bars).

이 장에서 배운 도구들은 4장 수열(Sequences)과 알고리즘 분석의 기초가 됩니다.

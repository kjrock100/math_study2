# 6.1.1 Building Generating Functions 분석

이 문서는 **6.1.1 Building Generating Functions** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 수열을 생성 함수(Generating Function)로 변환하는 기본적인 정의와, 자주 사용되는 표준적인 수열들의 생성 함수를 유도하는 방법을 다룹니다.

## 1. 섹션 개요 (Overview)

생성 함수를 만드는 것은 수열의 항들을 $x$의 거듭제곱 계수로 나열하는 것입니다. 이 섹션에서는 가장 단순한 수열부터 시작하여, 등비수열이나 유한 수열과 같은 패턴을 가진 수열들을 닫힌 형태(Closed Form)의 함수로 표현하는 법을 배웁니다.

## 2. 정의 (Definition)

수열 $A = (a_0, a_1, a_2, \dots)$의 생성 함수 $A(x)$는 다음과 같은 무한 급수(Infinite Series)로 정의됩니다.

$$ A(x) = \sum_{n=0}^{\infty} a_n x^n = a_0 + a_1 x + a_2 x^2 + \dots $$

여기서 $x$는 수렴성을 고려하기 위한 변수가 아니라, 각 항 $a_n$의 위치(인덱스)를 표시하는 **형식적인 변수(Formal Variable)**입니다.

## 3. 주요 예시 (Key Examples)

가장 기본이 되는 생성 함수들은 기하급수(Geometric Series) 공식을 기반으로 합니다.

### 3.1 상수 수열 (The Constant Sequence)

수열: $1, 1, 1, 1, \dots$ ($a_n = 1$)

* 생성 함수: $1 + x + x^2 + x^3 + \dots$
* 닫힌 공식: $\frac{1}{1-x}$
* 이것은 생성 함수 이론의 가장 기초가 되는 식입니다.

### 3.2 등비 수열 (Geometric Sequences)

수열: $1, 2, 4, 8, \dots$ ($a_n = 2^n$)

* 생성 함수: $1 + 2x + 4x^2 + 8x^3 + \dots = 1 + (2x) + (2x)^2 + \dots$
* 닫힌 공식: 위 식의 $x$ 자리에 $2x$를 대입하면 됩니다.
    $$ \frac{1}{1-2x} $$
* 일반화: 수열 $1, c, c^2, \dots$의 생성 함수는 $\frac{1}{1-cx}$입니다.

### 3.3 유한 수열 (Finite Sequences)

수열: $1, 1, 1, 1, 1$ (항이 5개이고 나머지는 0)

* 생성 함수: $1 + x + x^2 + x^3 + x^4$
* 유도: 무한 급수의 차를 이용합니다.
    $$ (1 + x + \dots) - (x^5 + x^6 + \dots) = \frac{1}{1-x} - x^5 \frac{1}{1-x} = \frac{1-x^5}{1-x} $$

## 4. 수열의 조작 (Operations on Sequences)

생성 함수는 선형성(Linearity)을 가집니다.

* 두 수열을 더하면, 생성 함수도 더해집니다.
* 수열에 상수를 곱하면, 생성 함수에도 상수가 곱해집니다.
* 예: 수열 $a_n = 1 + 2^n$의 생성 함수는 $\frac{1}{1-x} + \frac{1}{1-2x}$입니다.

## 5. 요약

**6.1.1 Building Generating Functions**는 수열을 다항식의 세계로 가져오는 첫 단계입니다.

* **핵심 도구:** $\frac{1}{1-x} = \sum x^n$.
* **전략:** 주어진 수열을 기본 기하급수 형태의 변형(치환, 합, 차)으로 표현하여 닫힌 공식을 찾습니다.

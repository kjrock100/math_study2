# 6.1 Generating Functions 분석

이 문서는 **6.1 Generating Functions** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 수열을 다루는 매우 강력한 도구인 **생성 함수(Generating Functions)**를 소개합니다. 이는 수열의 항들을 다항식의 계수로 취급하여, 수열 문제를 대수적 문제로 변환해 해결하는 기법입니다.

## 1. 섹션 개요 (Overview)

생성 함수는 이산 수학의 수열과 연속 수학의 급수(Series)를 연결하는 다리입니다. 수열 $a_0, a_1, a_2, \dots$에 대한 정보를 함수 $A(x) = \sum_{n=0}^{\infty} a_n x^n$ 안에 압축하여 저장합니다. 여기서 $x$는 값을 대입하기 위한 변수가 아니라, 각 항의 위치를 표시하는 '옷걸이(clothesline)' 역할을 합니다.

## 2. 생성 함수 만들기 (Building Generating Functions) - 6.1.1

수열을 생성 함수로 변환하는 기본 정의와 예시를 다룹니다.

* **정의:** 수열 $a_n$의 생성 함수는 $A(x) = a_0 + a_1 x + a_2 x^2 + \dots$ 입니다.
* **주요 예시:**
  * 상수 수열 $(1, 1, 1, \dots)$: 기하급수 공식에 의해 $\frac{1}{1-x}$로 표현됩니다.
  * 유한 수열 $(1, 1, \dots, 1)$ ($n$개): $\frac{1-x^{n+1}}{1-x}$.
  * 등비 수열 $(1, 2, 4, 8, \dots)$: $\frac{1}{1-2x}$.

## 3. 차분 (Differencing) - 6.1.2

생성 함수를 조작하여 수열의 성질을 분석하는 기법 중 하나입니다.

* **아이디어:** $A(x)$에 $(1-x)$를 곱하면, 결과 다항식의 계수는 원래 수열의 계차(Difference)가 됩니다.
    $$ (1-x) \sum a_n x^n = \sum (a_n - a_{n-1}) x^n $$
* **응용:** 이 성질을 이용하면 $a_n = n^2$과 같은 다항 수열의 생성 함수를 찾을 수 있습니다. 미분 연산과 유사하게 수열의 차수를 낮추는 효과가 있습니다.

## 4. 곱셈과 부분합 (Multiplication and Partial Sums) - 6.1.3

두 생성 함수의 곱셈이 수열에 미치는 영향을 다룹니다.

* **곱셈 (Convolution):** $A(x)B(x)$의 $x^n$ 계수는 $\sum_{k=0}^n a_k b_{n-k}$가 됩니다. 이는 조합론에서 두 개의 독립적인 선택 과정을 결합할 때 유용합니다.
* **부분합:** 수열 $a_n$의 부분합 수열 $S_n = \sum_{i=0}^n a_i$의 생성 함수는 $A(x) \cdot \frac{1}{1-x}$입니다. 즉, $\frac{1}{1-x}$를 곱하는 것은 수열을 누적하여 더하는 효과를 가집니다.

## 5. 점화식 풀이 (Solving Recurrence Relations) - 6.1.4

생성 함수의 가장 강력한 응용 중 하나는 점화식을 푸는 것입니다.

1. **설정:** 점화식의 양변에 $x^n$을 곱하고 $n$에 대해 합을 취합니다.
2. **변환:** 합의 식을 생성 함수 $A(x)$에 대한 대수 방정식으로 바꿉니다. (인덱스 이동 주의)
3. **해결:** 방정식을 풀어 $A(x)$를 구합니다 (주로 유리함수 형태).
4. **전개:** 부분분수 분해(Partial Fraction Decomposition) 등을 이용해 $A(x)$를 다시 급수 형태로 전개하여 $a_n$의 일반항을 찾습니다.

## 6. 요약

**6.1 Generating Functions**는 복잡한 수열 문제를 단순한 대수 계산으로 바꿉니다.

* 수열의 합, 차분, 점화식 풀이 등을 통합된 관점에서 처리할 수 있습니다.
* 특히 "조합론적 증명"이 어렵거나 직관이 통하지 않는 복잡한 문제에서 기계적이고 강력한 해결책을 제공합니다.

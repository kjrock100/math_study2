# 6.1.3 Multiplication and Partial Sums 분석

이 문서는 **6.1.3 Multiplication and Partial Sums** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 두 생성 함수의 곱셈이 수열의 **합성곱(Convolution)**에 해당한다는 사실과, 이를 응용하여 수열의 **부분합(Partial Sums)**을 구하는 방법을 다룹니다.

## 1. 섹션 개요 (Overview)

생성 함수의 덧셈이 수열의 항별 덧셈에 대응된다면, 곱셈은 무엇에 대응될까요? 단순히 항별 곱셈($a_n b_n$)이 아닙니다. 두 다항식을 곱할 때 항들이 섞이는 방식을 생각해보면, 이는 조합론에서 매우 중요한 연산인 **합성곱**과 연결됨을 알 수 있습니다.

## 2. 생성 함수의 곱셈 (Multiplication)

두 수열 $a_n$과 $b_n$의 생성 함수를 각각 $A(x), B(x)$라고 합시다. 두 함수의 곱 $A(x)B(x)$를 전개했을 때 $x^n$의 계수는 어떻게 될까요?

$$
\left(\sum_{i=0}^{\infty} a_i x^i \right) \left(\sum_{j=0}^{\infty} b_j x^j \right) = \sum_{n=0}^{\infty} \left( \sum_{k=0}^{n} a_k b_{n-k} \right) x^n
$$

* **합성곱 (Convolution):** 새로운 수열의 $n$번째 항은 $c_n = \sum_{k=0}^{n} a_k b_{n-k}$가 됩니다.
* **의미:** $a_k$는 첫 번째 그룹에서 $k$개를 선택하는 방법의 수, $b_{n-k}$는 두 번째 그룹에서 나머지 $n-k$개를 선택하는 방법의 수라고 할 때, $c_n$은 두 그룹을 합쳐서 총 $n$개를 선택하는 모든 방법의 수가 됩니다.

## 3. 부분합 (Partial Sums)

곱셈의 특수한 경우로, 한 수열이 상수 수열 $1, 1, 1, \dots$인 경우를 생각해 봅시다. 이 수열의 생성 함수는 $\frac{1}{1-x}$입니다.

어떤 수열 $a_n$의 생성 함수 $A(x)$에 $\frac{1}{1-x}$를 곱하면 어떻게 될까요?
위의 합성곱 공식에서 $b_k = 1$ (모든 $k$에 대해)을 대입하면 됩니다.

$$ c_n = \sum_{k=0}^{n} a_k \cdot 1 = a_0 + a_1 + \dots + a_n $$

즉, 결과 수열의 $n$번째 항은 원래 수열의 **부분합($S_n$)**이 됩니다.

* **공식:** 수열 $a_n$의 부분합 수열의 생성 함수는 $\frac{A(x)}{1-x}$입니다.

### 예시: 삼각수 (Triangular Numbers)

삼각수 수열 $1, 3, 6, 10, \dots$은 자연수 수열 $1, 2, 3, 4, \dots$의 부분합입니다.

1. 자연수 수열의 생성 함수: $\frac{1}{(1-x)^2}$ (이는 $1, 1, 1, \dots$의 미분 혹은 차분 역연산으로 유도 가능)
2. 삼각수 수열의 생성 함수: 여기에 $\frac{1}{1-x}$를 곱합니다.
    $$ \frac{1}{(1-x)^3} $$
3. 이항 정리 등을 이용해 계수를 구하면 $\binom{n+2}{2} = \frac{(n+1)(n+2)}{2}$가 나옵니다.

## 4. 요약

**6.1.3 Multiplication and Partial Sums**는 생성 함수를 조작하는 강력한 도구를 제공합니다.

* **곱셈:** 두 독립적인 사건을 결합하여 총 $n$개를 만드는 경우의 수를 셀 때 사용합니다.
* **나눗셈 ($(1-x)$로 나누기):** 수열을 누적하여 더하는 부분합을 구할 때 사용합니다. (반대로 $(1-x)$를 곱하면 차분을 구합니다.)

# 6.1.2 Differencing 분석

이 문서는 **6.1.2 Differencing** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 생성 함수를 조작하여 수열의 **계차(Difference)**를 구하고, 이를 통해 다항 수열(Polynomial Sequences)의 생성 함수를 찾는 방법을 다룹니다.

## 1. 섹션 개요 (Overview)

등비수열이 아닌 수열, 예를 들어 $1, 4, 9, 16, \dots$ ($a_n = (n+1)^2$)과 같은 수열의 생성 함수는 어떻게 구할까요?
이 섹션에서는 생성 함수에 $(1-x)$를 곱하는 연산이 수열의 **차분(Difference)**을 계산하는 것과 같다는 점을 이용하여, 복잡한 수열을 더 단순한 수열(상수 수열 등)로 변환하는 기법을 소개합니다.

## 2. 차분 연산의 원리 (The Mechanics of Differencing)

수열 $a_n$의 생성 함수를 $A(x) = \sum_{n=0}^{\infty} a_n x^n$이라고 합시다.
이때 $A(x)$에 $(1-x)$를 곱하면 다음과 같은 일이 일어납니다.

$$
\begin{aligned}
(1-x)A(x) &= (1-x)(a_0 + a_1 x + a_2 x^2 + \dots) \\
&= (a_0 + a_1 x + a_2 x^2 + \dots) - (a_0 x + a_1 x^2 + a_2 x^3 + \dots) \\
&= a_0 + (a_1 - a_0)x + (a_2 - a_1)x^2 + (a_3 - a_2)x^3 + \dots
\end{aligned}
$$

즉, 결과 다항식의 계수는 원래 수열의 **계차수열**이 됩니다.

* 상수항: $a_0$
* $x^n$의 계수: $a_n - a_{n-1}$ (단, $n \ge 1$)

## 3. 다항 수열에의 응용 (Application to Polynomial Sequences)

이 기법은 일반항이 $n$에 대한 다항식인 수열의 생성 함수를 찾을 때 매우 강력합니다.
다항식 수열을 차분하면 차수(degree)가 1씩 낮아지며, 결국에는 **상수 수열**이 되기 때문입니다.

### 예시: 제곱수 수열 $1, 4, 9, 16, \dots$

이 수열의 일반항은 $(n+1)^2$입니다. (인덱스 0부터 시작한다고 가정 시)

1. **원래 수열 ($A$):** $1, 4, 9, 16, 25, \dots$
2. **1차 차분 ($(1-x)A$):**
    * 첫 항: $1$
    * 차이: $4-1=3, 9-4=5, 16-9=7, \dots$
    * 수열: $1, 3, 5, 7, \dots$ (등차수열)
3. **2차 차분 ($(1-x)^2 A$):**
    * 첫 항: $1$
    * 차이: $3-1=2, 5-3=2, 7-5=2, \dots$
    * 수열: $1, 2, 2, 2, \dots$ (거의 상수 수열)

이제 역으로 추적합니다.

* 수열 $1, 2, 2, 2, \dots$의 생성 함수는 $1 + 2x + 2x^2 + \dots = 1 + \frac{2x}{1-x} = \frac{1+x}{1-x}$ 입니다.
* 따라서, $(1-x)^2 A(x) = \frac{1+x}{1-x}$ 입니다.
* 결론적으로 $A(x) = \frac{1+x}{(1-x)^3}$ 입니다.

## 4. 요약

**6.1.2 Differencing**은 미분(Differentiation)의 이산 수학 버전입니다.

* **핵심:** $(1-x)$를 곱하면 수열의 차분을 구할 수 있다.
* **용도:** 다항식 패턴을 가지는 수열의 닫힌 생성 함수 공식을 유도하는 데 사용된다.
* 이 방법은 수열의 규칙성을 껍질 벗기듯이 한 단계씩 낮추어 가장 단순한 형태(상수)로 만든 뒤, 다시 식을 재구성하는 전략입니다.

# 4.1.4 Sequences in python 분석

이 문서는 **4.1.4 Sequences in python** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 앞서 배운 수열의 수학적 개념(닫힌 공식, 재귀적 정의)을 실제 프로그래밍 언어인 **Python**을 사용하여 구현하는 방법을 다룹니다.

## 1. 섹션 개요 (Overview)

이산 수학은 컴퓨터 과학의 언어이며, 수열은 프로그래밍에서 가장 기본적인 데이터 구조 중 하나입니다. 이 섹션에서는 수학적인 수열 정의를 Python 코드로 변환하는 방법을 배우며, 이를 통해 수열의 항을 실제로 계산하고 패턴을 탐구하는 능력을 기릅니다.

## 2. 수열의 표현 (Representing Sequences)

### 2.1 리스트 (Lists)

* **용도:** 유한한 수열(Finite Sequences)을 저장할 때 사용합니다.
* **특징:** Python의 리스트 인덱스는 0부터 시작하므로, 수학적 표기인 $a_0, a_1, a_2, \dots$와 자연스럽게 매핑됩니다.
* **예시:**

    ```python
    my_sequence = [1, 3, 5, 7, 9]
    print(my_sequence[2]) # 출력: 5 (a_2)
    ```

### 2.2 함수 (Functions)

* **용도:** 무한 수열(Infinite Sequences)을 표현하거나, 특정 항 $a_n$을 계산할 때 사용합니다.
* **닫힌 공식 구현:**

    ```python
    def a(n):
        return 2*n + 1 # a_n = 2n + 1
    ```

## 3. 수열 생성하기 (Generating Sequences)

### 3.1 리스트 컴프리헨션 (List Comprehension)

* 닫힌 공식을 사용하여 수열의 처음 $k$개 항을 생성하는 가장 파이썬다운(Pythonic) 방법입니다.
* **수학:** $a_n = n^2$ ($n=0$ to $9$)
* **코드:**

    ```python
    squares = [n**2 for n in range(10)]
    # 결과: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
    ```

### 3.2 재귀 함수 (Recursive Functions)

* 재귀적 정의(점화식 + 초기 조건)를 코드로 옮길 때 사용합니다.
* **수학:** $a_n = a_{n-1} + a_{n-2}$, $a_0=0, a_1=1$ (피보나치)
* **코드:**

    ```python
    def fib(n):
        if n == 0: return 0      # 초기 조건
        if n == 1: return 1      # 초기 조건
        return fib(n-1) + fib(n-2) # 점화식
    ```

* **주의:** 단순 재귀는 계산 중복으로 인해 $n$이 커지면 매우 느려질 수 있습니다. (메모이제이션 등으로 최적화 필요)

## 4. 요약

**4.1.4 Sequences in python**은 수학과 코딩의 연결 고리를 보여줍니다.

* **닫힌 공식** $\leftrightarrow$ **함수 (return 수식)** 또는 **리스트 컴프리헨션**
* **재귀적 정의** $\leftrightarrow$ **재귀 함수 (if base_case ... else recursive_call)**

이러한 구현 능력은 알고리즘 문제를 해결하거나 수학적 가설을 검증할 때 필수적인 도구가 됩니다.

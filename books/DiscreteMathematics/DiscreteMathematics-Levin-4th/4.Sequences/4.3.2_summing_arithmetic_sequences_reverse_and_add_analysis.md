# 4.3.2 Summing Arithmetic Sequences: Reverse and Add 분석

이 문서는 **4.3.2 Summing Arithmetic Sequences: Reverse and Add** 섹션의 내용을 분석하고 설명합니다. 이 섹션은 등차수열의 합을 구하는 매우 우아하고 유명한 기법인 **"거꾸로 더하기(Reverse and Add)"**를 소개합니다.

## 1. 섹션 개요 (Overview)

이 섹션은 등차수열의 부분합(partial sum)을 구하는 닫힌 공식을 유도하는 데 초점을 맞춥니다. 이 방법은 어린 시절의 수학자 가우스가 1부터 100까지의 합을 순식간에 계산했다고 알려진 바로 그 방법으로, 복잡한 시그마($\Sigma$) 계산 없이 직관적으로 공식을 이끌어냅니다.

## 2. "거꾸로 더하기" 기법 (The "Reverse and Add" Technique)

이 기법의 핵심 아이디어는 구하려는 합을 두 번, 한 번은 순서대로, 다른 한 번은 거꾸로 써서 더하는 것입니다.

### 2.1 과정

1. 구하려는 합을 $S_n$이라고 둡니다.
    $$ S_n = a_1 + a_2 + \dots + a_{n-1} + a_n $$

2. $S_n$을 항의 순서를 거꾸로 하여 다시 씁니다.
    $$ S_n = a_n + a_{n-1} + \dots + a_2 + a_1 $$

3. 두 식을 세로로 더합니다.
    $$ 2S_n = (a_1 + a_n) + (a_2 + a_{n-1}) + \dots + (a_{n-1} + a_2) + (a_n + a_1) $$

4. **핵심 관찰:** 등차수열의 성질에 의해, 각 세로 항의 합은 모두 같습니다.
    * $a_k = a_1 + (k-1)d$
    * $a_{n-k+1} = a_n - (k-1)d$
    * 따라서 $a_k + a_{n-k+1} = a_1 + a_n$ 입니다.

5. 결과적으로, $n$개의 $(a_1 + a_n)$ 항이 더해집니다.
    $$ 2S_n = n \cdot (a_1 + a_n) $$

## 3. 등차수열의 합 공식 (The Formula)

위 식을 $S_n$에 대해 정리하면 등차수열의 합 공식을 얻습니다.

$$ S_n = \frac{n(a_1 + a_n)}{2} $$

* **의미:** (항의 개수) $\times$ (첫 항과 끝 항의 평균)

## 4. 예시: 1부터 100까지의 합

* $S = 1 + 2 + \dots + 99 + 100$
* $S = 100 + 99 + \dots + 2 + 1$
* $2S = (1+100) + (2+99) + \dots + (100+1) = 101 \times 100$
* $S = \frac{101 \times 100}{2} = 5050$

## 5. 요약

**4.3.2 Summing Arithmetic Sequences: Reverse and Add**는 단순한 계산 기술을 넘어, 문제의 대칭성을 활용하여 복잡도를 낮추는 아름다운 알고리즘적 사고를 보여줍니다.

* **핵심:** 대칭성을 이용한 쌍(Pairing) 만들기.
* **결과:** 등차수열(1차식)의 합은 $n$에 대한 2차식이 됩니다. ($S_n = \frac{n(a_1 + a_1 + (n-1)d)}{2} = \frac{d}{2}n^2 + \dots$) 이는 1차 수열의 '적분'이 2차 수열이 된다는 이산 미적분학의 원리를 확인시켜 줍니다.
